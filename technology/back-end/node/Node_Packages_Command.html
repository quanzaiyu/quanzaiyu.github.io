                # npm 和 yarn 命令对比

                # 初始化

                npm：npm init -y

                yarn: yarn init -y

                # 安装

                # 在已有项目中安装依赖

                npm：npm i

                yarn: yarn

                # 全局安装

                npm：npm i -g <package>

                yarn：yarn global add <package>

                注：

                • i 是 install 的缩写
                • -g 是 --global 的缩写

                # 添加依赖

                npm：npm i [–S/–D] <package>

                yarn：yarn add [–S/–D] <package>

                注：

                • --save 是将包添加到项目生产时的依赖，在 package.json 中的 dependencies 下
                • --save-dev 是将包添加到开发的依赖，在 package.json 中的 devDependencies 下
                • -S 是 --save 的缩写，默认参数，可省略
                • -D 是 --save-dev 的缩写

                # 更新

                npm：npm update [-g] [–S/–D] <package>

                yarn：yarn [global] upgrade <package>

                # 卸载

                npm：npm un [-g] [–S/–D] <package>

                yarn: yarn remove <package>

                注：

                • un 是 uninstall 的缩写

                # 重装

                npm: rm -rf node_modules && npm install

                yarn：yarn upgrade

                # 清除缓存

                npm：npm cache clean

                yarn：yarn cache clean

                # 查看信息

                查看包信息：

                yarn info <package>
npm info <package>

                查看配置信息：

                yarn config list
npm config list

                2

                获取指定配置信息：

                yarn config get registry
npm config get registry
