# 搭建 Git 服务器

# 服务器端

  1. 先从 yum 安装 git
yum –y install git
1
  1. 创建一个 git 用户并赋予密码
useradd git && passwd git
1
  1. 在需要的位置创建一个裸仓库（最后以 .git 结尾）
cd /home/git && mkdir git-server && cd git-server
git init --bare learngit.git
1
2
  1. 赋予 git 用户权限

需要给新建的 git 仓库赋予 git 用户的完全控制权，使用 -R 参数将仓库的子文件/目录全部赋予 git 用户所有

chown -R git:git learngit.git
1
  1. 禁用 git 用户 shell 登录（一定要禁用）
vim /etc/passwd
1

将 git 用户修改如下（一般在最后一行）

git:x:1001:1001:,,,:/home/git:/bin/bash
1

改为

git:x:1000:1000:,,,:/home/git:/usr/bin/git-shell
1

其他的不用改。服务端配置完成。

# 客户端

使用 clone 命令克隆仓库，格式如下

git clone git@172.23.29.183:/home/git/git-server/learngit.git
git clone root@172.23.29.183:/home/git/git-server/learngit.git
1
2

可以将客户端公钥保存到服务器，以便下次操作不需输入密码。
