    # Jenkins 通过 SSH 发布构建产物

    1. 安装 Publish Over SSH 插件
    1. 系统管理 > 系统设置 > Publish over SSH
    • Passphrase: 私钥密码
    • Path to key: 私钥路径
    • Key: 私钥内容(如果设置Key则Path to key无效)
    • SSH Servers: SSH 服务器配置
    1. 流水线配置

    项目 > Pipeline Syntax > 片段生成器 > sshPublisher

    生成相关构建语句:

    • Source files 构建中生成的文件
    • Remove prefix SSH服务器的前缀, 通常留空
    • Remote directory SSH服务器的远程目录
    • Exec command 在SSH服务器中执行的命令
    1. 创建 Jenkinsfile

    Jenkinsfile 示例, 本站点的构建语句:

    pipeline {
    agent { docker 'node:latest' }
    triggers {
      cron 'H H * * *'
    }
    stages {
        stage('prepare') {
            steps {
                sh 'whoami'
                sh 'node --version'
                sh 'npm --version'
            }
        }
        stage('config') {
            steps {
                sh 'npm config set chromedriver_cdnurl http://cdn.npm.taobao.org/dist/chromedriver'
                sh 'npm config set sass_binary_site https://npm.taobao.org/mirrors/node-sass/'
                sh 'npm config set phantomjs_cdnurl https://npm.taobao.org/mirrors/phantomjs/'
                sh 'npm config set electron_mirror https://npm.taobao.org/mirrors/electron/'
            }
        }
        stage('build') {
            steps {
                sh 'npm install'
                sh 'npm run build'
                sh 'mkdir -p www'
                sh 'rm -rf www/*'
                sh '\\cp -r -a dist/* www'
                sh 'ls www'
                sh 'pwd'
            }
        }
        stage('publish') {
            steps {
              sshPublisher(
                publishers: [
                  sshPublisherDesc(
                    configName: 'root@sshk3',
                    transfers: [
                      sshTransfer(
                        sourceFiles: 'dist/**/*',
                        remoteDirectory: '/usr/share/nginx/html/www/xiaoyulive',
                        cleanRemote: true,
                        excludes: '',
                        execCommand:  '''
                                        echo \'ssh transfer ok\'
                                        pwd
                                        cd /usr/share/nginx/html/www/xiaoyulive
                                        pwd
                                        \\cp -r -a dist/* .
                                        rm -rf dist
                                        cd ..
                                        pwd
                                        \\cp -r -a xiaoyulive/* quanzaiyu.coding.me
                                        cd quanzaiyu.coding.me
                                        pwd
                                        git add .
                                        git commit -am update
                                        git push origin master
                                      ''',
                        execTimeout: 120000,
                        flatten: false,
                        makeEmptyDirs: false,
                        noDefaultExcludes: false,
                        patternSeparator: '[, ]+',
                        remoteDirectorySDF: false,
                        removePrefix: ''
                      )
                    ],
                    usePromotionTimestamp: false,
                    useWorkspaceInPromotion: false,
                    verbose: false
                  )
                ]
              )
            }
        }
        stage('test') {
            steps {
                sh 'curl blog.xiaoyulive.top'
            }
        }
    }
}
