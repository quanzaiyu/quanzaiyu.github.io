        # 在 Coding 中使用 webhook 部署应用(PHP)

        # 准备工作

        1. 在 Coding 中创建个人项目
        2. 在服务器中安装git, nginx, php等相关服务

        # 一步一步搭建

        1. 创建web服务器用户目录，这里以apache用户为例，不同的环境请根据自己环境自行修改：
        $ mkdir /home/www/.ssh
$ chown -R apache:apache /home/www/.ssh/
        1. 生成公钥

        公钥有两个：1. git用户公钥 2. 部署公钥

        git用户公钥, 我们使用 root 用户生成:

        $ ssh-keygen
$ cat /root/.ssh/id_rsa.pub

        部署公钥, 我们使用 apache 用户生成:

        $ sudo -Hu apache ssh-keygen -t rsa
$ cat /var/www/.ssh/id_rsa.pub
        • 将 git用户公钥 粘到 Coding 个人设置页
        • 将 部署公钥 粘到 Coding项目 > 设置 > 部署公钥
        1. 拉取远程项目

        比如在 /usr/share/nginx/html/repos/www.test.com 目录中拉取了coding中的代码

        $ cd /usr/share/nginx/html/repos
$ sudo -Hu apache git clone git@git.dev.tencent.com:zcntpz/test.git www.test.com

        注意: 一定要使用 apache 用户进行拉取

        1. 创建一个 hook.php
        <?php
echo shell_exec("whoami");

error_reporting(1);

$target = '/usr/share/nginx/html/repos/www.test.com'; // 生产环境web目录

$json = json_decode(file_get_contents('php://input'), true);

var_dump($json);

$cmd = "cd $target;git pull";

echo 'cmd: ' . $cmd;

$ret = shell_exec($cmd);

echo $ret;

        在 nginx 中配置相关服务(php, 虚拟域名), 确保在浏览器中能访问此文件, 比如 http//:www.test.com/hook.php

        1. 在 Coding 中添加 hook

        至此, 所有配置完毕, 其实实际搭建中可能会遇到很多问题, 比如用户权限问题等等, 耐心一点慢慢调, 最终总会成功的。
