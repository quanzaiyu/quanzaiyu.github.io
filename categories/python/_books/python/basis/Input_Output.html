# 输入输出语句

# 输出

用print()在括号中加上字符串，就可以向屏幕上输出指定的文字。

print('hello world')

print() 函数也可以接受多个字符串，用逗号“,”隔开，就可以连成一串输出。print()会依次打印每个字符串，遇到逗号“,”会输出一个空格。

print('hello', 'world')

# 输入

Python提供了一个 input()，可以让用户输入字符串，并存放到一个变量里。

name = input()
print('hello', name)

也可使用 sys.stdout 命名空间下的输出语句:

import sys;
x = 'hello world';
sys.stdout.write(x + '\n')
