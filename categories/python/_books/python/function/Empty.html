# 空函数

使用 pass 语句可以创建一个空函数

def nop():
  pass

pass语句什么都不做，可以用来作为占位符，比如现在还没想好怎么写函数的代码，就可以先放一个pass，让代码能运行起来。

pass还可以用在其他语句里，比如：

if age >= 18:
    pass

缺少了pass，代码运行就会有语法错误。
