              # Python 简介、安装与使用

              • Python3
              • FTP镜像
              • Unofficial Windows Binaries for Python Extension Packages

              Python是一门动态语言。变量本身类型不固定的语言称之为动态语言，与之对应的是静态语言。静态语言在定义变量时必须指定变量类型，如果赋值的时候类型不匹配，就会报错。

              Python的3.0版本，常被称为Python 3000，或简称Py3k。相对于Python的早期版本，这是一个较大的升级。相对于Python2 (最新版本为2.7) ，有了更多的特性，很多语法规则也有了改变，在设计的时候没有考虑向下兼容。

              # venv

              在开发Python应用程序的时候，系统安装的 Python3 只有一个版本, 比如3.7。所有第三方的包都会被pip安装到Python3的site-packages目录下。

              如果我们要同时开发多个应用程序，那这些应用程序都会共用一个 Python，就是安装在系统的 Python3。如果应用A需要 jinja 2.7，而应用B需要jinja 2.6 怎么办？

              这种情况下，每个应用可能需要各自拥有一套“独立”的Python运行环境。virtualenv 就是用来为一个应用创建一套“隔离”的Python运行环境。(可理解成每个 node 项目下的 node_modules)

              安装 venv

              $ pip3 install virtualenv

              进入项目目录, 创建一个独立的Python运行环境，命名为venv

              $ virtualenv --no-site-packages venv

              加上 --no-site-packages 选项，已经安装到系统Python环境中的所有第三方包都不会复制过来，这样，我们就得到了一个不带任何第三方包的“干净”的Python运行环境。

              安装成功后, 可以在项目目录下看到一个 venv 文件夹, 激活后终端也可看到加了 (venv) 前缀:

              PS D:\Documents\crawler> & d:/Documents/crawler/venv/Scripts/activate.ps1
(venv) PS D:\Documents\crawler>

              默认应该会自动激活 venv 环境, 如果没有激活, 可执行:

              $ ./venv/Scripts/activate

              如果安装不成功, 可能需要先更新 pip

              $ python -m pip install --upgrade pip

              编写代码的时候, 可以安装 pylint 以进行代码检查

              $ python -m pip install -U pylint

              # 安装与使用

              下载链接: Python Download Python.org

              安装并添加到环境变量，一般安装的时候勾选 Add Python to Environment variables 即可(Windows)

              Linux 一般系统自带 Python，不过只是 Python2，需要使用 Python3 需自行安装。

              如果安装请使用以下命令:

              $ apt-get install python3 -y
$ apt-get install build-essential -y
$ apt-get install python-pip -y
$ pip install --upgrade pip

              使用 python 进入交互模式，使用 exit() 退出交互模式，使用 python -V 查看python版本号。

              $ python
Python 3.7.0a2 (v3.7.0a2:f7ac4fe, Oct 17 2017, 17:06:29) [MSC v.1900 64 bit (AMD64)] on win32
Type "help", "copyright", "credits" or "license" for more information.
# 退出Python交互模式
>>> exit()
# 查看python 版本，注意是大写的V
$ python -V
Python 3.7.0a2

              安装 pylint (用于python代码检测)

              $ python -m pip install pylint

              # 执行py文件

              执行.py文件，直接在 python 后跟上文件名(绝对路径或相对路径)即可，注意必须带上.py扩展名。

              $ python test.py

              # Cython

              Cython语言是Python的一个超集，编译成C语言，产生的性能提升可以从几个百分点到几个数量级，具体取决于手头的任务。 对于受Python原生对象类型约束的工作，加速将不会很大。 但是对于数值操作，或任何不涉及Python自身内部的操作，收益可能是巨大的。 这样，许多Python的本地限制可以被绕过或完全超越。

              # Cython优势

              除了能够加速已经编写的代码之外，Cython还具有其他几个优点：

              • 使用外部C库可以更快
              • 可以同时使用C和Python内存管理
              • 可以根据需要选择安全性或速度
              • Cython可以使用Python类型的提示语法

              # Anaconda

              Anaconda指的是一个开源的Python发行版本，其包含了conda、Python等180多个科学包及其依赖项。因为包含了大量的科学包，Anaconda 的下载文件比较大（约 531 MB），如果只需要某些包，或者需要节省带宽或存储空间，也可以使用Miniconda这个较小的发行版（仅包含conda和 Python）。

              # UnicodeDecodeError的解决方案

              如果安装后使用的时候会报

              错误

              UnicodeDecodeError: 'utf-8' codec can't decode byte 0xa1 in position 43: invalid start byte

              的错误，解决方法如下:

              打开c:\program files\python36\lib\site-packages\pip\compat\__init__.py约75行

              return s.decode('utf_8')

              改为

              return s.decode('cp936')
