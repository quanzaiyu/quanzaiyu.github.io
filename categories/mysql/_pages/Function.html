# SQL Aggregate 聚合函数

SQL Aggregate 函数计算从列中取得的值，返回一个单一的值。

有用的 Aggregate 函数：

  • AVG() - 返回平均值
  • SUM() - 返回总和
  • COUNT() - 函数返回指定列的值的数目
  • MAX() - 返回最大值
  • MIN() - 返回最小值

举例：

SELECT AVG(count) AS CountAverage FROM access_log; -- 求 count 列的平均值，输出 nums 为 174.3333
SELECT site_id, count FROM access_log WHERE count > (SELECT AVG(count) FROM access_log); -- 求大于平均值的指定数据

SELECT COUNT(count) AS nums FROM access_log WHERE site_id=3; -- 求 count 列所有行的条数
SELECT COUNT(*) AS nums FROM access_log; -- 统计 access_log 表有多少条数据，输出 nums 为 9
SELECT COUNT(DISTINCT site_id) AS nums FROM access_log; -- 去重后的 site_id 的条数，输出 nums 为 5

# GROUP BY

GROUP BY 语句用于结合聚合函数，根据一个或多个列对结果集进行分组。

简单查询

SELECT site_id, SUM(access_log.count) AS nums FROM access_log GROUP BY site_id;

返回：

+---------+-------+
| site_id | nums |
+---------+-------+
|       1 |   275 |
|       2 |   10  |
|       3 |   521 |
|       4 |   13  |
|       5 |   750 |
+---------+-------+

下面的 SQL 语句统计所有网站的访问的记录数：

SELECT Websites.name,COUNT(access_log.aid) AS nums FROM access_log
LEFT JOIN Websites
ON access_log.site_id=Websites.id
GROUP BY Websites.name;

返回：

+---------+------+
| name    | nums |
+---------+------+
| Facebook |  2 |
| Google   |  2 |
| 微博      |  1 |
| 淘宝      |  1 |
| 菜鸟教程  |  3 |
+---------+-----+

# HAVING

在 SQL 中增加 HAVING 子句原因是，WHERE 关键字无法与聚合函数一起使用。HAVING 子句可以让我们筛选分组后的各组数据。

查找总访问量大于 200 的网站：

SELECT Websites.name, Websites.url, SUM(access_log.count) AS nums
FROM (access_log INNER JOIN Websites ON access_log.site_id=Websites.id)
GROUP BY Websites.name
HAVING SUM(access_log.count) > 200;

# SQL Scalar 标量函数

SQL Scalar 函数基于输入值，返回一个单一的值。

有用的 Scalar 函数：

  • UCASE() - 将某个字段转换为大写
  • LCASE() - 将某个字段转换为小写
  • SubString(字段，1，end) - 从某个文本字段提取字符，MySql 中使用 MID() 函数
  • LEN() - 返回某个文本字段的长度，MySQL 中为 LENGTH() 函数
  • ROUND() - 对某个数值字段进行指定小数位数的四舍五入
  • NOW() - 返回当前的系统日期和时间
  • FORMAT() - 格式化某个字段的显示方式

举例：

SELECT UCASE('Hello'); -- 'HELLO'
SELECT LCASE('Hello'); -- 'hello'

select ROUND(-1.23); -- 返回 -1
select ROUND(-1.58); -- 返回 -2
select ROUND(1.58); -- 返回 2

# MID

MID() 函数用于从文本字段中提取字符。

SELECT MID(column_name,start[,length]) FROM table_name;

比如：

SELECT MID(name,1,4) AS ShortTitle FROM Websites;

返回：

+--------------+
| ShortTitle   |
+--------------+
| Face         |
| Goog         |
| 微博         |
| 淘宝         |
| 菜鸟教程      |
+-------------+

# NOW

NOW() 函数返回当前系统的日期和时间。

SELECT NOW(); -- 2018-07-16 17:32:23

# FORMAT

DATE_FORMAT() 函数用于格式化时间格式的字段。

比如：

SELECT name, DATE_FORMAT(Now(),'%Y-%m-%d') AS date FROM Websites;

返回：

+---------+--------------+
| name    | date         |
+---------+--------------+
| Facebook |  2018-07-16 |
| Google   |  2018-07-16 |
| 微博      |  2018-07-16 |
| 淘宝      |  2018-07-16 |
| 菜鸟教程  |  2018-07-16 |
+---------+--------------+

# SQL Date 时间日期函数

MySQL 中有如下时间日期函数

函数 描述
NOW() 返回当前的日期和时间
CURDATE() 返回当前的日期
CURTIME() 返回当前的时间
DATE() 提取日期或日期/时间表达式的日期部分
EXTRACT() 返回日期/时间的单独部分
DATE_ADD() 向日期添加指定的时间间隔
DATE_SUB() 从日期减去指定的时间间隔
DATEDIFF() 返回两个日期之间的天数
DATE_FORMAT() 用不同的格式显示日期/时间

比如：

SELECT NOW(); -- 2018-07-17 19:04:07
SELECT CURDATE(); -- 2018-07-17
SELECT CURTIME(); -- 19:04:07
SELECT DATE('2018-01-01 12:00:01'); -- 2018-01-01

# EXTRACT

EXTRACT() 函数用于返回日期/时间的单独部分，比如年、月、日、小时、分钟等等。

格式：

EXTRACT(unit FROM date)

date 参数是合法的日期表达式。unit 参数可以是下列的值：

Unit 值
MICROSECOND
SECOND
MINUTE
HOUR
DAY
WEEK
MONTH
QUARTER
YEAR
SECOND_MICROSECOND
MINUTE_MICROSECOND
MINUTE_SECOND
HOUR_MICROSECOND
HOUR_SECOND
HOUR_MINUTE
DAY_MICROSECOND
DAY_SECOND
DAY_MINUTE
DAY_HOUR
YEAR_MONTH

举例：

SELECT EXTRACT(YEAR FROM '2018-01-01 14:00:02') as Date; -- 2018

# DATE_ADD 和 DATE_SUB

  • DATE_ADD() 函数向日期添加指定的时间间隔。
  • DATE_SUB() 函数从日期减去指定的时间间隔。

格式：

DATE_ADD(date, INTERVAL expr type)
DATE_SUB(date, INTERVAL expr type)

date 参数是合法的日期表达式。expr 参数是您希望添加的时间间隔。

type 参数的值跟上面 EXTRACT 中的 unit 一致。

举例：

SELECT DATE_ADD('2018-01-01 14:00:02', INTERVAL 45 DAY) as Date; -- 2018-02-15 14:00:02

# DATEDIFF

DATEDIFF() 函数返回两个日期之间的天数。

格式：

DATEDIFF(date1,date2)

举例：

SELECT DATEDIFF('2008-11-30','2008-11-29') AS DiffDate; -- 1

# DATE_FORMAT

DATE_FORMAT() 函数用于以不同的格式显示日期/时间数据。

格式：

DATE_FORMAT(date,format)

format 规定日期/时间的输出格式，可以使用的格式有：

格式 描述
%a 缩写星期名
%b 缩写月名
%c 月，数值
%D 带有英文前缀的月中的天
%d 月的天，数值（00-31）
%e 月的天，数值（0-31）
%f 微秒
%H 小时（00-23）
%h 小时（01-12）
%I 小时（01-12）
%i 分钟，数值（00-59）
%j 年的天（001-366）
%k 小时（0-23）
%l 小时（1-12）
%M 月名
%m 月，数值（00-12）
%p AM 或 PM
%r 时间，12-小时（hh:mm:ss AM 或 PM）
%S 秒（00-59）
%s 秒（00-59）
%T 时间, 24-小时（hh:mm:ss）
%U 周（00-53）星期日是一周的第一天
%u 周（00-53）星期一是一周的第一天
%V 周（01-53）星期日是一周的第一天，与 %X 使用
%v 周（01-53）星期一是一周的第一天，与 %x 使用
%W 星期名
%w 周的天（0=星期日, 6=星期六）
%X 年，其中的星期日是周的第一天，4 位，与 %V 使用
%x 年，其中的星期一是周的第一天，4 位，与 %v 使用
%Y 年，4 位
%y 年，2 位

举例：

DATE_FORMAT(NOW(),'%b %d %Y %h:%i %p'); -- Nov 04 2008 11:45 PM
DATE_FORMAT(NOW(),'%m-%d-%Y'); -- 11-04-2008
DATE_FORMAT(NOW(),'%d %b %y'); -- 04 Nov 08
DATE_FORMAT(NOW(),'%d %b %Y %T:%f'); -- 04 Nov 2008 11:45:34:243
