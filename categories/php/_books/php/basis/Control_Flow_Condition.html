# 条件语句

在 PHP 中，提供了下列条件语句：

  • if
  • if...else
  • if...else if....else
  • switch

# if-elseif-else

示例代码

$x = 2;
if ($x === 1) {
  echo "\$x === 1";
} else if ($x === 2) {
  echo "\$x === 2";
} else {
  echo "other value";
}

else if 也可写为 elseif :

$x = 2;
if ($x === 1) {
  echo "\$x === 1";
} elseif ($x === 2) {
  echo "\$x === 2";
} else {
  echo "other value";
}

# switch-case-default

示例代码

$x = 2;
switch ($x) {
  case '1':
    echo "\$x === 1";
    break;
  case '2':
    echo "\$x === 2";
    break;
  default:
    echo "other value";
    break;
}

提示

不要在使用switch语句的时候漏掉 break，否则可能会出现意料之外的情况。
