# 配置

数据库配置文件位于 config/database.php 下。

比如配置 :

// 数据库类型
'type'            => 'mysql',
// 服务器地址
'hostname'        => '127.0.0.1',
// 数据库名
'database'        => 'test',
// 用户名
'username'        => 'root',
// 密码
'password'        => 'root',

如果端口默认未改动，mysql 默认为 3306，则不需要配置 hostport 字段。

# 针对单个模块进行数据库配置

如果要针对某个模块进行单独的数据库配置，可以在 config 下新建一个同名模块的文件夹，在此文件夹下新建 database.php 即可覆盖全局的数据库配置。

# 全局配置

配置好数据库连接文件 database.php 后，就可在控制器中进行数据库连接。

使用 table 语句选择数据表。

测试是否连接正确，在控制器中 :

namespace app\index\controller;

use think\Controller;
use think\Db;

class Index extends Controller
{
    public function index()
    {
      return Db::table('user')
        ->where('id', 1)
        ->value('name');
    }
}

访问 http://www.my-tp5.com/ 可以看到输出 xiaoyu。

# 动态配置

如果没有配置 连接文件 database.php，或者需要一次性连接某个数据库，可以使用 动态配置。

namespace app\index\controller;

use think\Controller;
use think\Db;

class Index extends Controller
{
    public function index()
    {
      return Db::connect([
        'type'            => 'mysql',
        'hostname'        => '127.0.0.1',
        'database'        => 'test',
        'username'        => 'root',
        'password'        => 'root'
      ])
        ->table('user')
        ->where('id', 1)
        ->value('name');
    }
}

# DSN 连接

使用以下格式的连接字符串 :

数据库类型://用户名:密码@数据库地址:端口号/数据库名称#字符集

进行连接。

如:

namespace app\index\controller;

use think\Controller;
use think\Db;

class Index extends Controller
{
    public function index()
    {
      $dsn = 'mysql://root:root@127.0.0.1:3306/test#utf8';
      return Db::connect($dsn)
        ->table('user')
        ->where('id', 2)
        ->value('name');
    }
}

如果输出 xiaoqiao 则连接正确。
