# 数据库: 查询

静态调用 think\db\Query.php 类中的查询方法实现基本操作。

# where

使用 where 设置查询条件，参数是表达式或数组。

格式

where(field, op, condition)

具体操作为:

  1. 单个条件，使用表达式
  2. 多个条件，使用数组

# 查询条件

第一个参数是要查询的字段名，必填; 第二个参数是条件表达式，可以省略，如果省略则是 等于; 第三个参数是查询条件，必填。

$res = Db::table('user')
  ->where('id',1)
  ->find();
dump(is_null($res) ? 'Not find!' : $res);

以上语句转换问 SQL 为

SELECT * from `user` WHERE id=1

# 单个条件

public function find() {
  $res = Db::table('user')
    ->where('id','=',1)
    ->find();
  dump(is_null($res) ? 'Not find!' : $res);
}

以上语句转换问 SQL 为

SELECT * from `user` WHERE id=1

# 多个条件

$res = Db::table('user')
->where([
  ['id','>',1],
  ['password','>',100],
])
->find();

以上语句转换问 SQL 为

SELECT * from `user` WHERE id>1 AND password>100

# 主键查询

如果为主键查询，可以省略 where 语句，而在 find 中指定查询的主键

$res = Db::table('user')
  ->find(1);
dump(is_null($res) ? 'Not find!' : $res);

以上语句转换问 SQL 为

SELECT * from `user` WHERE id=1

# 单条查询

find 返回复合条件的 第一条记录，没有的话返回 NULL。

$res = Db::table('user')
  ->find(1);

转换为 SQL 为:

SELECT * FROM `user` WHERE `id`=1

# 多条查询

select 返回复合条件的所有记录。

$res = Db::table('user')
  ->where([
    ['id','>',1],
    ['password','>',100],
  ])
  ->select();
if (empty($res)) {
  return 'Not find !';
} else {
  foreach ($res as $row) {
    dump($row);
  }
}

转换为 SQL 为:

SELECT * FROM `user` WHERE `id` > 1 AND `password` > '100'

# field

设置返回的字段，如果不设置则为返回所有字段 (*)

$res = Db::table('user')
    ->field('id, name')
    ->where('id','=',1)
    ->find(1);
dump(is_null($res) ? 'Not find!' : $res);

转换为 SQL 为:

SELECT id, name from user WHERE id = 1

# 别名

可以使用关联数组的方式为返回的字段设置别名。

$res = Db::table('user')
    ->field(['id' => '编号', 'name' => '姓名'])
    ->where('id','=',1)
    ->find(1);
dump(is_null($res) ? 'Not find !' : $res);

转换为 SQL 为:

SELECT id AS `编号`, name AS `姓名` from user WHERE id = 1
