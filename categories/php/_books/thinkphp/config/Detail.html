# 设置调试面板显示位置

在 ThinkPHP5.1 中，在 config/trace.php 中可以配置调试面板显示位置:

// +----------------------------------------------------------------------
// | Trace设置 开启 app_trace 后 有效
// +----------------------------------------------------------------------
return [
  // 内置Html Console 支持扩展
  'type' => 'Html',
];

# 伪静态

在配置文件中可以设置 url_html_suffix，指定伪静态后缀

'url_html_suffix' => 'html',

此时，以下两种访问均为可行:

http://serverName/Home/Blog/read/id/1
http://serverName/Home/Blog/read/id/1.html

URL 伪静态可以满足更好的 SEO 效果。

也可设置多个伪静态后缀:

'url_html_suffix' => 'html|shtml|xml'

关闭伪静态:

'url_html_suffix' => false,

// 默认 AJAX 数据返回格式,可选 json xml ...

// 默认 JSONP 格式返回的处理方法
'default_jsonp_handler'  => 'jsonpReturn',
// 默认 JSONP 处理方法

// 默认时区

// 默认全局过滤方法 用逗号分隔多个
'default_filter'         => '',

// 应用类库后缀

// 控制器类后缀
'controller_suffix'      => false,
