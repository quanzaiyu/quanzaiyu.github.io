# 环境变量配置

在项目根目录下用 .env 文件做配置，文件使用 .ini 配置风格，也就是 配置项名称=配置值。

比如在 .env 文件中输入:

my_env=dev

在控制器中获取:

\think\Env::get('my_env');

也可使用 use 引入命名空间

use \think\Env;

// ...

Env::get('my_env');

# 系统环境变量

可以通过 Env 获取系统环境变量，在 ThinkPHP5.1 中， Env 已经包装在 facade 中。

use think\facade\ENV;

// 获取应用目录（不区分大小写）
echo Env::get('app_path');
// 或者
echo Env::get('APP_PATH');

支持的系统环境变量包括 :

系统路径 Env参数名称
应用根目录 root_path
应用目录 app_path
框架目录 think_path
配置目录 config_path
扩展目录 extend_path
composer目录 vendor_path
运行缓存目录 runtime_path
路由目录 route_path
当前模块目录 module_path
