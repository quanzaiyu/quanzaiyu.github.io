# 静态资源配置

存放静态资源的目录位于 public/static 下，我们可以将一些静态资源 (样式文件、JS文件) 放于此目录下。

定义静态资源存放路径的PHP文件位于 thinkphp/library/think/View.php 下:

$baseReplace = [
  '__ROOT__'   => $root,
  '__URL__'    => $base . '/' . $request->module() . '/' . Loader::parseName($request->controller()),
  '__STATIC__' => $root . '/static',
  '__CSS__'    => $root . '/static/css',
  '__JS__'     => $root . '/static/js',
];

因此，在模板中引入静态资源只需使用对应的变量即可，比如:

<script type="text/javascript" src="__STATIC__/js/index.js"></script>

<link rel="stylesheet" type="text/css" href="__STATIC__/css/index.min.css" />

也可自定义配置:

'view_replace_str'  =>  [
  '__PUBLIC__'=>'/public/',
  '__ROOT__' => '/',
]

或者在调用模板渲染的时候动态设置:

return $this->fetch('index',[],['__PUBLIC__'=>'/public/']);

# 引入静态资源

可以使用 load 标签引入静态资源

语法

{load href='css或js文件路径' /}

比如，以下代码

<script type="text/javascript" src="__STATIC__/lib/jquery/1.9.1/jquery.min.js"></script>

可以替换为

{load href="__STATIC__/lib/jquery/1.9.1/jquery.min.js" /}

同样的，css 也如此

{load href="__STATIC__/static/css/index.min.css" /}

# 语义化引入

我们可以将 load 标签语义化进行引入，如

{js href="__STATIC__/lib/jquery/1.9.1/jquery.min.js" /}

{css href="__STATIC__/static/css/index.min.css" /}
