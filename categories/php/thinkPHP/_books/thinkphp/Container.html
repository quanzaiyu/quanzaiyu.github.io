# 容器

在 tp5.1，引入了容器类，核心文件位于 thinkphp/library/think/Container.php 。

# 绑定一个类到容器中

例子同依赖注入中的Demo类，在 index 控制器中加入方法 :

application/index/Index.php

public function bindClass () {
  \think\Container::set('temp', '\app\common\Demo');
  $temp = \think\Container::get('temp', ['name' => 'xiaoyu']);
  return $temp -> getName();
}

使用 \think\Container::set 可以绑定一个类到容器中，使用 \think\Container::get 可以获取容器中类。

也可使用助手函数 :

public function bindClass () {
  bind('temp', '\app\common\Demo');
  $temp = app('temp', ['name' => 'xiaoyu']);
  return $temp -> getName();
}

其中 bind 对应 \think\Container::set，app 对应 \think\Container::get。

输入 http://www.my-tp5.com/index.php/index/index/bindClass 即可看到效果。

# 绑定一个匿名方法到容器中

例子同依赖注入中的Demo类，在 index 控制器中加入方法 :

application/index/Index.php

public function bindClosure () {
  bind('temp', function ($domin) {
    return 'Domin is' . $domin;
  });
  return app('temp', ['domin' => 'xiaoyulive.top']);
}

输入 http://www.my-tp5.com/index.php/index/index/bindClosure 即可看到效果。
