          # 使用IDM批量下载ts视频并合并

          # 什么是 ts 视频

          ts是日本高清摄像机拍摄下进行的封装格式，全称为MPEG2-TS。ts即"Transport Stream"的缩写。MPEG2-TS格式的特点就是要求从视频流的任一片段开始都是可以独立解码的。(百度百科)

          # 合并多个 ts 视频文件

          编写 Python 进行视频合并:

          join.py

          import os

filedir = r"D:\User\quanzaiyu\Downloads"
new_file = u"%s\\out.ts" % filedir
f = open(new_file, 'wb+')

for i in range(0, 324):
  filepath = u"%s\\1ecf49a0bc1%03d.ts" % (filedir, i)
  print(filepath)
  for line in open(filepath, 'rb'):
    f.write(line)
  f.flush()

f.close()
