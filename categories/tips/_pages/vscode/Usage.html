# 操作锦囊

# 多列选择

按住 Shift + Alt , 同时拖动鼠标左键, 可进行多列选择, 进入多点编辑模式.

# 相同项匹配替换

选中代替换项, 按 ctrl + F2 , 或者右键选择 "更改所有匹配", 会选择文件中所有相同的匹配, 进入多点编辑模式.

# 跳转定义

  • F12 跳转到定义
  • Ctrl + F12 列出所有定义

# 格式调整

  • Ctrl + [ / Ctrl + ] 代码缩进与反缩进
  • Ctrl + Shift + [ / Ctrl + Shift + ] 折叠代码块

# 光标操作

  • Ctrl + D 选中下一个匹配项
  • Ctrl + U 回退上一个光标操作
  • Ctrl + I 选中当前行
  • Ctrl + ↓ / Ctrl + ↑ 向上或向上滚动一行
  • Alt + ↓ / Alt + ↑ 向上或向上移动一行
  • Ctrl + Shift + ↓ / Ctrl + Shift + ↑ 将当前行向下或向上复制一行
  • Ctrl + Enter / Ctrl + Shift + Enter 在当前行上边或下边插入一行

# 编辑

  • Ctrl + X 若不选中, 剪切一行
  • Ctrl + C 若不选中, 复制一行
  • Ctrl + Shift + K 删除当前行
