# 快捷键

设置快捷键 File | Settings | Keymap

# 编辑相关

  • Ctrl + Z 撤销
  • Ctrl + Y 重做（默认为Ctrl + Shift + Z，建议设置为Ctrl + Y）
  • Ctrl + X 删除并剪切行
  • Ctrl + D 复制并粘贴行
  • Ctrl + C 复制并当前行
  • Ctrl + Shift + Up/Down 代码在作用域中向上/下移动。

# 光标相关

  • Shift + Enter 重新开始一行
  • Ctrl + Enter 在后面添加一行，但是光标不跟随
  • Ctrl + W 扩选，多次点选可不断扩选作用域
  • Ctrl + Shift + W 缩选，多次点选可不断缩选作用域
  • F2 或 Shift + F2 高亮错误或警告快速定位
  • Alt + Up/Down 跳转到上一个/下一个方法

# 查找替换

  • Ctrl + G 查找行
  • Ctrl + R 替换文本
  • Ctrl + F 查找文本
  • Ctrl + Shift + F 全局查找文本
  • Ctrl + Shift + R 全局替换文本
  • Ctrl + Shift + L 当前文件查找文本
  • F3 查找下一个
  • Shift + F3 查找上一个
  • Ctrl + Shift + F7 高亮显示所有与选中相同的文本，可选择多个文本，点ESC取消高亮
  • Ctrl + Shift + Alt + N 通过一个字符快速查找位置（必记）

# 代码帮助

  • Tab 写代码时生成代码
  • Alt + / 代码提示
  • Alt + Enter 快速修复、代码建议
  • Ctrl + / 注释当前行（//）
  • Ctrl + Shift + / 多行注释（/**/）
  • Ctrl + Alt + L 格式化代码
  • Alt + F1 查找代码所在位置

# 编辑器

  • Ctrl + E 最近打开的文件
  • Alt + Shift + F 将当前文件加入收藏夹
  • Ctrl + Shift + N 通过文件名快速查找工程内的文件

# 版本控制

  • Ctrl + T 拉取代码
  • Ctrl + K 代码提交
  • Ctrl + + Alt + K 提交并推仓（提交对话框中）

# Project

  • Shift + F6 重构-重命名
  • F6 移动文件

  • Fix ESLint Problems：Alt + Q
