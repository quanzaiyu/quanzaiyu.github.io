# 常用配置

# 颜色主题

File | Settings | Appearance & Behavior | Appearance

可选择

更多颜色主题访问：Color Themes

# 缩进控制

File | Settings | Editor | Code Style | ...

选择具体的语言，进行相关语言的设置

# 字体与颜色

File | Settings | Editor | Font

File | Settings | Editor | Color Scheme

更多颜色主题参考： http://www.phpstorm-themes.com/

# 代码片段

File | Settings | Editor | Live Templates

一些配置变量：

  • $MODULE_REF$ 补全代码之后光标停留的位置

  • $PARAM$

  • 使PHPStorm支持微信小程序代码提示,代码高亮

# 文件模板

File | Settings | Editor | File and Code Templates

可以设置“新建”的代码模板，也可添加新的代码模板

  • Reformat according to style 勾选后根据默认样式重置代码模板
  • Enable Live Template 勾选后将解析模板变量

通过 #[[$var$]]# 插入变量

一些常用的模板变量：

  • $Title$ 标题名称
  • $END$ 光标结束位置

# 代码检测

File | Settings | Editor | Inspections

# 默认编码

File | Settings | Editor | File Encodings

# 关联文件高亮

File | Settings | Editor | File Types
