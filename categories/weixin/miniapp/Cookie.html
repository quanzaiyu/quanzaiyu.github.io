    # 小程序模拟 Cookie

    小程序不是浏览器, 并没有 Cookie, 当然服务器也存不了 Session 了, 要解决这个问题, 需要拿到后端种下的 Cookie, 保存以供之后使用:

    function getSmsCode (mobile) {
  axios.base({
    url: `${baseDomain}/api/code/sms`,
    header: {
      'Content-Type': 'application/x-www-form-urlencoded'
    },
    data: { mobile }
  }, 'POST').then(res => {
    uni.setStorageSync('Cookie', res.header['Set-Cookie'])
  })
}

    注意这一句:

    uni.setStorageSync('Cookie', res.header['Set-Cookie'])

    后端种下的 Cookie 会在头部通过 Set-Cookie 传递, 这里将其存到 Storage 中.

    在需要使用的地方, 在头部 Cookie 字段传递即可:

    function smsLogin (mobile, smsCode) {
  this.$axios.base({
    url: `${baseDomain}/api/authentication/mobile`,
    header: {
      'Content-Type': 'application/x-www-form-urlencoded',
      'Cookie': uni.getStorageSync('Cookie')
    },
    data: { mobile, smsCode }
  }, 'POST').then(res => {
    if (res.data.status) {
      uni.setStorageSync('user', res.data.data)
      uni.setStorageSync('loginStatus', true)
      uni.redirectTo({ url: 'index/index' })
    } else {
      uni.showToast({ title: res.data.message, icon:'none', duration:2000 })
    }
  })
}
