        # RN第三方库 - 吐司

        React Native 的吐司 API ToastAndroid 只支持 Android 平台, 第三方有很多解决方案, 解决了 iOS 和 Android 可以调用同一套 API 实现吐司的效果。

        # react-native-root-toast

        安装:

        $ yarn add react-native-root-toast

        以API方式使用:

        import Toast from 'react-native-root-toast';

// Add a Toast on screen.
let toast = Toast.show('This is a message', {
    duration: Toast.durations.LONG,
    position: Toast.positions.BOTTOM,
    shadow: true,
    animation: true,
    hideOnPress: true,
    delay: 0,
    onShow: () => {
        // calls on toast\`s appear animation start
    },
    onShown: () => {
        // calls on toast\`s appear animation end.
    },
    onHide: () => {
        // calls on toast\`s hide animation start.
    },
    onHidden: () => {
        // calls on toast\`s hide animation end.
    }
});

// You can manually hide the Toast, or it will automatically disappear after a `duration` ms timeout.
setTimeout(function () {
    Toast.hide(toast);
}, 500);

        以组件方式使用使用:

        import React, {Component} from 'react-native';
import Toast from 'react-native-root-toast';

class Example extends Component{
    constructor() {
        super(...arguments);
        this.state = {
            visible: false
        };
    }

    componentDidMount() {
        setTimeout(() => this.setState({
            visible: true
        }), 2000); // show toast after 2s

        setTimeout(() => this.setState({
            visible: false
        }), 5000); // hide toast after 5s
    };

    render() {
        return <Toast
            visible={this.state.visible}
            position={50}
            shadow={false}
            animation={false}
            hideOnPress={true}
        >This is a message</Toast>;
    }
}

        参数:

        • duration number 持续时长
          • Toast.durations.SHORT (equals to 2000) 默认值
          • Toast.durations.LONG (equals to 3500)
        • position number 显示位置
          • Toast.positions.TOP (equals to 20)
          • Toast.positions.BOTTOM (equals to -20) 默认值
          • Toast.positions.CENTER (equals to 0)
        • animation bool 是否显示动画
        • visible bool 可见性, 默认 false(仅用于组件中设置)
        • delay number 延迟执行时长, 默认0
        • shadow bool 是否显示阴影, 默认true
        • shadowColor color 阴影颜色
        • textColor color 文字颜色
        • backgroundColor color 背景色
        • hideOnPress bool 默认true
        • opacity number 透明度, 默认0.8
        • onShow function Callback for toast`s appear animation start
        • onShown function Callback for toast`s appear animation end
        • onHide function Callback for toast`s hide animation start
        • onHidden function Callback for toast`s hide animation end

        # react-native-toaster

        安装:

        $ yarn add react-native-toaster

        使用:

        import React, { Component } from 'react'
import { View } from 'react-native'
import Toaster, { ToastStyles } from 'react-native-toaster'

export default class App extends Component {
  constructor (props) {
    super(props)
    this.state = { message: null }

    const messages = [
      { text: 'FYI' },
      { text: 'Hooray!', styles: ToastStyles.success },
      { text: 'Eek', styles: ToastStyles.warning },
      { text: 'Oh noe!', styles: ToastStyles.error }
    ]

    // Send each message 1 second apart
    messages.forEach((message, i) => {
      setTimeout(() => this.setState({ message }), i * 1000)
    })
  }

  render () {
    return (
      <View>
        <Toaster message={this.state.message} />
      </View>
    )
  }
}

        参数:

        • message object 要展示的信息
          • text string 显示的文字
          • styles enum of ToastStyles 样式
            • success/warning/error
          • duration number 持续时间, 默认3000
          • height number 高度, 默认100
        • onShow
        • onHide
        • onPress
