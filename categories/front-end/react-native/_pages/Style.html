        # react-native 尺寸、布局与样式

        # StyleSheet

        在 React Native 中, 使用 StyleSheet.create 来创建样式表, 比如

        import React, { Component } from 'react';
import { AppRegistry, StyleSheet, Text, View } from 'react-native';

export default class LotsOfStyles extends Component {
  render() {
    return (
      <View>
        <Text style={styles.red}>just red</Text>
        <Text style={styles.bigBlue}>just bigBlue</Text>
        <Text style={[styles.bigBlue, styles.red]}>bigBlue, then red</Text>
        <Text style={[styles.red, styles.bigBlue]}>red, then bigBlue</Text>
      </View>
    );
  }
}

const styles = StyleSheet.create({
  bigBlue: {
    color: 'blue',
    fontWeight: 'bold',
    fontSize: 30,
  },
  red: {
    color: 'red',
  },
})

        # 尺寸是像素无关的

        在 React Native 中尺寸是没有单位的，它代表了设备独立像素。

        <View style={ {width:100,height:100,margin:40,backgroundColor:'gray'}}>
        <Text style={ {fontSize:16,margin:20}}>尺寸</Text>
</View>

        上述代码，运行在Android上时，View的长和宽被解释成：100dp 100dp单位是dp，字体被解释成16sp 单位是sp，运行在iOS上时尺寸单位被解释称了pt，这些单位确保了布局在任何不同dpi的手机屏幕上显示不会发生改变；

        # 屏幕适应

        在屏幕尺寸与设计稿的比例上, 有以下公式:

        设计稿元素宽度（px） / 设计稿总宽度（px） = 元素的宽度（dp） / 屏幕的总宽度（dp）

        进而求得

        元素的宽度（dp） = 设计稿元素宽度（px）* 屏幕的总宽度（dp） / 设计稿总宽度（px）

        于是可以写出转换库:

        import {Dimensions} from "react-native";

// 设备宽度，单位 dp
const deviceWidthDp = Dimensions.get('window').width;

// 设计稿宽度，单位 px
const uiWidthPx = 750;

export default function (px) {
  return px * deviceWidthDp / uiWidthPx;
}

        使用的时候:

        import px2dp from '../utils/px2dp';
import {StyleSheet} from 'react-native';

const style = StyleSheet.create({
  item: {
    width: px2dp(700)
  }
});

        附:

        • PixelRatio.get() 获取设备像素比
        • Dimensions.get('window').width 获取设备的宽度
        • Dimensions.get('window').height 获取设备的高度

        # flex布局

        容器属性

        • flexDirection enum(‘row’, ‘column’,’row-reverse’,’column-reverse’)
        • flexWrap enum(‘wrap’, ‘nowrap’)
        • justifyContent enum(‘flex-start’, ‘flex-end’, ‘center’, ‘space-between’, ‘space-around’)
        • alignItems enum(‘flex-start’, ‘flex-end’, ‘center’, ‘stretch’)

        元素属性

        • alignSelf enum(‘auto’, ‘flex-start’, ‘flex-end’, ‘center’, ‘stretch’)
        • flex number

        # React Native中的FlexBox 和Web CSSS上FlexBox的不同之处

        React Native中的FlexBox 和Web CSSS上FlexBox工作方式是一样的。但有些地方还是有些出入的

        • flexDirection: React Native中默认为flexDirection:'column'，在Web CSS中默认为flex-direction:'row'
        • alignItems: React Native中默认为alignItems:'stretch'，在Web CSS中默认align-items:'flex-start'
        • flex: 相比Web CSS的flex接受多参数，如:flex: 2 2 10%;，但在 React Native中flex只接受一个参数
        • 不支持的属性：align-content，flex-basis，order，flex-basis，flex-flow，flex-grow，flex-shrink

        # 颜色

        # 红-绿-蓝

        React Native 支持 rgb() 和 rgba() 两种十六进制与函数方法

        • '#f0f' (#rgb)
        • '#ff00ff' (#rrggbb)
        • 'rgb(255, 0, 255)'
        • 'rgba(255, 255, 255, 1.0)'
        • '#f0ff' (#rgba)
        • '#ff00ff00' (#rrggbbaa)

        # 色调-饱和度-亮度

        也支持 hsl() 和 hsla() 函数方法：

        • 'hsl(360, 100%, 100%)'
        • 'hsla(360, 100%, 100%, 1.0)'

        # 透明度

        rgba(0,0,0,0) 的快捷方式是：'transparent'
