# RN第三方库 - 视频

# react-native-video

使用 react-native-video 像操作 H5 的 video标签 一样操作 RN 的视频, 不过更强大。

# 注意事项

对版本的要求 (重要)

  1. Version 4.x (ExoPlayer 2.9.0)
  • react-native >= 0.57.0 (默认 Gradle 3 & SDK 27)
  • Android target SDK 26+
  • Gradle 3 plugin
  1. Version 3.x
  • react-native >= 0.40.0

If you need to support an older React Native version, you should use react-native-video 3.2.1.

# 安装

$ yarn add react-native-video
$ react-native link react-native-video

也可手动配置:

在 android/settings.gradle 中:

// 使用 ExoPlayer (新)
include ':react-native-video'
project(':react-native-video').projectDir = new File(rootProject.projectDir, '../node_modules/react-native-video/android-exoplayer')

// 使用 Android MediaPlayer (旧)
include ':react-native-video'
project(':react-native-video').projectDir = new File(rootProject.projectDir, '../node_modules/react-native-video/android')

在 android/app/build.gradle 中

dependencies {
  compile project(':react-native-video')
}

在 MainActivity.java (react-native < 0.29) 或 MainApplication.java (react-native >= 0.29) 中:

//...
import com.brentvatne.react.ReactVideoPackage;
//...
@Override
protected List<ReactPackage> getPackages() {
  return Arrays.<ReactPackage>asList(
    new MainReactPackage(),
    new ReactVideoPackage()
  );
}

# 使用

import React, { Component } from 'react'
import { StyleSheet } from 'react-native'
import { View } from 'react-native'
import Video from "react-native-video";

export default class App extends Component {
    constructor (props) {
        super(props)
        this.state={
            player: null
        }
    }
    render() {
        return (
            <View style={{flex: 1, justifyContent: 'space-evenly', alignItems: 'center', marginVertical: 24}}>
                <Video source={{uri: "https://xxx.com/xxx.mp4"}}   // Can be a URL or a local file.
                    controls={true}
                    style={styles.backgroundVideo}
                    ref={(ref) => {
                      this.player = ref
                    }}
                />
            </View>
        )
    }
}

var styles = StyleSheet.create({
    backgroundVideo: {
        position: 'absolute',
        top: 0,
        left: 0,
        bottom: 0,
        right: 0,
    },
});

# 配置属性

  • controls 是否显示控制条

  • source 播放源, 可为本地地址或网络地址 {uri: ''}, 详见: source

  • repeat 是否重复播放, 默认 false

  • resizeMode 拉伸模式

    • "none" (default) - 不拉伸
    • "contain"
    • "cover"
    • "stretch"

  • rate 播放速度

    • 0.0 - Pauses the video
    • 1.0 - Play at normal speed
    • Other values - Slow down or speed up playback

  • volume 音量大小

    • 1.0 (default) - Play at full volume
    • 0.0 - Mute the audio
    • Other values - Reduce volume

  • muted 是否静音

    • false (default) - Don't mute audio
    • true - Mute audio

  • paused 是否暂停

    • false (default) - Don't pause the media
    • true - Pause the media

  • audioOnly 仅仅播放声音

    • false (default) - Display the video as normal
    • true - Show the poster and play the audio
    • For this to work, the poster prop must be set.

  • progressUpdateInterval onProgress 事件触发的频率, 单位毫秒, 默认 250.0

  • poster 视频加载时显示的图像

  • posterResizeMode 加载时图像拉伸效果

    • "contain" (default) - Scale the image uniformly (maintain the image's aspect ratio) so that both dimensions (width and height) of the image will be equal to or less than the corresponding dimension of the view (minus padding).
    • "center" - Center the image in the view along both dimensions. If the image is larger than the view, scale it down uniformly so that it is contained in the view.
    • "cover" - Scale the image uniformly (maintain the image's aspect ratio) so that both dimensions (width and height) of the image will be equal to or larger than the corresponding dimension of the view (minus padding).
    • "none" - Don't apply resize
    • "repeat" - Repeat the image to cover the frame of the view. The image will keep its size and aspect ratio. (iOS only)
    • "stretch" - Scale width and height independently, This may change the aspect ratio of the src.

  • headers 设置请求头, 比如:

source={{
  uri: "https://www.example.com/video.mp4",
  headers: {
    Authorization: 'bearer some-token-value',
    'X-Custom-Header': 'some value'
  }
}}
  • playInBackground 程序在后台运行时是否继续播放
    • false (default) - Don't continue playing the media
    • true - Continue playing the media
    • To use this feature on iOS, you must: Enable Background Audio in your Xcode project. Set the ignoreSilentSwitch prop to "ignore"

# 事件属性

  • onLoadStart(payload) 视频开始加载时调用
    • payload.isNetwork
    • payload.type
    • payload.uri
  • onLoad(payload) 视频初始化完毕准备进行播放时调用
    • payload.currentPosition 当前位置
    • payload.duration 持续时长
    • payload.naturalSize 视频尺寸 {width, height, orientation}
  • onProgress(payload) 播放时调用, 调用频率为 progressUpdateInterval
    • payload.currentTime 当前位置
    • payload.playableDuration 已加载时长
    • payload.seekableDuration 总时长
  • onSeek(payload) 全部视频加载完毕时调用
    • payload.currentTime 当前播放时长
    • payload.seekTime 总时长
  • onEnd 视频播放完成调用

# 实例方法

调用实例方法需要通过指定 ref 调用。

  • seek(seconds) 跳转到指定位置
  • seek(seconds, tolerance)
  • save() 将视频保存到相册
