                  # 巧妙的文字特效

                  # 镂空文字特效

                  示例

                  -webkit-text-stroke: 1px red;
-webkit-text-fill-color: transparent;
                  Hello world

                  # 渐变文字特效

                  示例

                  background-image:-webkit-linear-gradient(#eee,#000);
-webkit-background-clip:text;
-webkit-text-fill-color:transparent;
-webkit-text-stroke:1px transparent;
                  Hello world

                  示例

                  background: linear-gradient(to bottom,blue,red);
-webkit-text-fill-color: transparent;
-webkit-background-clip: text;
                  Hello world

                  # 文字阴影特效

                  示例

                  text-shadow:4px 4px 2px #ccc;
                  Hello world

                  示例: 多重阴影

                  text-shadow:4px 4px 2px #ccc, -4px -4px 2px #fcc;
                  Hello world

                  示例: Glow and Extra Glow effect(也就是NEON effect)

                  text-shadow: 0 0 20px red;
                  Hello world

                  # 透明文字特效

                  示例

                  background: pink;
color: rgba(0,0,0,0.1);
opacity: 0.1;
                  Hello world

                  # 模糊文字特效

                  示例

                  text-shadow: 0 0 5px red;
-webkit-text-fill-color: transparent;
                  Hello world

                  # 图片背景文字

                  示例

                  height: auto;
width: auto;
background: url(true.jpg) repeat;
background-origin: border-box;
background-size: contain;
-webkit-background-clip: text;
-webkit-text-fill-color: transparent;
                  Hello world

                  # CSS3实现文字流光渐变特效

                  <div class="masked">
    <h4>日暮苍山远，天寒白屋贫。柴门闻犬吠，风雪夜归人。</h4>
</div>

                  .masked h4{
    display: block;
    width: 600px;
    height: 100px;
    /*渐变背景*/
    background-image: -webkit-linear-gradient(left, #3498db, #f47920 10%, #d71345 20%, #f7acbc 30%,
    #ffd400 40%, #3498db 50%, #f47920 60%, #d71345 70%, #f7acbc 80%, #ffd400 90%, #3498db);
    color: transparent; /*文字填充色为透明*/
    -webkit-text-fill-color: transparent;
    -webkit-background-clip: text;          /*背景剪裁为文字，只将文字显示为背景*/
    background-size: 200% 100%;            /*背景图片向水平方向扩大一倍，这样background-position才有移动与变化的空间*/
    /* 动画 */
    animation: masked-animation 4s infinite linear;
}
@keyframes masked-animation {
    0% {
        background-position: 0 0;   /*background-position 属性设置背景图像的起始位置。*/
    }
    100% {
        background-position: -100% 0;
    }
}

                  日暮苍山远，天寒白屋贫。柴门闻犬吠，风雪夜归人。
