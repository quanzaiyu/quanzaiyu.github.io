# 字体图标

  • react-native-vector-icons

# 安装与配置

安装：

$ yarn add react-native-vector-icons

自动配置：

$ react-native link react-native-vector-icons

# 手动配置

针对于 Android, 还需要一些额外的配置

android/settings.gradle

include ':react-native-vector-icons'
project(':react-native-vector-icons').projectDir = new File(rootProject.projectDir, '../node_modules/react-native-vector-icons/android')

android/app/build.gradle

project.ext.vectoricons = [
    iconFontNames: [ 'MaterialIcons.ttf', 'EvilIcons.ttf' ] // 需要拷贝的字体文件, 非必须配置, 可手动拷贝
]
apply from: "../../node_modules/react-native-vector-icons/fonts.gradle"

dependencies {
    implementation project(':react-native-vector-icons')
}

# 拷贝字体

如果不配置 project.ext.vectoricons 节点, 执行 link 则会自动将所有字体拷贝到 android/app/src/main/assets/fonts, 配置的话, 只拷贝指定的字体

如果没有自动拷贝成功, 可以手动将 node_modules/react-native-vector-icons/Fonts 下的字体 (ttf) 拷贝到 android/app/src/main/assets/fonts

react-native-vector-icons 关于 Android 的配置, 详见 https://github.com/oblador/react-native-vector-icons#android

# 使用内置字体图标

内置字体包括：

import Icon from 'react-native-vector-icons/dist/Ionicons';
// or # import Icon from 'react-native-vector-icons/Ionicons';
<Icon name='ios-settings' size={16} color="#f00"/>
  • name：图标名
  • size：图标大小
  • color：图标颜色

# 解析字体

安装 fonttools （使用到python）

$ pip install fonttools

下载或克隆 react-native-iconfont-mapper，并将 ttf 文件放于此目录下

使用命令解析字体：

$ python iconfont-mapper.py iconfont.ttf IconfontMap.js

可以看到目录中多了一个 IconfontMap.js

# 正则替换

打开 IconfontMap.js，使用正则将里面的字符串类型的数字替换为真正的数字：

  • from："(\d*)"
  • to：$1

替换后的结构为：

let map = {
  "shouye": 58880,
  "wode": 588801
}

;module.exports = (name)=>String.fromCharCode(map[name]);
;module.exports.map = map;

完事后将 iconfont.ttf 拷贝到 android/app/src/main/assets/fonts

# 编写字体库

创建 Iconfont.js

"use strict";
import { createIconSet } from 'react-native-vector-icons'
import { map as IconfontMap } from './IconfontMap.js'

export const iconSet = createIconSet(IconfontMap, 'iconfont', 'iconfont.ttf')

export const Button = iconSet.Button;
export const TabBarItem = iconSet.TabBarItem;
export const TabBarItemIOS = iconSet.TabBarItemIOS;
export const ToolbarAndroid = iconSet.ToolbarAndroid;
export const getImageSource = iconSet.getImageSource;

export default iconSet

# 使用

import Iconfont from '../../assets/fonts/Iconfont'

<Iconfont
  name='shouye'
  size={36}
  style={{color: tintColor}}
/>
