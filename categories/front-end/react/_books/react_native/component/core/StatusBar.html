# 状态栏

# StatusBar

StatusBar 是控制应用状态栏的组件。

<StatusBar backgroundColor={'#f00'} />

# 常量

  • currentHeight (仅限Android)状态栏的当前高度。

# 属性

  • animated bool 指定状态栏的变化是否应以动画形式呈现。目前支持这几种样式：backgroundColor, barStyle和hidden。
  • barStyle enum<StatusBarStyle> 设置状态栏文本的颜色。
    • enum('default', 'light-content', 'dark-content')
    • default 默认的样式（IOS为白底黑字、Android为黑底白字）
    • light-content 黑底白字
    • dark-content 白底黑字（需要Android API>=23）
  • hidden bool 是否隐藏状态栏。
  • backgroundColor color 状态栏的背景色。
  • translucent bool 指定状态栏是否透明。设置为true时，应用会延伸到状态栏之下绘制（即所谓“沉浸式”——被状态栏遮住一部分）。常和带有半透明背景色的状态栏搭配使用。
  • networkActivityIndicatorVisible bool 指定是否显示网络活动提示符。
  • showHideTransition enum<StatusBarAnimation> 通过hidden属性来显示或隐藏状态栏时所使用的动画效果。默认值为'fade'。
    • enum('fade', 'slide')
    • none 没有动画
    • fade 渐变效果
    • slide 滑动效果

# 方法

  • setHidden(hidden: boolean, [animation]: StatusBarAnimation) 显示／隐藏状态栏
  • setBarStyle(style: StatusBarStyle, [animated]: boolean) 设置状态栏的样式
  • setNetworkActivityIndicatorVisible(visible: boolean) 显示／隐藏网络活动指示器
  • setBackgroundColor(color: string, [animated]: boolean) 设置状态栏的背景色
  • setTranslucent(translucent: boolean) 指定状态栏是否透明。设置为true时，应用会在状态栏之下绘制（即所谓“沉浸式”——被状态栏遮住一部分）。常和带有半透明背景色的状态栏搭配使用。

# 沉浸式APP

如果想要整个APP实现沉浸式，只需要在 App.js（根入口组件）中加入：

<StatusBar backgroundColor="#ff0000"
            translucent={true}
            hidden={true}
            animated={true}
/>

比如：

import React, {Component} from 'react';
import {createAppContainer} from "react-navigation"
import AppNavigations from './src/navigators/AppNavigations'
import {Provider} from "react-redux"
import store from './src/stores'
import {StatusBar} from "react-native"

let Navigation = createAppContainer(AppNavigations);

type Props = {};
export default class App extends Component<Props> {
  render() {
    return (
      <Provider store={store}>
        <StatusBar backgroundColor="#ff0000"
                   translucent={true}
                   hidden={true}
                   animated={true}
        />
        <Navigation />
      </Provider>
    );
  }
}
