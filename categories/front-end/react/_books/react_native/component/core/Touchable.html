# 可点击组件

提示

除了以下可点击组件外，其余普通组件通常不支持点击事件（onPress），如果要使普通的View可点击，可在外部套一层Touchable

提示

Touchable组件只支持一个子元素

# Button

这是一个简单的 Button 示例:

import { Button } from 'react-native';
...

<Button
  onPress={() => { alert('hello') }}
  title="Click me"
  color="#841584"
  accessibilityLabel="Please click this button"
/>

# 属性

  • title string 按钮内显示的文本
  • accessibilityLabel string 用于给残障人士显示的文本（比如读屏应用可能会读取这一内容）
  • color color 文本的颜色(iOS)，或是按钮的背景色(Android)
  • disabled bool 设置为 true 时此按钮将不可点击

# 事件

  • onPress function 用户点击此按钮时所调用的处理函数
  • onLongPress function 用户长按此按钮时所调用的处理函数

# TouchableOpacity

可以通过 TouchableOpacity 封装一个自定义的按钮, 当按下的时候，封装的视图的不透明度会降低。

import {Text, View, TouchableOpacity} from 'react-native';

...

<TouchableOpacity
  ref={el => this.btn = el}
  activeOpacity={.5}
  style={style.button}
  onPress={() => { this.btn.setOpacityTo(1, 1000) }}
>
  <Text style={style.buttonText}>Click me</Text>
</TouchableOpacity>

...

const style = StyleSheet.create({
  button: {
    justifyContent: 'center',
    alignItems: 'center',
    height: 40,
    width: 100,
    borderRadius: 20,
    backgroundColor: '#c35',
  },
  buttonText: {
    color: '#fff'
  }
})

效果: 点击后, 按钮透明度降到 0.5, 过 1000ms 后透明度回复到 1

props:

  • onPress function 用户点击此按钮时所调用的处理函数
  • onLongPress function 用户长按此按钮时所调用的处理函数
  • style style 样式
  • activeOpacity number 指定封装的视图在被触摸操作激活时以多少不透明度显示（0到1之间）。默认值为0.2。

方法:

  • setOpacityTo((value: number), (duration: number)) 将本组件的不透明度设为指定值（伴有过渡动画）, 第一个参数是透明度, 第二个参数是持续多少毫秒。

# TouchableHighlight

可以通过 TouchableHighlight 封装一个自定义的按钮, 当按下的时候，封装的视图的不透明度会降低，同时会有一个底层的颜色透过而被用户看到，使得视图变暗或变亮。

import {Text, TouchableHighlight} from 'react-native';

...

<TouchableHighlight
  style={style.button}
  activeOpacity={0.5}
  underlayColor='green'
  onHideUnderlay={()=>{
    console.log('onHideUnderlay')
  }}
  onShowUnderlay={()=>{
    console.log('onShowUnderlay')
  }}
  onPress={()=>{
    console.log('onPress')
  }}
  onLongPress={()=>{
    console.log('onLongPress')
  }}
>
  <Text style={style.buttonText}>
    Click me
  </Text>
</TouchableHighlight>

# 属性

  • activeOpacity number 指定封装的视图在被触摸操作激活时以多少不透明度显示（0到1之间，默认值为0.85）。需要设置underlayColor
  • underlayColor color 有触摸操作时显示出来的底层的颜色
  • style style 样式

# 事件

  • onPress function 用户点击此按钮时所调用的处理函数
  • onLongPress function 用户长按此按钮时所调用的处理函数
  • onHideUnderlay function 底层的颜色被隐藏的时候调用
  • onShowUnderlay function 当底层的颜色被显示的时候调用

# TouchableNativeFeedback

可以通过 TouchableNativeFeedback (仅 Android) 封装一个自定义的按钮, 当按下的时候，封装的视图将带来一些反馈(比如涟漪)。

import {Text, View, TouchableNativeFeedback} from 'react-native';

...

<TouchableNativeFeedback
  background={TouchableNativeFeedback.Ripple('#ccc', false)}>
  <View style={style.button}>
    <Text style={style.buttonText}>Click me</Text>
  </View>
</TouchableNativeFeedback>

# 属性

  • background backgroundPropType 决定在触摸反馈的时候显示什么类型的背景。

background 接收的方法:

  • SelectableBackground()
  • SelectableBackgroundBorderless()
  • Ripple(color: string, borderless: boolean)

# 事件

  • onPress function 用户点击此按钮时所调用的处理函数
  • onLongPress function 用户长按此按钮时所调用的处理函数

另外还有一个与之相反的组件 TouchableWithoutFeedback, 它将不会提供任何视觉反馈, 不建议使用。
