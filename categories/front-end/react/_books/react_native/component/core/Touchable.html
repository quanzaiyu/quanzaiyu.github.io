# 可点击组件

提示

除了以下可点击组件外，其余普通组件通常不支持点击事件（onPress），如果要使普通的View可点击，可在外部套一层Touchable

提示

Touchable组件只支持一个子元素

# Button

这是一个简单的 Button 示例:

import { Button } from 'react-native';
...

<Button
  onPress={() => { alert('hello') }}
  title="Click me"
  color="#841584"
  accessibilityLabel="Please click this button"
/>

# 属性

  • title string 按钮内显示的文本
  • accessibilityLabel string 用于给残障人士显示的文本（比如读屏应用可能会读取这一内容）
  • color color 文本的颜色(iOS)，或是按钮的背景色(Android)
  • disabled bool 设置为 true 时此按钮将不可点击

# 事件

  • onPress function 用户点击此按钮时所调用的处理函数
  • onLongPress function 用户长按此按钮时所调用的处理函数

# TouchableOpacity

可以通过 TouchableOpacity 封装一个自定义的按钮, 当按下的时候，封装的视图的不透明度会降低。

import {Text, View, TouchableOpacity} from 'react-native';

...

<TouchableOpacity
  ref={el => this.btn = el}
  activeOpacity={.5}
  style={style.button}
  onPress={() => { this.btn.setOpacityTo(1, 1000) }}
>
  <Text style={style.buttonText}>Click me</Text>
</TouchableOpacity>

...

const style = StyleSheet.create({
  button: {
    justifyContent: 'center',
    alignItems: 'center',
    height: 40,
    width: 100,
    borderRadius: 20,
    backgroundColor: '#c35',
  },
  buttonText: {
    color: '#fff'
  }
})

效果: 点击后, 按钮透明度降到 0.5, 过 1000ms 后透明度回复到 1

props:

  • onPress function 用户点击此按钮时所调用的处理函数
  • onLongPress function 用户长按此按钮时所调用的处理函数
  • style style 样式
  • activeOpacity number 指定封装的视图在被触摸操作激活时以多少不透明度显示（0到1之间）。默认值为0.2。

方法:

  • setOpacityTo((value: number), (duration: number)) 将本组件的不透明度设为指定值（伴有过渡动画）, 第一个参数是透明度, 第二个参数是持续多少毫秒。

# TouchableHighlight

可以通过 TouchableHighlight 封装一个自定义的按钮, 当按下的时候，封装的视图的不透明度会降低，同时会有一个底层的颜色透过而被用户看到，使得视图变暗或变亮。

import {Text, TouchableHighlight} from 'react-native';

...

<TouchableHighlight
  style={style.button}
  activeOpacity={0.5}
  underlayColor='green'
  onHideUnderlay={()=>{
    console.log('onHideUnderlay')
  }}
  onShowUnderlay={()=>{
    console.log('onShowUnderlay')
  }}
  onPress={()=>{
    console.log('onPress')
  }}
  onLongPress={()=>{
    console.log('onLongPress')
  }}
>
  <Text style={style.buttonText}>
    Click me
  </Text>
</TouchableHighlight>

# 属性

  • activeOpacity number 指定封装的视图在被触摸操作激活时以多少不透明度显示（0到1之间，默认值为0.85）。需要设置underlayColor
  • underlayColor color 有触摸操作时显示出来的底层的颜色
  • style style 样式

# 事件

  • onPress function 用户点击此按钮时所调用的处理函数
  • onLongPress function 用户长按此按钮时所调用的处理函数
  • onHideUnderlay function 底层的颜色被隐藏的时候调用
  • onShowUnderlay function 当底层的颜色被显示的时候调用

# TouchableNativeFeedback

可以通过 TouchableNativeFeedback (仅 Android) 封装一个自定义的按钮, 当按下的时候，封装的视图将带来一些反馈(比如涟漪)。

import {Text, View, TouchableNativeFeedback} from 'react-native';

...

<TouchableNativeFeedback
  background={TouchableNativeFeedback.Ripple('#ccc', false)}>
  <View style={style.button}>
    <Text style={style.buttonText}>Click me</Text>
  </View>
</TouchableNativeFeedback>

# 属性

  • background backgroundPropType 决定在触摸反馈的时候显示什么类型的背景。

background 接收的方法:

  • SelectableBackground()
  • SelectableBackgroundBorderless()
  • Ripple(color: string, borderless: boolean)

# 事件

  • onPress function 用户点击此按钮时所调用的处理函数
  • onLongPress function 用户长按此按钮时所调用的处理函数

另外还有一个与之相反的组件 TouchableWithoutFeedback, 它将不会提供任何视觉反馈, 不建议使用。

# 封装

有了以上基础，我们就可以定制属于自己的按钮了，下面示例，我将封装两个组件：Touchable和Button

Touchable 是为了区分iOS和Android平台，因为本人更喜欢 TouchableNativeFeedback，而iOS并不支持

Button 是一个可扩展的按钮组件，定制灵活，底层实现为 Touchable

components/Touchable.js

import {Platform, TouchableHighlight, TouchableNativeFeedback} from "react-native"
import React from "react"

/*
* 封装的可触摸组件
* props：
* touchableProps 触摸属性
*   onPress 点击事件
* content 需要渲染的内容
*
* */
export default class Touchable extends React.Component {
  render() {
    let { touchableProps, content } = this.props
    return (
      Platform.OS === "ios" ?
        <TouchableHighlight
          { ...touchableProps }
          activeOpacity={0.5}
          underlayColor='#ccc'
        >
          { content }
        </TouchableHighlight> :
        <TouchableNativeFeedback
          { ...touchableProps }
          background={TouchableNativeFeedback.Ripple('#ccc', false)}
        >
          { content }
        </TouchableNativeFeedback>
    )
  }
}

components/Button.js

import {Text, View} from "react-native"
import {rem} from "../utils/px2dp"
import React from "react"
import Touchable from './Touchable'

/*
* 封装的按钮组件
* props：
* touchableProps 触摸属性
*   onPress 点击事件
* styleProps 样式属性
*   btn 按钮的样式
*   text 文字的样式
* text 文字内容
* type 按钮类型
*   normal 普通按钮
*   plain 自定义按钮
*
* */
export default class Button extends React.Component {
  render() {
    let { touchableProps = {}, styleProps = {}, text = '', type = 'normal' } = this.props
    if (type === 'normal') {
      styleProps = {
        btn: [styles.btn, ...(styleProps.btn || [])],
        text: [styles.text, ...(styleProps.text || [])]
      }
    }
    return (
      <Touchable
        touchableProps={touchableProps}
        content={
          <View style={styleProps.btn}>
            <Text style={styleProps.text}>{text}</Text>
          </View>
        }
      ></Touchable>
    )
  }
}

const styles = {
  btn: {
    textAlign: 'center',
    alignItems: 'center',
    justifyContent: 'center',
    backgroundColor: '#1ABC9C',
    paddingLeft: rem(.47),
    borderRadius: rem(1),
  },
  text: {
    fontSize: rem(.53),
    color: '#fff',
  }
}

使用的时候：

<Touchable
  touchableProps={{
    onPress: () => { navigation.goBack() }
  }}
  content={ renderRight() }
></Touchable>

<Button
  touchableProps={{
    onPress: () => this.getCode()
  }}
  styleProps={{
    btn: [{marginRight: rem(.39)}],
    text: [{color: '#1ABC9C', fontSize: rem(.37)}]
  }}
  text={'获取验证码'}
  type={'plain'}
></Button>

...

<Button
  touchableProps={{
    onPress: () => this.goHome()
  }}
  styleProps={{
    btn: [{
      width: rem(8),
      height: rem(1.2),
      marginTop: rem(1.15)
    }]
  }}
  text={'登录'}
></Button>
