        # React 组件插槽

        # 默认插槽

        暂且使用 "组件插槽" 这个词汇表述, 在 React 官网中叫 "包含关系", 这里沿用 Vue 的叫法。

        在React中默认插槽使用 props.children 来接收:

        function FancyBorder(props) {
  return (
    <div className={'FancyBorder FancyBorder-' + props.color}>
      {props.children}
    </div>
  );
}

class App extends React.Component {
  render() {
    return (
      <FancyBorder color="blue">
        <h1 className="Dialog-title">
          Welcome
        </h1>
        <p className="Dialog-message">
          Thank you for visiting our spacecraft!
        </p>
      </FancyBorder>
    );
  }
}

        # 具名插槽

        如果有多个插槽的话可以在子组件声明要接收的插槽名字, 在父组件中通过属性传入:

        function SplitPane(props) {
  return (
    <div className="SplitPane">
      <div className="SplitPane-left">
        {props.left}
      </div>
      <div className="SplitPane-right">
        {props.right}
      </div>
    </div>
  );
}

class App extends React.Component {
  render() {
    return (
      <SplitPane
      left={
        <div>Left</div>
      }
      right={
        <div>Right</div>
      } />
    );
  }
}

        # 默认插槽与具名插槽共存

        默认插槽和具名插槽可以同时存在:

        function SplitPane(props) {
  return (
    <div className="SplitPane">
      <div className="SplitPane-left">
        {props.left}
      </div>
      <div>
        {props.children}
      </div>
      <div className="SplitPane-right">
        {props.right}
      </div>
    </div>
  );
}

class App extends React.Component {
  render() {
    return (
      <SplitPane
      left={
        <div>Left</div>
      }
      right={
        <div>Right</div>
      }>
        <div>Hello world</div>
      </SplitPane>
    );
  }
}
