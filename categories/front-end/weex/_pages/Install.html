      # weex 的安装与运行

      # weex toolkit 的安装与配置

      安装:

      $ yarn global add weex-toolkit

      测试:

      $ weex

      创建weex项目:

      $ weex create weex-app

      运行到浏览器:

      $ weex run web

      # Android 环境配置

      1. 下载并安装JDK、Android Studio、Android SDK
      2. 配置环境变量, 比如:
      • JAVA_HOME: C:\Program Files\Java\jdk1.8.0_201
      • ANDROID_HOME: D:\Software\android-sdk
      • PATH:
        • %JAVA_HOME%\bin
        • %ANDROID_HOME%\tools
        • %ANDROID_HOME%\platform-tools
      1. 添加安卓平台
      $ weex platform add android
