# 坐标系统

在使用Canvas坐标变换之前，需要先了解Canvas的坐标系统。详细请参考 Canvas坐标系

Canvas 的坐标系统并不是一尘不变的。可以对 Canvas 坐标系统进行移动、旋转和缩放等操作。而这些操作被称为坐标变换。

坐标变换包括: translate、rotate、scale
