# 基础标签

# svg 标签

SVG 代码必须放在顶层标签的 svg 标签里面，比如：

<svg viewBox="0 0 500 500" width="100%" height="100%">
  <circle id="mycircle" cx="50%" cy="50%" r="50" />
</svg>

效果如下：

  • width 属性和 height 属性，指定了 SVG 图像在 HTML 元素中所占据的宽度和高度。可以使用绝对长度也可以使用相对长度。
  • viewBox 属性的值有四个数字，分别是左上角的横坐标和纵坐标、视口的宽度和高度。如果横纵坐标不为0，可以对图像做裁剪。

# circle 标签

circle 标签代表圆形，上面例子已经有所表现，说一下具体参数的意义：

  • cx、cy、r 属性分别为横坐标、纵坐标和半径，单位为像素。坐标都是相对于 svg 画布的左上角原点。

# line 标签

line 标签用来绘制线段。

<div style='width:100%;height:200px;'>
<svg viewBox="0 0 500 500" width="100%" height="100%" >
  <line x1="100" y1="100" x2="200" y2="200" />
</svg>
</div>

效果如下：

  • x1 属性和 y1 属性，表示线段起点的横坐标和纵坐标
  • x2 属性和 y2 属性，表示线段终点的横坐标和纵坐标

# polyline 标签

polyline 标签用于绘制一根折线。

<div style='width:100%;height:200px;'>
<svg viewBox="0 0 500 500" width="100%" height="100%" >
  <polyline points="10,10 110,110, 50,250" fill="none" stroke="green" />
</svg>
</div>

效果如下：

  • points 属性指定了每个端点的坐标，横坐标与纵坐标之间与逗号分隔，点与点之间用空格分隔。

# rect 标签

rect 标签用于绘制矩形。

<div style='width:100%;height:200px;'>
<svg viewBox="0 0 500 500" width="100%" height="100%" >
  <rect x="0" y="0" height="100" width="200" style="stroke: #70d5dd; fill: #dd524b" />
</svg>
</div>

效果如下：

  • x 属性和 y 属性，指定了矩形左上角端点的横坐标和纵坐标；
  • width 属性和 height 属性指定了矩形的宽度和高度（单位像素）。

# ellipse 标签

ellipse 标签用于绘制椭圆。

<div style='width:100%;height:200px;'>
<svg viewBox="0 0 500 500" width="100%" height="100%" >
  <ellipse cx="100" cy="150" ry="100" rx="200" stroke="black" stroke-width="5" fill="silver"/>
</svg>
</div>

效果如下：

  • cx属性和cy属性，指定了椭圆中心的横坐标和纵坐标（单位像素）；
  • rx属性和ry属性，指定了椭圆横向轴和纵向轴的半径（单位像素）。

# polygon 标签

polygon 标签用于绘制多边形。

<div style='width:100%;height:200px;'>
<svg viewBox="0 0 500 500" width="100%" height="100%" >
  <polygon fill="green" stroke="orange" stroke-width="1" points="0,0 100,0 200,100 0,200 0,0"/>
</svg>
</div>

效果如下：

points属性指定了每个端点的坐标，横坐标与纵坐标之间与逗号分隔，点与点之间用空格分隔。

其中，最后一个点为多边形起始点，可以指定也可以不指定，如果不指定默认闭合，比如，以下两种写法效果一致：

<polygon fill="green" stroke="orange" stroke-width="1" points="0,0 100,0 200,100 0,200 0,0"/>
<polygon fill="green" stroke="orange" stroke-width="1" points="0,0 100,0 200,100 0,200"/>

# path 标签

path 标签用于制路径。

<div style='width:100%;height:200px;'>
<svg viewBox="0 0 500 500" width="100%" height="100%" >
  <path d="
    M 18,3
    L 46,3
    L 46,40
    L 61,40
    L 32,68
    L 3,40
    L 18,40
    Z
  "></path>
</svg>
</div>

效果如下

path 的 d 属性表示绘制顺序，它的值是一个长字符串，每个字母表示一个绘制动作，后面跟着坐标。

  • M：移动到（moveto）
  • L：画直线到（lineto）
  • Z：闭合路径

# text 标签

text 标签用于绘制文本。

<svg viewBox="0 0 100 100" width="100%" height="100%" >
  <text x="100" y="100" font-size='50px'>Hello World</text>
</svg>

效果如下：

x 属性和 y 属性，表示文本区块基线（baseline）起点的横坐标和纵坐标。

文字的样式可以用 class 或 style 属性指定，跟普通的 html 文字同样设置，也可直接写到属性值。

# image 标签

image 标签用于插入图片文件。

<svg viewBox="0 0 100 100" width="100%" height="100%" >
  <image xlink:href="/logo.png" width="50%" height="50%"/>
</svg>

效果如下：

image 标签使用 xlink:href 属性表示图像的来源。
