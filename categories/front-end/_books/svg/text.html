# 文本

在SVG中有两种截然不同的文本模式. 一种是写在图像中的文本，另一种是SVG字体。现在我们主要集中前者：写在图像中的文本。在一个SVG文档中，元素内部可以放任何的文字。

一个简单的示例：

<svg width="100%" height="200" viewBox="0 0 250 80" version="1.1" xmlns="http://www.w3.org/2000/svg">
	<text x="100" y="50" font-weight="bold" fill="red" font-size="30" text-anchor="middle" stroke="black">Hello</text>
</svg>
  • 属性 x 和属性 y 决定了文本在视口中显示的位置
  • 属性 text-anchor，可以有这些值：start、middle、end 或 inherit，允许决定从这一点开始的文本流的方向
  • 属性 fill 可以给文本填充颜色
  • 属性 stroke 可以给文本描边

形状元素和文本元素都可以引用渐变或图案, 相比较CSS2.1只能绘制简单的彩色文字，SVG显得更具有优势。

# 设置字体属性

文本的一个至关重要的部分是它显示的字体。SVG提供了一些属性，类似于它们的CSS同行，用来激活文本选区。下列每个属性可以被设置为一个SVG属性或者成为一个CSS声明：font-family、font-style、font-weight、font-variant、font-stretch、font-size、font-size-adjust、kerning、letter-spacing、word-spacing和text-decoration。

# tspan

tspan 元素用来标记大块文本的子部分，它必须是一个text元素或别的tspan元素的子元素。一个典型的用法是把句子中的一个词变成粗体红色。

<svg width="100%" height="200" viewBox="0 0 250 80" version="1.1" xmlns="http://www.w3.org/2000/svg">
	<text x="100" y="50">
		This is <tspan font-weight="bold" fill="red">bold</tspan>
	</text>
</svg>

# 路径文本

除了笔直地绘制一行文字以外， SVG 也可以根据 <path> 元素的形状来放置文字。 只要在 textPath 元素内部放置文本，并通过其 xlink:href 属性值引用 <path> 元素，我们就可以让文字块呈现在 <path> 元素给定的路径上了。

<svg width="100%" height="200" viewBox="0 0 500 300" version="1.1" xmlns="http://www.w3.org/2000/svg">
	<path id="my_path" d="M 100 200 Q 200 100 300 200 T 500 200" stroke="rgb(0,255,0)" fill="none" />
	<text font-size='30px'>
	  <textPath xlink:href="#my_path">This text follows a curve.</textPath>
	</text>
</svg>
  • startOffset 属性 确定排列起始
  • dx, dy 属性 切线和法线方向的偏移

# 超链接

可以添加到任意的图形上。

<svg width="100%" height="200" viewBox="0 0 300 100" version="1.1" xmlns="http://www.w3.org/2000/svg">
  <a xlink:href="#" xlink:title="打开简书"  target="_blank">
    <text x="100" y="50" fill="red" font-size='30px'>Click Me</text>
  </a>
</svg>
  • xlink:href 指定连接地址
  • xlink:title 指定连接提示
  • target 指定打开目标
