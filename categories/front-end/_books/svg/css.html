# CSS 属性

svg 画布同样可以使用 css 属性进行控制，不过与普通网页元素的 css 属性有所不同，svg 包括以下 css 属性：

  • fill：填充色
  • stroke：描边色
  • stroke-width：边框宽度
<svg viewBox="0 0 500 500" width="100%" height="100%">
  <circle id="mycircle" cx="50%" cy="50%" r="50" class="circle"/>
</svg>
<style>
.circle {
	fill: #f00;
	stroke: #00f;
	stroke-width: 2px;
}
</style>

效果如下：

这些 css 属性也可以直接写在标签的属性之中：

<svg viewBox="0 0 500 500" width="100%" height="100%">
	<circle id="mycircle" cx="50%" cy="50%" r="150" fill="#f00" stroke="#00f" stroke-width="2px"/>
</svg>
