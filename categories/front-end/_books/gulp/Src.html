# gulp.src

gulp.src 指定需要操作的源文件路径。

格式:

gulp.src(globs[, options])

其中 globs 为匹配模式， options 为匹配选项，后文中详细介绍。

# globs String Array

创建任务的时候路径使用匹配模式语法。

所要读取的 glob 或者包含 globs 的数组。

比如:

*.less 代表任何以 .less 结尾的文件

src/**/* 代表 src 目录下所有文件夹(包括子文件夹)下所有文件。

src/**/*.less 代表 src 目录下所有文件夹(包括子文件夹)下所有以 .less 结尾的文件。

参见: https://github.com/isaacs/node-glob

# options Object

options.buffer Boolean

默认值： true

如果该项被设置为 false，那么将会以 stream 方式返回 file.contents 而不是文件 buffer 的形式。这在处理一些大文件的时候将会很有用。

options.read Boolean

默认值： true

如果该项被设置为 false， 那么 file.contents 会返回空值（null），也就是并不会去读取文件。

options.base String

默认值： 将会加在 glob 之前
