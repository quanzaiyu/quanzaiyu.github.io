# 文件包含

包含（include）功能允许把另外的文件内容插入进来

# 简单使用

index.pug

doctype html
html
  include includes/head
  body
    h1 我的网站
    p 欢迎来到我这简陋得不能再简陋的网站。
    include includes/foot

includes/head.pug

head
  title 我的网站

includes/foot.pug

footer#footer
  p Copyright (c) foobar

index.pug的渲染结果 - index.html

<!DOCTYPE html>
<html>
  <head>
    <title>我的网站</title>
  </head>
  <body>
    <h1>我的网站</h1>
    <p>欢迎来到我这简陋得不能再简陋的网站。</p>
    <footer id="footer">
      <p>Copyright (c) foobar</p>
    </footer>
  </body>
</html>

# 包含非 pug 文件

被包含的如果不是 Pug 文件，那么就只会当作文本内容来引入

index.pug

doctype html
html
  head
    style
      include style.css
  body
    script
      include script.js

style.css

/* style */
* {
  margin: 0;
  padding: 0;
}

script.js

console.log(1)

index.pug的渲染结果 - index.html

<!DOCTYPE html>
<html>
  <head>
    <style>/* style */
* {
  margin: 0;
  padding: 0;
}
    </style>
  </head>
  <body>
    <script>console.log(1)</script>
  </body>
</html>
