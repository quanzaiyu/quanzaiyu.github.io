# 简介

Pug 原名不叫 Pug，原来是大名鼎鼎的 jade，后来由于商标的原因，改为 Pug，哈巴狗。

Pug 是根据缩进式语法，是一种 破坏式的、侵入式的、强依赖的模板引擎。

一旦使用 jade，便不能与传统的 HTML 共存。
