      # 在 React 中使用 Redux

      • react-redux - GitHub

      # 安装与使用

      如果是结合 React 使用, 可以选择性地安装以下包:

      $ yarn add redux react-redux
$ yarn add --dev redux-devtools

      修改 src/index.js

      import React from 'react'
import { render } from 'react-dom'
import { Provider } from 'react-redux'
import store from './store'
import App from './components/App'

render(
  <Provider store={store}>
    <App />
  </Provider>,
  document.getElementById('root')
)

      使用 React Redux 组件的 <Provider> 来让所有容器组件都可以访问 store，而不必显示地传递它。只需要在渲染根组件时使用即可。

      # 容器组件和展示组件

      容器组件（Smart/Container Components）和展示组件（Dumb/Presentational Components）

      展示组件 容器组件
      作用 描述如何展现（骨架、样式） 描述如何运行（数据获取、状态更新）
      直接使用 Redux 否 是
      数据来源 props 监听 Redux state
      数据修改 从 props 调用回调函数 向 Redux 派发 actions
      调用方式 手动 通常由 React Redux 生成

      # 在展示组件组件中使用

      在展示组件中使用需要用到 connect() 连接到 Redux, 通过 props 接收。

      用个官方的示例 Todo 为例

      import React from 'react'
import { connect } from 'react-redux'
import { addTodo } from '../actions'

class AddTodo extends React.Component {
  constructor (props) {
    super(props)
  }

  render() {
    let input
    let dispatch = this.props.dispatch
    return (
      <div>
        <form
          onSubmit={e => {
            e.preventDefault()
            if (!input.value.trim()) return
            dispatch(addTodo(input.value))
            input.value = ''
          }}
        >
          <input ref={node => { input = node }} />
          <button type="submit"> Add Todo </button>
        </form>
      </div>
    )
  }
}

export default connect()(AddTodo)

      # 在容器组件中使用

      容器组件中, 需要手动实现 mapStateToProps 和 mapDispatchToProps

      mapStateToProps 接收 state 作为参数, mapDispatchToProps 接收 dispatch 作为参数, 处理好数据之后可以通过参数的形式传入 connect(), 并在需要连接的下一个组件中接收

      import { connect } from 'react-redux'
import { toggleTodo } from '../actions'
import TodoList from '../components/TodoList'

const getVisibleTodos = (todos, filter) => {
  switch (filter) {
    case 'SHOW_COMPLETED':
      return todos.filter(t => t.completed)
    case 'SHOW_ACTIVE':
      return todos.filter(t => !t.completed)
    case 'SHOW_ALL':
    default:
      return todos
  }
}

const mapStateToProps = state => {
  return {
    todos: getVisibleTodos(state.todos, state.visibilityFilter)
  }
}

const mapDispatchToProps = dispatch => {
  return {
    onTodoClick: id => {
      dispatch(toggleTodo(id))
    }
  }
}

const VisibleTodoList = connect(
  mapStateToProps,
  mapDispatchToProps
)(TodoList)

export default VisibleTodoList

      以上将处理好的数据连接到 TodoList, 接收时可直接使用:

      import React from 'react'
import PropTypes from 'prop-types'
import Todo from './Todo'

const TodoList = ({ todos, onTodoClick }) => (
  <ul>
    {todos.map(todo => (
      <Todo key={todo.id} {...todo} onClick={() => onTodoClick(todo.id)} />
    ))}
  </ul>
)
export default TodoList

      详细的示例参考官网的 Todo示例

      # 直接使用

      其实, 在 React 中安装 react-redux 是可选的, 也可以直接使用, 不过得手动实现其 subscribe()

      import React from 'react';
import {
  View,
  Text,
  Button
} from 'react-native';
import store from '../stores'
import {setAppName} from '../stores/actions/appActions'

export default class App extends React.Component {
  constructor(props) {
    super(props);
    this.state = {
      a: 0,
      b: 1,
      app: 'app',
      ...store.getState()
    }
    // 准备一个监听器
    const changeListener = () => {
      // 当 store 中的状态发生变化时，
      // 将 store 中的新状态通过this.setState 设置到本组件,
      // 从而触发本组件被重新渲染输出
      this.setState({
        ...this.state,
        ...store.getState()
      });
    };
    // 监听Redux store中状态的变化
    store.subscribe(changeListener);
  }

  render() {
    return (
        <View>
          <Button onPress={() => {
            store.dispatch({
              type: 'SET_APP_NAME',
              text: 'RN_Test'
            })
          }} title={'SET_APP_NAME'}/>
          <Button onPress={() => {
            store.dispatch(setAppName('Hello world'))
          }} title={'SET_APP_NAME'}/>
          <Text>{this.state.app}</Text>
        </View>
    );
  }
}
