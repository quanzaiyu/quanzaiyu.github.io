    # 在 Vue 中使用 CKEditor

    CKEditor 是一款非常不错的富文本编辑器, 目前的版本为 5.x, 已经可以集成到Vue中, 方法也非常的简单

    安装

    $ yarn add @ckeditor/ckeditor5-vue @ckeditor/ckeditor5-build-classic

    在 main.js 中引入:

    import CKEditor from '@ckeditor/ckeditor5-vue';
Vue.use(CKEditor)

    在组件中使用:

    <template>
<div>
  <ckeditor style="height: 500px;"
    :editor="editor"
    v-model="textarea"
    :config="editorConfig"
  ></ckeditor>
</div>
</template>

<script>
import ClassicEditor from '@ckeditor/ckeditor5-build-classic';

export default {
  data () {
    return {
      textarea: '',
      editor: ClassicEditor,
      editorConfig: {
        language: "zh-cn",
        toolbar: [
          "heading", "|",
          "bold", "italic", "link", "bulletedList", "numberedList", "blockQuote", "|",
          "undo", "redo"
        ],
        placeholder: "请输入章节内容",
      },
    }
  }
}
</script>
