# 异步数据

asyncData 方法，使得我们可以在设置组件的数据之前能异步获取或处理数据。

asyncData 方法返回的数据在融合 data 方法返回的数据后，一并返回给模板进行展示。

以下示例, 创建了一个 php 文件返回相应数据:

echo json_encode([
    'msg' => 'ok'
]);

# axios 返回 async...await

<template lang='pug'>
.container
  div {{ datas }}
</template>

<script>
import axios from 'axios'
export default {
  async asyncData(parms) {
    let { data } = await axios.get('http://localhost')
    return { datas: data.msg }
  },
  created() {
    console.log(this.datas)
  }
}
</script>

# axios 返回 Promise

<template lang='pug'>
.container
  div {{ datas }}
</template>

<script>
import axios from 'axios'
export default {
  asyncData(parms) {
    return axios.get(`http://localhost`).then(res => {
      return { datas: res.data.msg }
    })
  },
  mounted() {
    console.log(this.datas)
  }
}
</script>

# axios 返回 回调函数

<template lang='pug'>
.container
  div {{ datas }}
</template>

<script>
import axios from 'axios'
export default {
  asyncData({ params }, callback) {
    axios.get('http://localhost').then(res => {
      callback(null, { datas: res.data.msg })
    })
  },
  created() {
    console.log(this.datas)
  }
}
</script>

# 获取参数请求数据

将 php 改为:

$id = $_REQUEST['id'];

echo json_encode([
  'msg' => 'hello ' . $id
]);

# 获取路由参数

<template lang='pug'>
.container
  div {{ title }}
</template>

<script>
import axios from 'axios'
export default {
  asyncData({ params, error }) {
    return axios
      .get(`http://localhost/?id=${params.id}`)
      .then(res => {
        return { title: res.data.msg }
      })
      .catch(e => {
        error({ statusCode: 404, message: 'Page not found' })
      })
  },
  created() {
    console.log(this.title)
  }
}
</script>

在 asyncData 中, params 可接收路由参数。

# 错误处理

如果请求中出错, 可以使用 error() 抛出一个错误:

import axios from 'axios'
export default {
  asyncData({ params, error }) {
    return axios.get(`http://localhost/?id=${params.id}`)
      .then(res => {
        return { title: res.data.msg }
      }).catch(e => {
        error({ statusCode: 404, message: 'Page not found' })
      })
  }
}

如果使用 回调函数 的方式, 可以将错误的信息对象直接传给该回调函数， Nuxt.js 内部会自动调用 error 方法:

import axios from 'axios'
export default {
  asyncData ({ params }, callback) {
    axios.get(`http://localhost/?id=${params.id}`)
    .then((res) => {
      callback(null, { title: res.data.msg })
    }).catch((e) => {
      callback({ statusCode: 404, message: 'Page not found' })
    })
  }
}

# Vue 实例注入

以上代码, 我们都单独引入 axios 进行操作, 其实 asyncData 可以指定一个 app 参数, 通过 context.app.$axios 直接调用 axios, 当然, 也可通过 context.app 调用其他 Vue 实例的方法。

比如前面的代码可以写为:

asyncData({ app, params }) {
  return app.$axios.get(`http://localhost/?id=${params.id}`).then(res => {
    return { title: res.data.msg }
  })
}
