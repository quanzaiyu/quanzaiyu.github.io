# 插件 plugins

# install

Vue.js 的插件必须有一个公开方法 install。这个方法的第一个参数是 Vue 构造器，第二个参数是一个可选的选项对象。

比如，在同目录下有以下三个文件:

  • MyPlugin.js - 插件定义
  • componentTest.vue - 组件
  • main.js - 入口

MyPlugin.js

import componentTest from './componentTest'
export default {
	install (_Vue, oprions) {
    	_Vue.component('componentTest', componentTest)
    }
}

componentTest.vue

<template>
	<div>test</div>
</template>

<script>
export default {}
</script>

<style></style>

此时，只需要在 main.js 中使用 Vue 的 use 方法引入就可以插件即可:

import Vue from 'vue'
import MyPlugin from 'MyPlugin'

Vue.use(MyPlugin)

如果需要传入参数，则放到第二参数:

Vue.use(MyPlugin, {})

比如需要在组件中使用 vuex 或 其他选项，都能通过这个选项对象传入。

# 添加全局方法或属性

可以通过第一参数 Vue 添加全局属性或方法:

MyPlugin.js

export default {
	install (Vue, oprions) {
        // 添加实例方法
        Vue.prototype.$testFunc = function () {
          return 'Hello Vue'
        }
        // 添加实例属性
        Vue.prototype.$num = 0
        Object.defineProperty(Vue.prototype, '$testName', {
          get: function () {
            return 'quanzaiyu'
          }
        })
    }
}

在 main.js 中引入并使用:

import Vue from 'vue'
import MyPlugin from 'MyPlugin'

Vue.use(MyPlugin)

let vue = new Vue(...)

vue.$testName
vue.$testFunc()

# 使用状态管理

可以通过第二参数添加状态管理:

MyPlugin.js

export default {
	install (_Vue, store) {
    	store.commit(...)
        store.state...
        store...
    }
}

在 main.js 中引入并使用:

import Vue from 'vue'
import MyPlugin from 'MyPlugin'
import Vuex from 'vuex'
let store = new Vuex.Store({...})

Vue.use(MyPlugin, store)

let vue = new Vue(...)
