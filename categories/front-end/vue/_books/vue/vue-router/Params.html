# 路由传参

路由传参有两种方式: query 和 params

通过 query 传参:

// 注意: 使用 path 方式的路由才能使用 query
this.$router.push({
  path:'/xxx',
  query:{
    id: id
  }
})

// 接收参数:
this.$route.query.id

通过 params 传参:

// 注意: 使用 name 方式的路由才能使用 params
this.$router.push({
  name:'xxx',
  params:{
    id: id
  }
})

// 接收参数:
this.$route.params.id
