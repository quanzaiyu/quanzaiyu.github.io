          # 深入理解 不可变对象(Immutable Objects)

          JS 是一门动态语言，可以很方便地往对象中添加属性:

          let obj = {
    num: 10,
    obj: {
        content: "mutable object"
    }
}

obj.num = 5;
obj.obj = { content: "changed!" }

console.log(obj) // { num: 5, obj: { content: "changed!"  } }

          # 尝试: 使用 const

          ES6 中可以使用 const 定义常量，不过由于对象是引用类型，const 关键字只是修改了某个变量名和其值之间的链接，而不是实际值。

          const obj = {
  num: 10,
  obj: {
      content: "mutable object"
  }
}

obj.num = 5;
obj.obj = { content: "changed!" }
obj.new = 0

console.log(obj); // {num: 5, obj: {content: "changed!"}, new: 0}

obj = 0 // Uncaught TypeError: Assignment to constant variable.

          相当于在栈内存中引了一根指针指向堆内存中分配的空间，这空间里面的属性指向真实值，改变属性值只是让堆内存中的指针指向变化，而没有改变栈中指针的指向。而如果直接对定义在栈中的变量重新赋值，则会报错。

          # 尝试: Object.freeze()

          有很多关于 ES2015 新特性的文章和讨论。我们知道 ES2015 比 ES5 更好。例如，我们可以使用一个 Object 方法来实现我们的目的：Object.freeze()。该方法使对象的原始属性不可变。我们把这个方法应用到我们原来的 obj 对象上：

          let obj = {
  num: 10,
  obj: {
      content: "mutable object"
  }
}

Object.freeze(obj);
obj.num = 5;
obj.obj = { content: "changed!" }

console.log(obj) // { num: 10, obj: { content: "changed!"  } }

          结果比原先的尝试稍后好一点，原始值现在已经修复，不可更改，但是我们仍然可以更改嵌套对象。

          # 最终解决方案

          为了使对象完全不可变，我们还需要freeze()所有的嵌套对象。例如（方法来自 MDN）：

          function deepFreeze(obj) {
    var propNames = Object.getOwnPropertyNames(obj);
    propNames.forEach(function(name) {
        var prop = obj[name];
        if (typeof prop == 'object' && prop !== null) {
            deepFreeze(prop);
        }
    });
    return Object.freeze(obj);
}

          使用这个函数，现在我们可以创建完全不可变的对象：

          let obj = {
  num: 10,
  obj: {
      content: "mutable object"
  }
}

deepFreeze(obj);
obj.num = 5;
obj.obj = { content: "changed!" }

console.log(obj) // { num: 10, obj: { content: "mutable object"  } }

          # defineProperty

          使用 defineProperty 定义的属性默认是不可变的:

          let obj = {}

Object.defineProperty(obj, 'num', {
  get () {
    return 10
  }
})

obj.num = 20

console.log(obj) // { num: 10 }

          或者直接对其 value 赋值

          let obj = {}

Object.defineProperty(obj, 'num', {
  value: 10
})

obj.num = 20

console.log(obj) // { num: 10 }
