            # JavaScript 判断对象是否为空的几种方法

            # 方式一：Json 转化

            使用 JSON.stringify 将一个对象转化为字符串然后进行比较，也是最简单的一种方式：

            var data = {}
JSON.stringify(data) === "{}" // true

            # 方式二：第三方库

            可以借助第三方库进行判断，比如 jQuery 中的 isEmptyObject 方法：

            var data = {}
$.isEmptyObject(data) // true

            # 方式三：Object.getOwnPropertyNames()

            此方法是使用 Object 对象的getOwnPropertyNames方法，获取到对象中的属性名，存到一个数组中，返回数组对象，我们可以通过判断数组的length来判断此对象是否为空。

            注意：此方法不兼容 IE8

            var data = {}
var arr = Object.getOwnPropertyNames(data)
arr.length === 0 //true

            # 方式三：Object.keys()

            这是ES6的新方法, 返回值也是对象中属性名组成的数组：

            var data = {}
var arr = Object.keys(data)
arr.length === 0 //true
