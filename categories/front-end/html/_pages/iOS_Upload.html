                  # 图片压缩后在 iOS 中进行上传的问题

                  最近需要做图片压缩上传，心里飘出几个字：何其简单。不就是用 canvas 压压就好了。

                  然而现实是残酷的，在所有程序都感觉无懈可击的时候，在一切测试频频报捷的时候，PC 和 Android 都没有问题，但是使用 iOS 真机测试的时候挂掉了，所有的 iOS 手机都清一色报了个 500。但是，如果使用原图上传就一切正常。

                  纳尼纳尼纳尼，我的内心是奔溃的！！！

                  # 思路

                  先说说我的思路，我是使用 Vue 写的一个上传组件，默认压缩上传，可以手动勾选原图上传选项框进行原图上传。这是一个多图上传组件，可以一次性上传多张图片，原图和压缩后的图片都缓存到不同的数组中。

                  压缩图片的流程为：首先是使用 HTML 的 input（file）框获取文件，再通过 FileReader 读取文件，转化为 Base64，使用 Canvas 的 toDataURL 进行质量压缩操作，最后转化出来的仍然是 Base64，再将 Base64 转化为 File 类型，添加到压缩后的图片数组中，最后上传的时候遍历此数组，一个一个 append 到 FormData 中，进行批量上传。

                  # 猜想：异步压缩造成的问题

                  由于图片是进行异步压缩的，很有可能在执行上传动作的时候，图片压缩并没有完成，因此获取到的压缩后图片为空，导致上传报 500。

                  解决这个问题很简单，只需要判断压缩前和压缩后的图片数组长度是否相同即可：

                  submit () {
  // ...
  if (this.imgStoreCompress.length !== this.imgStore.length) {
    this.$toast('图片压缩处理中，请稍后...')
    return
  }
  // ...
}

                  然而，在 iOS 系统的手机上仍然报500，看来不是这个问题。

                  # 猜想：mime 类型的一致性

                  由于使用 canvas 进行压缩，有可能是压缩后的 mine 类型改变了，而我的扩展名并没有变（最开始我并没有更换文件名），其实理论上即使扩展名没变，也不应该影响文件上传，不过为了谨慎起见，我还是做了调整：

                  export function dataURLtoFile(urlData, filename) {
  var arr = urlData.split(','), mime = arr[0].match(/:(.*?);/)[1],
  bstr = atob(arr[1]), n = bstr.length, u8arr = new Uint8Array(n);
  while(n--){
    u8arr[n] = bstr.charCodeAt(n);
  }
  let parts = filename.split('.')
  parts[parts.length - 1] = mime.split('/')[1]
  filename = parts.join('.')
  return new File([u8arr], filename, {type:mime})
}

                  这里，完全使用 Base64 头部的数据作为 mime 类型，文件名称不变只是更换了扩展名。

                  然而，在 iOS 系统的手机上仍然报500，看来不是这个问题。

                  # 联动调试：问题排查

                  这个问题太奇怪了，因此决定将 iOS 连上 Fiddler 进行抓包调试，将本地 IP 设置为代理，边改边调。

                  结果发现，在 iOS 中上传的 FormData 没有 Content ！！！

                  早就听说 iOS 的 FormData 有坑，不过仍然不清楚坑在哪，在 append 进 FormData 之前，File 的 size 是能够打印出来的：

                  this.imgStoreCompress.forEach((item, index) => {
  formData.append(`imgFiles[${index}]`, item.file)
  alert(item.file.size)
  alert(item.file.name)
})

                  这里，size 和 name 都没问题。

                  在前端安装了 vConsole，在 iOS 手机上打印 FormData，发现他妈的 iOS FormData 原型中只有一个 append 方法！！！而我在 MDN 中查询 FormData 的标准接口中最起码应该有 get、getAll、set 等等方法才对，真的是日了狗了。

                  # 莫名解决

                  继续漫无目的地百度，结果有好些文章提到直接使用 Blob 进行上传。抱着一试的心理，将 Base64 直接搞成 Blob 进行上传试试。

                   export function dataURLToBlob(urlData, filename){
  var arr = urlData.split(','), mime = arr[0].match(/:(.*?);/)[1],
    bstr = atob(arr[1]), n = bstr.length, u8arr = new Uint8Array(n);
  while(n--){
    u8arr[n] = bstr.charCodeAt(n);
  }
  let parts = filename.split('.')
  parts[parts.length - 1] = mime.split('/')[1]
  filename = parts.join('.')
  return new Blob([u8arr], {type:mime});
}

                  结果，iOS 这货好了！！！

                  真的是莫名其妙了，File 是 Blob 的一个子类，只是在 Blob 的基础上增加了一些属性而已，并没什么啊，况且这是 w3c 标准类型，压缩之前也是 File 类型，使用 iOS 上传一点问题都没有，但是 canvas 压缩转化后就不行了，真的是奇了怪了。

                  虽然问题解决了，不过原因实际上还没搞清楚。

                  <template>

/* eslint-disable */
import { photoCompress, dataURLtoFile, dataURLToBlob } from './scripts/imgCompress.js'
import { mapState } from 'vuex'
export default {
  props: ['url'],
  data () {
    return {
      originImg: false,
      filesOrigin: [], // 原始文件
      filesCompress: [], // 压缩后的图片
      indexOrigin: 0, // 原始文件序列号
      indexCompress: 0, // 压缩后的图片序列号
      maxLength: 5 // 图片最大数量
    }
  },
  computed: {
    ...mapState({
      imgStoreCompress: state => state.imgstore.img_upload_cache_compress.slice(0, 5),
      imgStore: state => state.imgstore.img_upload_cache.slice(0, 5),
      imgPaths: state => state.imgstore.img_paths,
      imgStatus: state => state.imgstore.img_status
    })
  },
  methods: {
    // 选择图片
    selectImgs () {
      let fileList = this.$refs.file.files
      // 存储原始图片
      for (let i = 0, len = fileList.length; i < len; i++) {
        let item = {
          key: this.indexOrigin++,
          name: fileList[i].name,
          size: fileList[i].size,
          file: fileList[i]
        }
        // 将图片文件转成BASE64格式
        let reader = new FileReader()
        reader.onload = (e) => {
          this.$set(item, 'src', e.target.result)
        }
        reader.readAsDataURL(fileList[i])
        this.filesOrigin.push(item)
      }
      this.filesOrigin.splice(5)
      this.$store.commit('set_img_upload_cache', this.filesOrigin.slice(0, 5)) // 存储文件缓存
      this.$store.commit('set_img_status', 'selected') // 更新文件上传状态
      // 存储压缩后的图片
      for (let i = 0, len = fileList.length; i < len; i++) {
        let item = {
          key: this.indexCompress++,
          name: fileList[i].name,
          size: fileList[i].size,
          file: fileList[i]
        }
        // 将图片文件转成BASE64格式
        let reader = new FileReader()
        reader.onload = (e) => {
          this.$set(item, 'src', e.target.result)
        }
        reader.readAsDataURL(fileList[i])
        // console.log(`图片[${i}]，原图：${fileList[i].size / 1024}KB`);
        if (fileList[i].size / 1024 / 500 < 1) {
          // 图片小于500KB不进行压缩，如进行压缩图片大小可能会增大
          this.filesCompress.push(item)
          this.filesCompress.splice(5)
          this.$store.commit('set_img_upload_cache_compress', this.filesCompress.slice(0, 5)) // 存储文件缓存
        } else {
          // 对图片尺寸大于500KB的图片进行压缩
          photoCompress(fileList[i], {
            quality: 0.8
          }, (base64Codes) => {
            // console.log(`图片[${i}]，压缩后：${base64Codes.length / 1024}KB`);
            let file = dataURLToBlob(base64Codes, item.name)
            item.file = file
            item.name = file.name
            item.size = file.size
            this.filesCompress.push(item)
            this.filesCompress.splice(5)
            this.$store.commit('set_img_upload_cache_compress', this.filesCompress.slice(0, 5)) // 存储文件缓存
          })
        }
      }
      this.$store.commit('set_img_status', 'selected') // 更新文件上传状态
    },
    // 上传图片
    submit () {
      let formData = new FormData()
      if (this.originImg) {
        // 原图上传
        this.imgStore.forEach((item, index) => {
          formData.append(`imgFiles[${index}]`, item.file)
        })
      } else {
        // 压缩上传
        if (this.imgStoreCompress.length !== this.imgStore.length) {
          this.$toast('图片压缩处理中，请稍后...')
          this.$store.commit('set_img_status', 'selected') // 更新文件上传状态
          return
        }
        this.imgStoreCompress.forEach((item, index) => {
          formData.append(`imgFiles[${index}]`, item.file)
          // alert(item.file.size)
          // alert(item.file.name)
        })
      }
      // 新建请求
      const xhr = new XMLHttpRequest()
      xhr.open('POST', this.url, true)
      xhr.send(formData)
      // xhr.onreadystatechange = function () {
      //   alert(xhr.status)
      // }
      xhr.onload = () => {
        if (xhr.status === 200 || xhr.status === 304) {
          let datas = JSON.parse(xhr.responseText)
          // console.log(JSON.stringify(datas))
          // alert(JSON.stringify(datas))
          // return
          let flag = true
          datas.forEach(data => {
            if (data.error === -1) {
              flag = false
            }
          })
          // 存储返回的地址
          if (flag) {
            let imgUrlPaths = []
            datas.forEach(e => { // error === 0为成功状态
              e.error === 0 && imgUrlPaths.push(e.url)
            })
            this.$store.commit('set_img_paths', imgUrlPaths) // 存储返回的地址
            this.filesOrigin = [] // 清空文件缓存
            this.indexOrigin = 0 // 初始化序列号
            this.$store.commit('set_img_status', 'finished') // 更新文件上传状态
          } else {
            alert('图片上传失败，请重新上传!')
            this.$store.commit('set_img_status', 'ready') // 更新文件上传状态
            this.empty()
            flag = true
          }
        } else {
          this.$toast(`${xhr.status} 请求错误!`)
        }
      }
    },
    // 移除图片
    remove (index) {
      this.filesCompress = this.filesCompress.filter(file => {
        return file.key !== index
      })
      this.filesOrigin = this.filesOrigin.filter(file => {
        return file.key !== index
      })
      this.$store.commit('set_img_upload_cache', this.filesOrigin)
      this.$store.commit('set_img_upload_cache_origin', this.filesOrigin)
      this.$store.commit('set_img_upload_cache_compress', this.filesCompress)
    },
    // 清空图片
    empty () {
      this.filesCompress.splice(0, this.filesCompress.length)
      this.filesOrigin.splice(0, this.filesOrigin.length)
      this.$store.commit('set_img_upload_cache', [])
      this.$store.commit('set_img_upload_cache_origin', [])
      this.$store.commit('set_img_upload_cache_compress', [])
      this.$store.commit('set_img_paths', [])
    }
  },
  beforeCreate () {
    this.$store.commit('set_img_status', 'ready') // 更新文件上传状态
  },
  destroyed () {
    this.$store.commit('set_img_upload_cache', [])
    this.$store.commit('set_img_upload_cache_compress', [])
    this.$store.commit('set_img_upload_cache_origin', [])
    this.$store.commit('set_img_paths', [])
  },
  watch: {
    imgStatus () {
      if (this.imgStatus === 'uploading') {
        this.submit()
      }
    },
    imgStore () {
      if (this.imgStore.length <= 0) {
        this.$store.commit('set_img_status', 'ready') // 更新文件上传状态
      }
    }
  }
}
</script>

                  canvas 压缩脚本、类型转换脚本：imgCompress.js

                  /* eslint-disable */
/**
  file：一个是文件(类型是图片格式)，
  w：一个是文件压缩的后宽度，宽度越小，字节越小
  objDiv：一个是容器或者回调函数
  photoCompress()
*/
export function photoCompress(file,w,objDiv) {
  var ready=new FileReader();
  /*开始读取指定的Blob对象或File对象中的内容. 当读取操作完成时,readyState属性的值会成为DONE,如果设置了onloadend事件处理程序,则调用之.同时,result属性中将包含一个data: URL格式的字符串以表示所读取文件的内容.*/
  ready.readAsDataURL(file);
  ready.onload=function(){
    var re=this.result;
    canvasDataURL(re,w,objDiv)
  }
}
function canvasDataURL(path, obj, callback, quality = 0.7) {
  var img = new Image();
  img.src = path;
  img.onload = function(){
    var that = this;
    // 默认按比例压缩
    var w = that.width,
        h = that.height,
        scale = w / h;
    w = obj.width || w;
    h = obj.height || (w / scale);
    //生成canvas
    var canvas = document.createElement('canvas');
    var ctx = canvas.getContext('2d');
    // 创建属性节点
    var anw = document.createAttribute("width");
    anw.nodeValue = w;
    var anh = document.createAttribute("height");
    anh.nodeValue = h;
    canvas.setAttributeNode(anw);
    canvas.setAttributeNode(anh);
    ctx.drawImage(that, 0, 0, w, h);
    // 图像质量
    if(obj.quality && obj.quality <= 1 && obj.quality > 0){
        quality = obj.quality;
    }
    // quality值越小，所绘制出的图像越模糊
    var base64 = canvas.toDataURL('image/jpeg', quality);
    // 回调函数返回base64的值
    callback(base64);
  }
}

/**
  * 将以base64的图片 url 数据转换为 Blob
  * @param urlData
  * 用 url 方式表示的 base64 图片数据
  */
 export function dataURLToBlob(urlData, filename){
  var arr = urlData.split(','), mime = arr[0].match(/:(.*?);/)[1],
    bstr = atob(arr[1]), n = bstr.length, u8arr = new Uint8Array(n);
  while(n--){
    u8arr[n] = bstr.charCodeAt(n);
  }
  let parts = filename.split('.')
  parts[parts.length - 1] = mime.split('/')[1]
  filename = parts.join('.')
  return new Blob([u8arr], {type:mime});
}
/**
  * 将以 base64 的图片 url 数据转换为 File
  * @param urlData
  * 用 url 方式表示的 base64 图片数据
  */
 export function dataURLtoFile(urlData, filename) {
  var arr = urlData.split(','), mime = arr[0].match(/:(.*?);/)[1],
  bstr = atob(arr[1]), n = bstr.length, u8arr = new Uint8Array(n);
  while(n--){
    u8arr[n] = bstr.charCodeAt(n);
  }
  let parts = filename.split('.')
  parts[parts.length - 1] = mime.split('/')[1]
  filename = parts.join('.')
  return new File([u8arr], filename, {type:mime})
}

        $optput['url']   = Strings::fileWebLink($info->getRealPath());
        $suffix = strtolower(pathinfo($info->getRealPath(), PATHINFO_EXTENSION));
        if ($suffix == 'amr') {
          try {
            $amrfile = $info->getRealPath();
            $wavfile = str_replace('.'.$suffix, '.wav', $amrfile);
            $rs = $this->amr2wav($amrfile, $wavfile);
            if ($rs) {
              $optput['url']   = Strings::fileWebLink($wavfile);
            }
          } catch (\Exception $e) {
            $log_name = "log/" . date('Ym') . '/' . date('Y-m-d') . ".error.log";
            wfile($log_name, 'amr2wav 失败：'.$e->getLine() . ' ' . $e->getMessage());
            $optput['url']   = Strings::fileWebLink($info->getRealPath());
          }
        }
        $optput['error'] = 0;
        $optput['message'] = '上传成功';
      } else {
          $optput['message'] = $file->getError();
      }
    }
    debug('uploadend');
    $apiruntime = debug('uploadstart','uploadend',6);
    $optput['runtime'] = $apiruntime;
    return json_encode($optput, JSON_HEX_QUOT);
  }
}

//多图片上传
public function uploadpics()
{
  $request = Request::instance();
  if ($request->isPost()) {
    $files = $request->file('imgFiles');
    $outImgs=[];
    foreach ($files as $key =>$file){
      if (Loader::model('ImagesInfo')->imageSize('editor', $file) === false) {
        $outImgs[$key]['error'] = -1;
        $outImgs[$key]['message'] = Loader::model('ImagesInfo')->getError();
      } else {
        $myname = function (){return date('Ym') . DS. date('d') . DS. md5(microtime(true));};
        $info = $file->rule($myname)->move(UPLOAD_PATH . DS . 'api' . DS);
        if ($info) {
          Loader::model('ImagesInfo')->handingImage('editor', $info);
          $outImgs[$key]['error']  = 0;
          $outImgs[$key]['url']  = Strings::fileWebLink($info->getRealPath());
        } else {
          $outImgs[$key]['error'] = -1;
          $outImgs[$key]['message'] = $file->getError();
        }
      }
    }
    return json($outImgs);
  }
}
