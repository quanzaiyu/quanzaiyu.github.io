            # 模板引擎 - Ejs 的使用

            ejs 使用嵌入式语法，是一种 温和的、非侵入式的、若依赖的模板引擎。

            使用 ejs，可以嵌入 HTML 代码。

            首先需要下载 ejs

            $ yarn add ejs

            # 渲染文件

            使用 ejs.renderFile 渲染文件。

            const ejs = require('ejs')

ejs.renderFile('./views/index.ejs', {name: 'xiaoyu'}, (err, data) => {
  if (err) {
    console.log('编译失败')
  } else {
    console.log(data)
  }
})

            views/index.ejs

            <html>
  <head>
    <style></style>
  </head>
  <body>
    <div>Hello <%= name %></div>
  </body>
</html>

            编译为:

            <html>
  <head>
    <style></style>
  </head>
  <body>
    <div>Hello xiaoyu</div>
  </body>
</html>

            # 输出到文件

            const ejs = require('ejs')
const fs = require('fs')

ejs.renderFile('./views/index.ejs', {name: 'xiaoyu'}, (err, data) => {
  if (err) {
    console.log('编译失败')
  } else {
    fs.writeFile('./build/index.html', data, (err) => {
      if (err) {
        console.log('写入失败')
      } else {
        console.log('写入成功')
      }
    })
  }
})

            views/index.ejs

            <html>
  <head>
    <style></style>
  </head>
  <body>
    <div>Hello <%= name %></div>
  </body>
</html>

            渲染为

            <html>
  <head>
    <style></style>
  </head>
  <body>
    <div>Hello xiaoyu</div>
  </body>
</html>

            # 基础语法

            凡是 js 语法都能写在 <% %> 之中，比如

            <%
var arr = ['a', 'b', 'c']
for(var i = 0; i < arr.length; i++) { %>
  <div><%=arr[i]%></div>
<% } %>

            渲染为

            <div>a</div>
<div>b</div>
<div>c</div>

            # 转义输出与不转义输出

            • 使用 <%= %> 是转义输出
            • 使用 <%- %> 是不转义输出
            <% var str = '<p>Hello</p>' %>
<%=str%>
<%-str%>

            &lt;p&gt;Hello&lt;/p&gt;
<p>Hello</p>

            # 文件包含

            使用 <% include ... %> 可以指定文件包含，被包含的文件当作普通文本嵌入到指定位置，被包含的文件不限格式，可以是 html、ejs、css、js等。注意，include 语句不能与其他js语句共存，必须单独使用 <% %> 包含。

            index.ejs

            <html>
  <head>
    <style></style>
  </head>
  <body>
    <% include ./header.html %>
  </body>
</html>

            header.html

            <header>Hello</header>

            渲染为

            <html>
  <head>
    <style></style>
  </head>
  <body>
    <header>Hello</header>
  </body>
</html>
