      # MySQL 数据库命令行工具的使用

      # 登录、登出数据库

      使用 sudo mysql -u root -p 登录数据库，-u 指定是哪个用户进行登录，输入密码即可。

      Mac-mini:/ DYT$ sudo mysql -u root -p
Enter password:
Welcome to the MySQL monitor.  Commands end with ; or \g.
Your MySQL connection id is 46
Server version: 5.7.21 MySQL Community Server (GPL)

Copyright (c) 2000, 2018, Oracle and/or its affiliates. All rights reserved.

Oracle is a registered trademark of Oracle Corporation and/or its
affiliates. Other names may be trademarks of their respective
owners.

Type 'help;' or '\h' for help. Type '\c' to clear the current input statement.

mysql>

      如果要连接指定 IP 的数据库，带上 -h 参数指定：

      mysql -h localhost -u root -p

      使用 exit 退出 mysql 命令行。

      mysql> exit
Bye

      # 基础语句

      注意

      命令行是以英文分号 ; 为结束，只要没有分号，Mysql就认为命令没有输完，就不会执行命令。

      # 创建表并显示

      示例：

      mysql> use test;
Database changed
mysql> CREATE TABLE Persons
    -> (
    -> PersonID int,
    -> LastName varchar(255),
    -> FirstName varchar(255),
    -> Address varchar(255),
    -> City varchar(255)
    -> );
Query OK, 0 rows affected (0.31 sec)

mysql> show tables;
+----------------+
| Tables_in_test |
+----------------+
| Persons        |
+----------------+
1 row in set (0.00 sec)

mysql> select * from Persons;
Empty set (0.00 sec)

      # 其他常用命令

      • SELECT - 从数据库中提取数据
      • UPDATE - 更新数据库中的数据
      • DELETE - 从数据库中删除数据
      • INSERT INTO - 向数据库中插入新数据
      • CREATE DATABASE - 创建新数据库
      • ALTER DATABASE - 修改数据库
      • CREATE TABLE - 创建新表
      • ALTER TABLE - 变更（改变）数据库表
      • DROP TABLE - 删除表
      • CREATE INDEX - 创建索引（搜索键）
      • DROP INDEX - 删除索引
      set names utf8; -- 设置使用的字符集
SET PASSWORD FOR 'root'@'localhost' = PASSWORD('newpassword'); -- 修改指定用户的密码

DROP TABLE table_name; -- 删除表，包括数据和结构
TRUNCATE TABLE table_name; -- 删除表数据
DROP DATABASE database_name; -- 删除数据库

ALTER TABLE table_name ADD column_name datatype; -- 在表中添加列
ALTER TABLE table_name DROP COLUMN column_name; -- 在表中删除列
ALTER TABLE table_name ALTER COLUMN column_name datatype; -- 改变表中列的数据类型，SQL Server / MS Access
ALTER TABLE table_name MODIFY COLUMN column_name datatype; -- 改变表中列的数据类型，My SQL / Oracle

      其他的查询语句也可直接在命令行中使用。
