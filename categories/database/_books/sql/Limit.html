# Limit

在实际应用中，我们常常有分页的需求，在查询数据库的时候，可以使用 Limit 关键字进行分页操作。

在不同的数据库中，实现是不同的，Limit 是 MySQL 的实现方式，如果是 SQL Server 或 MS Access，则使用 SELECT TOP 实现，Oracle 使用 WHERE ROWNUM <= number 实现，这里不做讨论，只看 Limit 的实现方式。

基础语法：

SELECT column_name(s) FROM table_name LIMIT number;
-- or
SELECT column_name(s) FROM table_name LIMIT number, length;

如果只使用一个参数，number，则查询数据库中前面的 number 条数据。

如果使用两个参数，number 和 length，则从 number 行开始，查询之后的 length 条数据。

示例：

有如下一张表

mysql> SELECT * FROM Websites;
+----+---------------+---------------------------+-------+---------+
| id | name          | url                       | alexa | country |
+----+---------------+---------------------------+-------+---------+
|  1 | Google        | https://www.google.cm/    |     1 | USA     |
|  2 | 淘宝          | https://www.taobao.com/   |    13 | CN      |
|  3 | 菜鸟教程       | http://www.runoob.com/    |  5000 | USA     |
|  4 | 微博           | http://weibo.com/         |    20 | CN      |
|  5 | Facebook      | https://www.facebook.com/ |     3 | USA     |
|  7 | stackoverflow | http://stackoverflow.com/ |     0 | IND     |
+----+---------------+---------------------------+-------+---------+

SELECT * FROM Websites LIMIT 2;

查询出以下数据：

+----+---------------+---------------------------+-------+---------+
| id | name          | url                       | alexa | country |
+----+---------------+---------------------------+-------+---------+
|  1 | Google        | https://www.google.cm/    |     1 | USA     |
|  2 | 淘宝          | https://www.taobao.com/   |    13 | CN      |
+----+---------------+---------------------------+-------+---------+

SELECT * FROM Websites LIMIT 2,3;

查询出以下数据：

+----+---------------+---------------------------+-------+---------+
| id | name          | url                       | alexa | country |
+----+---------------+---------------------------+-------+---------+
|  3 | 菜鸟教程       | http://www.runoob.com/    |  5000 | USA     |
|  4 | 微博           | http://weibo.com/         |    20 | CN      |
|  5 | Facebook      | https://www.facebook.com/ |     3 | USA     |
+----+---------------+---------------------------+-------+---------+

# Limit 的其他使用技巧

查询第一条数据：

SELECT column_name FROM table_name ORDER BY column_name ASC LIMIT 1;

查询最后一条数据

SELECT column_name FROM table_name ORDER BY column_name DESC LIMIT 1;
