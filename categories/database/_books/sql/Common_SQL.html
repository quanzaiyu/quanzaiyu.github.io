# CURD

SQL，指结构化查询语言，全称是 Structured Query Language。

TIP

SQL 对大小写不敏感：SELECT 与 select 是相同的。

SQL 基本操作为 C（create）创建、U（update）更新、R（Retrieve）读取、D（delete）删除。

# 查询语句

SELECT 语句用于从数据库中选取数据。结果被存储在一个结果表中，称为结果集。

基础语法

SELECT * FROM table_name;
SELECT column_name1,column_name1 FROM table_name;

# DISTINCT

在表中，一个列可能会包含多个重复值，有时我们希望仅仅列出不同（distinct）的值。DISTINCT 关键词用于返回唯一不同的值。

基础语法

SELECT DISTINCT column_name,column_name FROM table_name;

比如：

select distinct age from api_age;

# WHERE

WHERE 子句用于提取那些满足指定标准的记录。

基础语法

SELECT column_name,column_name FROM table_name WHERE column_name operator value;

WHERE 子句中的运算符

运算符 描述
= 等于
<> 或 != 不等于
> 大于
< 小于
>= 大于等于
<= 小于等于

WHERE 子句的逻辑运算符

运算符 描述
not 非
and 与
or 或
() 改变运算符优先级

逻辑运算的优先级：

()  >  not  >  and  >  or

比如：

select * from api_wechat where id < 100 and subscribe = 1;
select * from emp where not sal > 1500;

空值判断

使用 is null 进行空值判断

Select * from emp where comm is null;

范围查询

使用 between and 查询指定范围的数据

Select * from emp where sal between 1500 and 3000;

包含查询

使用 In 操作符进行包含查询

Select * from emp where sal in (5000,3000,1500);

模糊查询

使用 Like 操作符进行模糊查询

Select * from emp where ename like 'M%';
  • % : 表示多个字值，_ 下划线表示一个字符；
  • M% : 为能配符，正则表达式，表示的意思为模糊查询信息为 M 开头的。
  • %M% : 表示查询包含M的所有内容。
  • %M_ : 表示查询以M在倒数第二位的所有内容。

# 排序

ORDER BY 关键字用于对结果集按照一个列或者多个列进行排序，默认按照升序对记录进行排序。如果需要按照降序对记录进行排序，可以使用 DESC 关键字。

SELECT column_name,column_name FROM table_name ORDER BY column_name,column_name ASC|DESC;

比如：

SELECT * FROM Websites ORDER BY alexa DESC;

多列排序：

SELECT * FROM Websites ORDER BY country,alexa;

首先按照 country 字段进行升序排列，然后按照 alexa 字段进行升序排列。

# 插入语句

INSERT INTO 语句用于向表中插入新记录。

第一种形式无需指定要插入数据的列名，只需提供被插入的值即可：

INSERT INTO table_name VALUES (value1,value2,value3,...);

第二种形式需要指定列名及被插入的值：

INSERT INTO table_name (column1,column2,column3,...) VALUES (value1,value2,value3,...);

比如：

INSERT INTO Websites (name, url, alexa, country) VALUES ('百度','https://www.baidu.com/','4','CN');

# 更新语句

UPDATE 语句用于更新表中的记录。

基础语法

UPDATE table_name SET column1=value1,column2=value2,... WHERE some_column=some_value;

比如：

UPDATE Websites SET alexa='5000', country='USA' WHERE name='xiaoyulive';

警告

在更新记录时要格外小心！在上面的实例中，如果我们省略了 WHERE 子句，如下所示：

UPDATE Websites SET alexa='5000', country='USA';

执行以上代码会将 Websites 表中所有数据的 alexa 改为 5000，country 改为 USA。

执行没有 WHERE 子句的 UPDATE 要慎重，再慎重。

TIP

在 MySQL 中可以通过设置 sql_safe_updates 这个自带的参数来解决，当该参数开启的情况下，你必须在update 语句后携带 where 条件，否则就会报错。

set sql_safe_updates=1; 表示开启该参数。

# 删除语句

DELETE 语句用于删除表中的记录。

基础语法

DELETE FROM table_name WHERE some_column=some_value;

比如：

DELETE FROM Websites WHERE name='百度' AND country='CN';

警告

WHERE 子句规定哪条记录或者哪些记录需要删除。如果您省略了 WHERE 子句，所有的记录都将被删除！
