# 项目搭建

Spring Cloud 项目是基于 Spring Boot 的, 因此需要先创建一个 Spring Boot 项目, 推荐使用 Spring Initializr 创建。

项目下的pom文件作为 父pom，起到依赖版本控制的作用，其他module工程 继承该pom, 创建完成后删除 src 目录, 后续将创建多个 module

# 版本对应关系

由于Spring Cloud基于Spring Boot构建，而Spring Cloud Alibaba又基于Spring Cloud Common的规范实现，所以当我们使用Spring Cloud Alibaba来构建微服务应用的时候，需要知道这三者之间的版本关系。

下表整理了目前Spring Cloud Alibaba的版本与Spring Boot、Spring Cloud版本的兼容关系：

Spring Boot Spring Cloud Spring Cloud Alibaba
2.1.x Greenwich 0.9.x
2.0.x Finchley 0.2.x
1.5.x Edgware 0.1.x
1.5.x Dalston 0.1.x
