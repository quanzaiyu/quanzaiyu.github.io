# 命令行

使用IDE开发虽然方便, 但是隐藏了很多细节, 这里使用最原始的方式编译及运行java

先创建一个测试文件

public class Main {
    public static void main(String[] args) {
        System.out.println("Hello World!");
    }
}

# 编译及运行

$ javac Main.java

注: 必须跟上 .java 后缀名

可以看到生成了一个 Main.class 的文件, 使用下面的命令运行:

$ java Main
Hello World!
