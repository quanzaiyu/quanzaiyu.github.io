# 代码热更新

springBoot对调试支持很好，修改之后可以实时生效，需要在 pom.xml 中添加以下配置：

<dependencies>
    <dependency>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-devtools</artifactId>
        <optional>true</optional>
    </dependency>
</dependencies>

<build>
    <plugins>
        <plugin>
            <groupId>org.springframework.boot</groupId>
            <artifactId>spring-boot-maven-plugin</artifactId>
            <configuration>
                <fork>true</fork>
            </configuration>
        </plugin>
    </plugins>
</build>

键入 Ctrl + Alt + Shift + / 选择 Registry..., 勾选 compiler.automake.allow.when.app.running

File | Settings | Build, Execution, Deployment | Compiler 勾选 Build project automatically
