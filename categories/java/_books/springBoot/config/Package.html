# 构建打包

运行 mvn package 可以对项目进行打包, 输出到 target/xxx.jar

可以使用 java -jar 命令运行该应用程序：

$ java -jar target/demo-0.0.1-SNAPSHOT.jar

在构建之前, 需要安装依赖以及清理之前的构建

mvn install
mvn clean
