# 模拟器

使用 Android Studio 自动下载的 gradle 目录为: C:\Users\Quanzaiyu\.gradle\wrapper\dists

# 终端启动 Android Studio 模拟器:

  1. 进入 Android SDK 的 emulator 目录
  2. 执行以下命令:
$ ./emulator.exe -netdelay none -netspeed full -avd Nexus_6_API_28
