    # Windows 错误日志整理

    在计算机上右键, 选择 管理 > 系统工具 > 事件查看器 > Windows 日志 > 系统 中可以查看所有系统日志, 前段时间公司的电脑经常蓝屏重启, 用这种方式比较容易排错。

    事件类型采用事件ID进行归档分类, 可以在右边的 操作 > 筛选当前日志 中进行事件ID的筛选

    常见的事件ID有:

    • 1001(错误) BugCheck 通常为电脑蓝屏所致
    • 7001(信息) Winlogon 登录操作系统
    • 6013(信息) EventLog 系统启动时间
    • 41(关键) Kernel-Power 系统已在未先正常关机的情况下重新启动
    • 20(错误) WindowsUpdateClient Windows 安装更新失败
