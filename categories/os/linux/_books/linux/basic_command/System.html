# 系统相关命令

  • login 登录
  • logout 注销
  • poweroff 普通关机
  • halt 立刻关机
  • reboot 普通重启
  • shutdown -r now 立刻重启
  • shutdown -r 10 过10分钟自动重启
  • shutdown -r 20:35 在时间为20:35时候重启
  • shutdown -h now 关闭系统(root用户使用)
  • shutdown -h 10 10分钟后关闭系统
  • shutdown -c 取消按预定时间关闭系统
  • init 0 关闭系统
  • telinit 0 关闭系统
