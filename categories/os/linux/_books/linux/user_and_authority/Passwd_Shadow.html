# /etc/passwd 中的字段分析

  • ACCOUNT：用户名
  • PASSWORD：密码占位符
  • UID：用户ID
  • GID：用户组ID
  • COMMAND：注释信息
  • HOME DIR：用户家目录
  • SHELL：用户的默认 shell

# /etc/shadow

密码占位符，其值是 x，显然这不是真正的密码。真正的密码保存在哪里呢？在 /etc/shadow 文件中，此文件中保存的也不是明文密码，而是经过加密处理之后的密码。我们来看一下 /etc/shadow 中的内容（root only）：
