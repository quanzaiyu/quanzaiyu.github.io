    # 给 rm 命令添加个垃圾桶

    当使用 rm 命令时，一定要小心，特别是使用 rm -rf 的时候，如果一不小心将整个服务器都删除，后悔晚矣。

    我们可以使用 saferm.sh 实现 rm 移至垃圾桶的目的.

    $ git clone https://github.com/lagerspetz/linux-stuff
$ mv linux-stuff/scripts/saferm.sh /bin
$ rm -Rf linux-stuff

    测试命令

    $ saferm.sh a
a is on /home. Unsafe delete (y/n)? y
Deleting a

$ saferm.sh d
d is on /home. Unsafe delete (y/n)? n
Moving d to /root/Trash

    修改 /root/.bashrc 文件

    $ vim .bashrc
alias rm=saferm.sh

    执行下面的命令使其生效，

    $ source ~/.bashrc

    测试

    $ rm b
b is on /home. Unsafe delete (y/n)? y
Deleting b

$ rm d
d is on /home. Unsafe delete (y/n)? n
Moving d to /root/Trash

    删除目录

    $ rm -r dirName

    一切就绪，现在可以执行 rm 命令，自动将文件移动到”垃圾桶”，而不是永久删除它们。

    查看帮助

    $ saferm.sh -h

    查看垃圾桶

    $ ls -lh /root/Trash/
