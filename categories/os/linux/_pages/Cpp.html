      # 使用 Linux 进行 C++ 开发

      # 安装

      yum 安装

      $ yum install -y gcc gcc-c++

      源码安装 GNU GCC

      $ wget http://ftp.gnu.org/gnu/gcc/gcc-7.3.0/gcc-7.3.0.tar.gz
$ tar -xzvf gcc-7.3.0.tar.gz
$ cd gcc-7.3.0
$ ./contrib/download_prerequisites
$ mkdir gcc-build-4.8.5
$ cd gcc-build-4.8.5
$ ../configure --prefix=/usr
$ make && make install

      验证安装

      $ gcc --version
$ gcc (GCC) 4.8.5

$ g++ --version
$ g++ (GCC) 4.8.5

$ which gcc
/usr/bin/gcc

$ which g++
/usr/bin/g++

$ rpm -qa | grep gcc

      # 编写源代码测试

      cpp 测试

      $ vim main.cpp

#include <iostream>
using namespace std;
int main() {
    cout << "Hello world!" << endl;
    return 0;
}

      c 测试

      $ vim main.cpp 

#include <stdio.h>
int main() {
    printf("Hello world!\n");
    return 0;
}

      $ ./test
Hello world

      编译

      $ g++ main.cpp -o main

      运行

      $ ./main
Hello world!
