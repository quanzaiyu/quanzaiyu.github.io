    # 使用Linux进行Python开发

    在部分系统中，已经集成 2.7，我们不动 2.7，可能有的脚本是基于 2.7 开发的，我们在此基础上安装 3.7

    # 安装 Python 3.7

    安装环境依赖

    $ yum -y install zlib-devel bzip2-devel openssl-devel ncurses-devel sqlite-devel readline-devel tk-devel gdbm-devel db4-devel libpcap-devel xz-devel

    下载并解压

    $ wget https://www.python.org/ftp/python/3.7.0/Python-3.7.0b3.tar.xz
$ xz -d Python-3.7.0b3.tar.xz
$ tar -xvf Python-3.7.0b3.tar
$ cd Python-3.7.0b3

    配置与安装

    $ mkdir /root/software/python3 -p
$ ./configure --prefix=/root/software/python3
$ ./configure --enable-optimizations

    建立 python3 的连接

    $ ln /root/software/python3/bin/python3.7 /usr/bin/python3

    测试

    $ python3
