      # 使用 on-my-zsh 打造终极终端神器

      安装 oh-my-zsh (GitHub)

      首先需要安装zsh

      $ apt install zsh

      更多定制也可参考: powerline-shell

      然后才能安装 oh-my-zsh

      $ sh -c "$(curl -fsSL https://raw.github.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"

# or
$ sh -c "$(wget https://raw.githubusercontent.com/robbyrussell/oh-my-zsh/master/tools/install.sh -O -)"

      一些关键路径:

      • 配置文件 ~/.zshrc
      • 程序自带插件 ~/.oh-my-zsh/plugins
      • 程序自带主题 ~/.oh-my-zsh/themes
      • 用户自定义插件 ~/.oh-my-zsh/custom/plugins
      • 用户自定义主题 ~/.oh-my-zsh/custom/themes

      与bash相互切换:

      ~/.oh-my-zsh(master*) # bash
root@DESKTOP-VEB85IN root@DESKTOP-VEB85IN:~/.oh-my-zsh# zsh
~/.oh-my-zsh(master*) #

      设置zsh为默认shell

      $ vim .bashrc
bash -c zsh

      卸载 oh-my-zsh 命令

      $ uninstall_oh_my_zsh

      查看所有的shell

      $ cat /etc/shells

      更改shell

      $ chsh -s /bin/zsh

      查看当前shell，但不能时时反映shell，需重启终端：

      $ echo $SHELL

      # 修改主题

      $ vim ~/.zshrc
ZSH_THEME="afowler"

      更多主题参考: https://github.com/robbyrussell/oh-my-zsh/wiki/themes

      # 修改前缀

      默认前缀为 →, 我们可以修改其中的主题文件自定义前缀, 比如我是用的主题是 robbyrussell

      $ cd ~/.oh-my-zsh/themes
$ vim robbyrussell.zsh-theme
local ret_status="%(?:%{$fg_bold[green]%}★:%{$fg_bold[red]%}➜ )"

      修改后结果:

      ★ custom git:(master) ✗

      # 添加插件

      本人使用了如下插件

      $ vim ~/.zshrc
plugins=(
z
git
bundler
dotenv
osx
rake
rbenv
ruby
extract
web-search
colored-man-pages
zsh-autosuggestions
zsh-syntax-highlighting
)
      • d 命令列出最近使用的目录, 使用数字快速跳转目录
      • z 命令列出常用的目录
      • x 命令解压一切文件
      • google 命令可以直接进行搜索

      其中 zsh-autosuggestions 和 zsh-syntax-highlighting 需要安装:

      $ cd ~/.oh-my-zsh/custom/plugins
$ git clone https://github.com/zsh-users/zsh-syntax-highlighting.git
$ git clone https://github.com/zsh-users/zsh-autosuggestions.git

      最后使其生效即可:

      $ source ~/.zshrc

      # incr 插件

      incr 是一款自动补全的插件, 下载:

      $ cd ~/.oh-my-zsh/custom/plugins
$ wget http://mimosa-pudica.net/src/incr-0.2.zsh

      使其生效, 在 ~/.zshrc 文件末尾加上

      $ source ~/.oh-my-zsh/custom/plugins/incr*.zsh
