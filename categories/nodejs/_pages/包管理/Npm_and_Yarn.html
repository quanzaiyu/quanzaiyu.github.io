        # npm 与 yarn 简介与使用

        # npm

        npm，全称 NodeJS Package Management，就是NodeJS的包管理工具。

        npm 有以下一些作用:

        1. 统一下载途径
        2. 自动下载依赖
        3. 提交代码到NPM官网进行托管
        4. 运行 package.json 中的脚本
        • NPM 官网
        • NPM 中文文档

        # 常用命令

        安装模块

        # 全局安装
$ npm install xxx -g
# 局部安装
$ npm install xxx --save
# 局部安装并添加到开发环境
$ npm install xxx --save-dev

        卸载模块

        $ npm uninstall xxx

        搜索包

        $ npm search xxx

        # 改变 npm 全局安装目录

        $ npm config set prefix "D:\Program Files\node\packages"

        # 关于 npmrc

        .npmrc 的意思为 npm runtime config 的意思，有全局配置，不同操作平台路径不一样, 用户使用的配置通常位于 ~/.npmrc。

        npmrc 文件分为以下几种权限：项目权限 > 用户权限 > 全局权限

        路径如下

        • 全局权限：$PREFIX/etc/npmrc
        • 用户权限：$HOME/.npmrc
        • 项目权限：$PROJECT/.npmrc 在项目更目录中，跟 node_modules 和 package.json 同路径

        # 在 Linux 中安装 yarn

        $ apt-get update && apt-get install -y curl apt-transport-https && \
  curl -sS https://dl.yarnpkg.com/debian/pubkey.gpg | apt-key add - && \
  echo "deb https://dl.yarnpkg.com/debian/ stable main" | tee /etc/apt/sources.list.d/yarn.list && \
  apt-get update && apt-get install -y yarn

        # npm 与 yarn 命令对比

        command yarn npm
        初始化 yarn init -y npm init -y
        在已有package.json的项目中安装依赖 yarn npm i / install
        全局安装 yarn global add <package> npm i -g / --global <package>
        添加运行时依赖 yarn add <package> npm i –S / --save <package>
        添加开发时依赖 yarn add -D <package> npm i –D / --save-dev <package>
        更新依赖 yarn [global] upgrade <package> npm update [-g][–S/–D] <package>
        卸载 yarn[global] remove <package> npm un / uninstall [-g][–S/–D] <package>
        重装/升级 yarn upgrade rm -rf node_modules && npm install
        清除缓存 yarn cache clean npm cache clean
        查看包信息 yarn info <package> npm info <package>
        查看配置信息 yarn config list npm config list
        获取指定配置信息 yarn config get registry npm config get registry

        # 运行 package.json 中的脚本

        比如 package.json 中包含以下内容:

        {
  "scripts": {
    "dev": "node build/dev-server.js"
  }
}

        此时，只需执行 npm run dev 或 yarn (run) dev，即可运行 node build/dev-server.js。

        scripts 中有一个特殊的命令: start。

        比如，将上述代码改为

        {
  "scripts": {
    "start": "node build/dev-server.js"
  }
}

        此时，只需执行 npm start 或 yarn start，即可运行 node build/dev-server.js。
