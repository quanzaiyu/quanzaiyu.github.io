    # node 包管理工具(npm/yarn)换源

    # npm 换源

    # 常用源

    npm 原始镜像

    • 搜索地址: https://www.npmjs.com
    • registry地址: https://registry.npmjs.org

    yarn 原始镜像

    • 搜索地址: https://yarnpkg.com/zh-Hans/packages
    • registry地址: https://registry.yarnpkg.com

    淘宝 npm 镜像

    • 搜索地址: http://npm.taobao.org
    • registry地址: http://registry.npm.taobao.org/

    cnpmjs 镜像

    • 搜索地址: http://cnpmjs.org
    • registry地址: http://r.cnpmjs.org/

    # 查看源

    $ npm config list # 查看 npm 所有设置
$ npm config get registry # 查看 npm 指定设置

    # 更换源

    第1种方法 直接更换 npm 源

    $ npm config set registry https://registry.npm.taobao.org

    第2种方法 另行安装 cnpm，并注册淘宝源

    $ npm install -g cnpm --registry=https://registry.npm.taobao.org

    第3种方法 修改 .npmrc

    在此文件中加入：

    registry='https://registry.npm.taobao.org'

    此方法适用于 npm，对 yarn 同样生效。

    第4中方法 临时使用

    $ npm install express --registry https://registry.npm.taobao.org

    第5种方法 使用别名配置 npm

    $ alias cnpm="npm --registry=https://registry.npm.taobao.org \
--cache=$HOME/.npm/.cache/cnpm \
--disturl=https://npm.taobao.org/dist \
--userconfig=$HOME/.cnpmrc"

# Or alias it in .bashrc or .zshrc
$ echo '\n#alias for cnpm\nalias cnpm="npm --registry=https://registry.npm.taobao.org \
  --cache=$HOME/.npm/.cache/cnpm \
  --disturl=https://npm.taobao.org/dist \
  --userconfig=$HOME/.cnpmrc"' >> ~/.zshrc && source ~/.zshrc

    之后使用 cnpm 进行安装即可, 这样不需要重新安装 cnpm 即可使用此命令

    # yarn 换源

    # 查看源

    $ yarn config list # 查看 yarn 所有设置
$ yarn config get registry # 查看 yarn 指定设置

    # 更换源

    第1种方法 直接更换源, 跟 npm 类似：

    $ yarn config set registry https://registry.npm.taobao.org

    第2种方法 另行安装 cYarn

    cYarn 是使用 cnpm 源的 Yarn。

    安装 cYarn：

    $ npm install -g cyarn # 使用 cnpm 默认源进行安装
$ npm install -g cyarn --registry=https://registry.npm.taobao.org # 使用淘宝源进行安装

    第3种方法 使用 yrm

    $ npm install -g yrm
$ yrm use cnpm

    # 更换 node-sass 等包的安装源

    第1种方法 更换源

    $ npm config set sass_binary_site https://npm.taobao.org/mirrors/node-sass/
$ npm config set phantomjs_cdnurl https://npm.taobao.org/mirrors/phantomjs/
$ npm config set electron_mirror https://npm.taobao.org/mirrors/electron/
$ npm config set chromedriver_cdnurl http://cdn.npm.taobao.org/dist/chromedriver

    第2种方法 安装时指定

    $ npm i node-sass --sass_binary_site=https://npm.taobao.org/mirrors/node-sass/

// 也可以设置系统环境变量的方式。示例
// linux、mac 下
$ SASS_BINARY_SITE=https://npm.taobao.org/mirrors/node-sass/ npm install node-sass

// window 下
$ set SASS_BINARY_SITE=https://npm.taobao.org/mirrors/node-sass/ && npm install node-sass

    第3种方法 使用VPN

    $ npm config set proxy=http://127.0.0.1:1080 #此处是VPN的代理地址
$ npm i node-sass

    下载完成后删除 http 代理

    $ npm config delete proxy

    # 错误解决

    提示没有安装python、build失败等

    执行

    $ npm install -g node-gyp
$ npm install --global --production windows-build-tools
$ npm rebuild node-sass
