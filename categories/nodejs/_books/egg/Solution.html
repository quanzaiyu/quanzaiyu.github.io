# 使用 WebStrom 不能热更新的问题

开发期默认开启了 development 插件，修改后端代码后，会自动重启 Worker 进程。

但是在 WebStrom 中并不生效, 可以设置:

路径: File | Settings | Appearance & Behavior | System Settings

取消勾选 Use "safe write"
