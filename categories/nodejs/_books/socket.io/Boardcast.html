# 广播消息

广播消息是服务器端向客户端进行发送的, 若想要客户端发送广播, 可以设置触发条件为接收到客户端消息时进行广播

# 全局广播

服务器端

有两种发送广播消息的方式:

  1. 通过 io.emit 进行广播, 这将向大厅和房间广播
  2. 通过 socket.broadcast.emit 进行广播, 这将向除了自己之外的其它客户端广播
io.emit('chat message', '您有新的消息');
socket.broadcast.emit('new user', '有新伙伴加入');

客户端

服务器端通过监听对应的方法接收通知:

socket.on('chat message', (msg) => {
  console.log(msg)
});
socket.on('new user', (msg) => {
  console.log(msg)
});

# 命名空间内广播

通过 of 可以指定在哪个命名空间中进行广播:

io.of('/news').emit('message',msg);

# 房间内广播

根据前面说的命名空间, 可以通过 in 指定在哪个房间进行广播

io.in('myRoom').emit('message', msg); // 根命名空间下的 myRoom 房间
io.of('/news').in('myRoom').emit('message',msg); // news 命名空间下的 myRoom 房间

向房间内广播, 从客户端的角度来提交事件, 提交者会排除在外 (to)

// 向myRoom广播一个事件，在此房间内除了自己外的所有客户端都会收到消息
socket.broadcast.to('myroom').emit('message', msg);
