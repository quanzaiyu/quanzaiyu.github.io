# 安装与使用

新建一个项目名叫myapp，进入此目录

$ npm init
$ yarn add koa
$ touch index.js
$ vim index.js

编写:

var koa = require('koa');
var app = new koa();
app.use(function* () {
  this.body = 'Hello World';
});
app.listen(3000);

或者

const Koa = require('koa');
const app = new Koa();
app.use(async (ctx, next) => {
  ctx.response.body = 'Hello World';
  // or
  // ctx.body = 'Hello World';
});
app.listen(3000);

运行node index，在浏览器输入http://localhost:3000/即可看到输出Hello World。

# 使用生成器生成项目

$ yarn global add koa-generator
$ koa2 <my-app>
$ cd <my-app> && npm i
$ npm start
