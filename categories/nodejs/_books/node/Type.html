# 数据类型

ECMAScript中包括以下数据类型

  • 字符串(String)
  • 数字(Number)
  • 布尔(Boolean)
  • 数组(Array)
  • 对象(Object)
  • 空(Null)
  • 未定义(Undefined)

其中 String、Number、Boolean、Null、Undefined 是基础类型，Array、Object 是引用类型。

# 字符串

字符串是存储字符的变量，可以使用单引号 ' 或 双引号 " 字面量声明。

var name = 'xiaoyu'

也可使用字符串模板(ES6)声明

var a = 'xiaoyu'
var content = `my name is ${a} ` // 'my name is xiaoyu '

使用构造函数声明

var a = new String('hello')
console.log(a) // [String: 'hello']，注意这里是String对象的形式
console.log(a + ' world') // hello5，注意这里有一个默认的拆箱操作

# 数字

var x1 = 34.00;      // 浮点数
var x2 = 34;         // 整数
// 科学计数法
var y=123e5;      // 12300000
var z=123e-5;     // 0.00123

# 布尔

var x = true;
var y = false;

# Null、Undefined

只包含null和undefined

# 转化为数值

使用parseInt、parseFloat将非数值类型转化为数值，使用toFixed将数值保留指定位数的小数。

console.log(parseInt('1.01')) // 1
console.log(parseFloat('1.01')) // 1.01

console.log(parseInt(true)) // NaN
console.log(parseInt(false)) // NaN
console.log(parseFloat(true)) // NaN
console.log(parseFloat(false)) // NaN

console.log(parseInt(null)) // NaN
console.log(parseInt(undefined)) // NaN
console.log(parseFloat(null)) // NaN
console.log(parseFloat(undefined)) // NaN

console.log(1.01.toFixed(0)) // 1
console.log(1.01.toFixed(4)) // 1.0100

# 转化为字符串

使用toString可以将其他类型转化为字符串，也可直接加空字符串进行转换。

console.log(1.01.toString()) // '1.01'
console.log(true.toString()) // 'true'
console.log(false.toString()) // 'false'
console.log(new Object().toString()) // [object Object]

// console.log(null.toString()) // 报错
// console.log(undefined.toString()) // 报错

console.log(1.01 + '') // '1.01'
