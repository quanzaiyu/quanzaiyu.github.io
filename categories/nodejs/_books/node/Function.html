# 函数是"一等公民"

实际上，在 JavaScript 中，没有所谓的第二类对象。JavaScript是完全的面向对象的编程语言，几乎所有在这门语言中存在的东西都是对象，因此函数也是对象，你可以设置它的属性(Properties)，把它当作参数去传递以及返回一个函数（对象）。

出自 Pedro Teixeira. Node.js 实战 (中国程序员 6) (Kindle 位置 392-394). Kindle 版本.
