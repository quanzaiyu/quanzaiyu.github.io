Vue 路由笔记

vue-router 主要用于创建单页应用，在结合 vue.js 的时候显得得心应手。

GitHub、中文文档

安装

CDN

https://unpkg.com/vue-router/dist/vue-router.js
1

NPM

npm i vue-router --save
1

使用

import Vue from 'vue'
import VueRouter from 'vue-router'

Vue.use(VueRouter)

基本概念

  • router-view 路由入口
  • router-link 路由链接

简单使用

  1. 定义路由组件
  2. 定义路由
  3. 创建 router 实例，然后传 routes 配置
  4. 创建和挂载根实例。
const Foo = { template: '<div>foo</div>' }
const Bar = { template: '<div>bar</div>' }

const routes = [
  { path: '/foo', component: Foo },
  { path: '/bar', component: Bar }
]

const router = new VueRouter({
  routes
})

const app = new Vue({
  router
}).$mount('#app')

模块化使用

  1. 导入 Vue 和 VueRouter，调用 Vue.use(VueRouter)
  2. 引入路由组件
  3. 定义路由
  4. 创建 router 实例，然后传 routes 配置
  5. 创建和挂载根实例。

  <p>
    <router-link to="/foo">Go to Foo</router-link>
    <router-link to="/bar">Go to Bar</router-link>

  <router-view></router-view>
</div>

import Vue from 'vue'
import Router from 'vue-router'

import foo from './foo.vue'
import bar from './bar.vue'

const mode = 'history'
const hashbang = true

const routes = [
  {
    path: '/foo',
    name: 'foo',
    component: foo
  },
  {
    path: '/bar',
    name: 'bar',
    component: foo
  }
]

const router = new Router({
  hashbang,
  mode,
  routes
})

Vue.use(Router)

const app = new Vue({
  router,
  el: '#app'
})

$route vs $router

使用模块注册 vue-router，将会在 Vue.prototype 中注入一个 $route 和 $router 的只读属性。

其中，$route 用于监测当前路由，$router 用于进行路由操作，事实上，$route 与 $router.currentRoute 相同。
