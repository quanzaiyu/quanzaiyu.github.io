$ 模板引擎

$ 文本插值渲染

Vue.js 的核心是一个允许采用简洁的模板语法来声明式地将数据渲染进 DOM 的系统：

<div id="app">
  {{ message }}
</div>
1
2
3
var app = new Vue({
  el: '#app',
  data: {
    message: 'Hello Vue!'
  }
})

在 Vue 中，数据和 DOM 已经被建立了关联，所有东西都是响应式的。

$ 元素属性绑定

除了文本插值之外，数据还可用于元素属性绑定:

<div id="app-2">
  <span v-bind:title="message">
    鼠标悬停几秒钟查看此处动态绑定的提示信息！
  </span>
</div>
