# PHP 函数

PHP 函数使用 function 声明:

function p () {
	echo 'Hello world';
}

p();

# 函数的参数

function p ($name) {
	echo $name;
}

p('quanzaiyu');

# 函数的返回值

函数的返回值使用 return 语句:

function add($x, $y) {
    $total = $x+$y;
    return $total;
}

echo "1 + 16 = " . add(1,16);
