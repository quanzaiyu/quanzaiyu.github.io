# PHP 日期时间

在PHP中，操作日期时间是非常容易的，甚至可以使用自然语言的方式进行操作。

# 获取当前时间

使用 time() 方法获取当前 Unix 时间戳，使用 date() 方法进行格式化输出时间：

$now = time(); // 1536132601
$time = date('Y-m-d H:i:s',$now); // 2018-09-05 15:31:24

date() 函数用于格式化时间/日期，返回将整数 timestamp 按照给定的格式字串而产生的字符串。

或者使用 strtotime('now') 直接获取当前时间戳：

date(strtotime('now')); // 2018-09-05 15:31:24

如果传入的是一个 DateTime 对象，可以使用 date_format() 方法进行格式化：

date_format(new \DateTime(), 'Y-m-d H:i:s'); // 2018-09-06 16:05:10

# 获取指定时间

使用 getdate() 方法可以获取指定时间戳的详细数据

array getdate ([ int $timestamp = time() ] )

如:

$today = getdate();
print_r($today);

输出

Array
(
    [seconds] => 40
    [minutes] => 58
    [hours]   => 21
    [mday]    => 17
    [wday]    => 2
    [mon]     => 6
    [year]    => 2003
    [yday]    => 167
    [weekday] => Tuesday
    [month]   => June
    [0]       => 1055901520
)

# 昨天、今天、明天

使用 strtotime() 方法可以以自然语言的方式操作时间，比如获取昨天、今天、明天:

date('Y-m-d H:i:s', strtotime('yesterday')); // 2018-09-05 00:00:00
date('Y-m-d H:i:s', strtotime('today')); // 2018-09-06 00:00:00
date('Y-m-d H:i:s', strtotime('tomorrow')); // 2018-09-07 00:00:00

这样的话，就可以获取到昨天、今天、明天的凌晨0点

当然，不嫌麻烦的话也可使用时间戳进行计算：

$nextDay = time() + (1 * 24 * 60 * 60)

# 时间之前、时间之后

这里我们将见识到 strtotime 的无比强大之处：

date('Y-m-d H:i:s', strtotime('-1 day')); // 一天前 2018-09-05 15:01:51
date('Y-m-d H:i:s', strtotime('+1 day')); // 一天后 2018-09-07 15:01:51
date('Y-m-d H:i:s', strtotime('-1 week')); // 一周前 2018-08-30 15:01:51
date('Y-m-d H:i:s', strtotime('+1 week')); // 一周后 2018-09-13 15:01:51

strtotime 支持第二参数，用于指定基准时间，比如：

date('Y-m-d H:i:s', strtotime('-1 day', strtotime('last year')); // 去年今天的前一天 2017-09-05 15:01:51

使用 ago 修饰词可以设置一段时间之前，注意这些时间词可以加复数也可不加，意义一致，比如 day 和 days 有相同的效果

date('Y-m-d H:i:s', strtotime('5 days ago')); // 2018-09-01 15:10:12
date('Y-m-d H:i:s', strtotime('3 weeks ago')); // 2018-08-16 15:10:12
date('Y-m-d H:i:s', strtotime('1 year ago')); // 2017-09-06 15:10:12

使用 first、last、previous、next 修饰词可以获取 第一个...、最后一个/上一个...、前一个...、下一个...

date('Y-m-d H:i:s', strtotime('last day')); // 2018-09-05 15:14:07
date('Y-m-d H:i:s', strtotime('next day')); // 2018-09-07 15:14:07
date('Y-m-d H:i:s', strtotime('previous week')); // 2018-08-27 15:26:17

date('Y-m-d H:i:s', strtotime('last sat of July 2008')); // 2008-07-26 00:00:00
date('Y-m-d H:i:s', strtotime('last day of next month')); // 2018-10-31 15:16:32
date('Y-m-d H:i:s', strtotime('first day of January 2008')); // 2008-01-01 00:00:00
date('Y-m-d H:i:s', strtotime('first day of this month')); // 2018-09-01 15:16:32

# 设置时间

使用 noon、midnight 修饰词可以将时间设置为中午、午夜

date('Y-m-d H:i:s', strtotime('today noon')); // 2018-09-06 12:00:00
date('Y-m-d H:i:s', strtotime('today midnight')); // 2018-09-06 00:00:00

也可以使用具体时间设置：

date('Y-m-d H:i:s', strtotime('today 14:30')); // 2018-09-06 14:30:00

使用 front、back 可以设置一刻钟（15min）

date('Y-m-d H:i:s', strtotime('front of 5am')); // 2018-09-06 04:45:00
date('Y-m-d H:i:s', strtotime('front of 23')); // 2018-09-06 22:45:00
date('Y-m-d H:i:s', strtotime('back of 7pm')); // 2018-09-06 19:15:00
date('Y-m-d H:i:s', strtotime('back of 15')); // 2018-09-06 15:15:00

如果是一个 DateTime 对象，可以使用 date_date_set() 方法进行时间设置：

$date = date_create();
date_date_set($date, 2001, 2, 3);
echo date_format($date, 'Y-m-d'); // 2001-02-03

# DateTime 对象

创建一个 DateTime 对象，可以使用以下两种方式：

方法一：直接new一个DateTime对象

$date = new DateTime('2001-01-01');
$date->setTime(14, 55, 24); // 2001-01-01 14:55:00
echo $date->format('Y-m-d H:i:s'); // 2001-01-01 14:55:24

方法二：使用date_create()方法创建DateTime对象

$date = date_create('2001-01-01');
date_time_set($date, 14, 55, 24); // 2001-01-01 14:55:00
echo date_format($date, 'Y-m-d H:i:s'); // 2001-01-01 14:55:24

# 时间戳

使用 mktime() 方法可以获取指定日期的 Unix 时间戳。

格式为：

int mktime(int $hour = date("H"), int $minute = date("i"), int $second = date("s"), int $month = date("n"), int $day = date("j"), int $year = date("Y"), int $is_dst = -1)

参数可以从右向左省略，任何省略的参数会被设置成本地日期和时间的当前值。

使用 time() 方法返回自从 Unix 纪元（格林威治时间 1970 年 1 月 1 日 00:00:00）到当前时间的秒数，使用 strtotime() 方法将任何字符串的日期时间描述解析为 Unix 时间戳。上面用得多了，不赘述了。

如果要从一个 DateTime 对象获取时间戳，可以使用 date_timestamp_get() 方法：

date_timestamp_get(date_create());

# 格式检查

使用 checkdate() 检查由参数构成的日期的合法性。如果每个参数都正确定义了则会被认为是有效的。

语法:

bool checkdate ( int $month , int $day , int $year )

如:

var_dump(checkdate(12, 31, 2000)); // true
var_dump(checkdate(2, 29, 2001)); // false

# 时间运算

  • DateTime date_add ( DateTime $object , DateInterval $interval )
  • DateTime date_sub ( DateTime $object , DateInterval $interval )

举例：

$date = date_create('2000-01-01');
date_add($date, date_interval_create_from_date_string('10 days'));
echo date_format($date, 'Y-m-d'); // 2000-01-11

$date = date_create('2000-01-20');
date_sub($date, date_interval_create_from_date_string('10 days'));
echo date_format($date, 'Y-m-d'); // 2000-01-10

# 时间比较

# 直接比较

$date1 = new DateTime("now");
$date2 = new DateTime("tomorrow");

var_dump($date1 == $date2); // bool(false)
var_dump($date1 < $date2); // bool(true)
var_dump($date1 > $date2); // bool(false)

# date_diff

使用 date_diff() 方法可以直接算出两个时间之间的差值：

$datetime1 = date_create('2009-10-11');
$datetime2 = date_create('2009-10-13');
$interval = date_diff($datetime1, $datetime2);
echo $interval->format('%R%a days'); // +2 days

# 时间修改

使用 date_modify() 可以修改时间：

$date = date_create('2006-12-12');
date_modify($date, '+1 day');
echo date_format($date, 'Y-m-d'); // 2006-12-13

# 常用时间相关字串

单位：

'sec' | 'second' | 'min' | 'minute' | 'hour' | 'day' | 'fortnight' | 'forthnight' | 'month' | 'year' | 'week' | 'weekday') 's'

月份的全写及其简写：

January | February | March | April | May | June | July | August | September | October | November | December
Jan     | Feb      | Mar   | Apr   | May | Jun  | Jul  | Aug    | Sep       | Oct     | Nov      | Dec

星期的全写及其简写：

sunday | monday | tuesday | wednesday | thursday | friday | saturday
sun    | mon    | tue     | wed       | thu      | fri    | sat

序数词：

first | second | third | fourth | fifth | sixth | seventh | eighth | ninth | tenth | eleventh | twelfth

其他相关词：

next | last | previous | this

# 常用时间格式字串

  • Y - 年 (四位数)
  • m - 月 (01 - 12)
  • d - 日 (01 - 31)
  • H - 24 小时制（00 到 23）
  • h - 12 小时制（01 到 12）
  • i - 分（00 到 59）
  • s - 秒（00 到 59）
  • M - 三个字母缩写表示的月份，Jan 到 Dec
  • y - 年 (二位数)
  • n - 月，没有前导零 (1 到 12)
  • j - 日，没有前导零 (1 到 31)
  • g - 12 小时制，没有前导零，1 到 12
  • G - 24 小时制，没有前导零，0 到 23
  • a - 小写的上午和下午值，am 或 pm
  • A - 大写的上午和下午值，AM 或 PM
  • w - 星期几，数字表示，0（表示星期天）到 6（表示星期六）
  • l - 星期几，完整的文本格式，Sunday 到 Saturday
  • D - 星期几，3 个字母简写，Mon 到 Sun
  • t - 指定的月份有几天，28 到 31
  • L - 是否为闰年，如果是闰年为 1，否则为 0
