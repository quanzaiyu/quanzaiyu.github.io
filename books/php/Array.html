# PHP 数组

# 数值数组

这里有两种创建数值数组的方法：

自动分配 ID 键（ID 键总是从 0 开始）：

$cars=array("Volvo","BMW","Toyota");

// or

$cars = ["Volvo","BMW","Toyota"];

print_r($cars); // Array ( [0] => Volvo [1] => BMW [2] => Toyota )

手动分配 ID 键：

$cars[0]="Volvo";
$cars[1]="BMW";
$cars[5]="Toyota";

print_r($cars); // Array ( [0] => Volvo [1] => BMW [5] => Toyota )

# 关联数组

关联数组是使用分配给数组的指定的键的数组。

对应其他语言的 Map、Dict 等数据类型，对应 JS 的 Object。

这里有两种创建关联数组的方法：

$ages=array("Peter"=>"35","Ben"=>"37","Joe"=>"43");

print_r($ages); // Array ( [Peter] => 35 [Ben] => 37 [Joe] => 43 )

或者:

$ages['Peter']="35";
$ages['Ben']="37";
$ages['Joe']="43";

print_r($ages); // Array ( [Peter] => 35 [Ben] => 37 [Joe] => 43 )

# 多维数组

普通的二维数组

$cars = array(
  array("Volvo",100,96),
  array("BMW",60,59),
  array("Toyota",110,100)
);

print_r($cars); // Array ( [0] => Array ( [0] => Volvo [1] => 100 [2] => 96 ) [1] => Array ( [0] => BMW [1] => 60 [2] => 59 ) [2] => Array ( [0] => Toyota [1] => 110 [2] => 100 ) )

多维数组可以组合使用关联数组和数值数组

$families = array(
  "Griffin"=>array("Peter", "Lois", "Megan"),
  "Quagmire"=>array("Glenn"),
  "Brown"=>array("Cleveland", "Loretta", "Junior")
);

print_r($families); // Array ( [Griffin] => Array ( [0] => Peter [1] => Lois [2] => Megan ) [Quagmire] => Array ( [0] => Glenn ) [Brown] => Array ( [0] => Cleveland [1] => Loretta [2] => Junior ) )

echo $families['Griffin'][2]; // Megan

# 获取数组长度

使用 count() 获取数组长度

$cars=array("Volvo","BMW","Toyota");

echo count($cars); // 3

# 数组填充: array_fill

array_fill(index,number,value) 函数用给定的键值填充数组。

$a1=array_fill(3,4,"blue");
print_r($a1);

输出:

Array
(
  [3] => blue
  [4] => blue
  [5] => blue
  [6] => blue
)

# 数组填充: array_fill_keys

array_fill_keys(keys,value) 函数用给定的指定键名的键值填充数组。

$keys=array("a","b","c","d");
$a1=array_fill_keys($keys,"blue");
print_r($a1);

输出:

Array
(
  [a] => blue
  [b] => blue
  [c] => blue
  [d] => blue
)

# 数组遍历

# 普通 for 循环遍历数组

$cars=array("Volvo","BMW","Toyota");

for($x=0; $x<count($cars); $x++) {
  echo $cars[$x]." ";
} // Volvo BMW Toyota

# 使用 foreach 循环遍历数组

遍历数值数组

$cars=array("Volvo","BMW","Toyota");

foreach($cars as $i => $v) {
  echo "Index=" . $i . ", Value=" . $v . " ";
} // Index=0, Value=Volvo Index=1, Value=BMW Index=2, Value=Toyota

遍历关联数组

$ages=array("Peter"=>"35","Ben"=>"37","Joe"=>"43");

foreach($ages as $k => $v) {
  echo "Key=" . $k . ", Value=" . $v . " ";
} // Key=Peter, Value=35 Key=Ben, Value=37 Key=Joe, Value=43

遍历的时候可以省略 key 或 index:

$cars=array("Volvo","BMW","Toyota");

foreach($cars as $v) {
  echo "Value=" . $v . ", ";
} // Value=Volvo, Value=BMW, Value=Toyota,

$ages=array("Peter"=>"35","Ben"=>"37","Joe"=>"43");

foreach($ages as $v) {
  echo "Value=" . $v . ", ";
} // Value=35, Value=37, Value=43,

# 数组排序

在 PHP 中，有以下排序函数:

  • sort() - 对数组进行升序排列
  • rsort() - 对数组进行降序排列
  • asort() - 根据关联数组的值，对数组进行升序排列
  • arsort() - 根据关联数组的值，对数组进行降序排列
  • ksort() - 根据关联数组的键，对数组进行升序排列
  • krsort() - 根据关联数组的键，对数组进行降序排列

譬如说:

$cars=array("Volvo","BMW","Toyota");
sort($cars);

print_r($cars); // Array ( [0] => BMW [1] => Toyota [2] => Volvo )

# 数组连接

将数组元素连接成的字符串，格式:

implode(separator,array)
join(separator,array)

示例

implode(" ", array('Hello','World!')) // 'Hello World!'

# 数组过滤

array_filter ( array $array [, callable $callback [, int $flag = 0 ]] ) 使用回调函数过滤数组中的元素。

该函数把输入数组中的每个键值传给回调函数。如果回调函数返回 true，则把输入数组中的当前键值返回给结果数组。数组键名保持不变。

$a1=array("a","b",2,3,4);
print_r(array_filter($a1,function($var)
{
  return($var > 1);
}));

输出:

Array
(
    [2] => 2
    [3] => 3
    [4] => 4
)

# 数组比较: array_diff

array_diff() 函数用于比较两个（或更多个）数组的键值，并返回差集。

格式: array_diff(array1,array2,array3...);

该函数比较两个（或更多个）数组的键值，并返回一个差集数组，该数组包括了所有在被比较的数组（array1）中，但是不在任何其他参数数组（array2 或 array3 等等）中的键值。

示例:

$a1=array("a"=>"red","b"=>"green","c"=>"blue","d"=>"yellow");
$a2=array("e"=>"red","f"=>"green","g"=>"blue");

$result=array_diff($a1,$a2);
print_r($result);

输出:

Array
(
  [d] => yellow
)

# 数组比较: array_diff_assoc

array_diff_assoc() 函数用于比较两个（或更多个）数组的键名和键值 ，并返回差集。

格式: array_diff_assoc(array1,array2,array3...);

该函数比较两个（或更多个）数组的键名和键值，并返回一个差集数组，该数组包括了所有在被比较的数组（array1）中，但是不在任何其他参数数组（array2 或 array3 等等）中的键名和键值。

$a1=array("a"=>"red","b"=>"green","c"=>"blue","d"=>"yellow");
$a2=array("a"=>"red","b"=>"green","c"=>"blue");

$result=array_diff_assoc($a1,$a2);
print_r($result);

输出:

Array
(
  [d] => yellow
)

# 数组比较: array_diff_uassoc

array_diff_uassoc() 函数用于比较两个（或更多个）数组的键名和键值 ，并返回差集。

格式: array_diff_uassoc(array1,array2,array3...,myfunction);

TIP

注释：该函数使用用户自定义函数比较键名！

该函数比较两个（或更多个）数组的键名和键值，并返回一个差集数组，该数组包括了所有在被比较的数组（array1）中，但是不在任何其他参数数组（array2 或 array3 等等）中的键名和键值。

$a1=array("a"=>"red","b"=>"green","c"=>"blue");
$a2=array("d"=>"red","b"=>"green","e"=>"blue");

$result=array_diff_uassoc($a1,$a2,function($a,$b)
{
  if ($a===$b)
  {
    return 0;
  }
  return ($a>$b)?1:-1;
});
print_r($result);

输出:

Array
(
  [a] => red
  [c] => blue
)

# 数组比较: array_intersect

array_intersect() 函数用于比较两个（或更多个）数组的键值，并返回交集。

格式: array_intersect(array1,array2,array3...);

该函数比较两个（或更多个）数组的键值，并返回一个交集数组，该数组包括了所有在被比较的数组（array1）中，同时也在任何其他参数数组（array2 或 array3 等等）中的键值。

$a1=array("a"=>"red","b"=>"green","c"=>"blue","d"=>"yellow");
$a2=array("e"=>"red","f"=>"green","g"=>"blue");

$result=array_intersect($a1,$a2);
print_r($result);

输出:

Array
(
  [a] => red
  [b] => green
  [c] => blue
)

# array_change_key_case

array_change_key_case(array,case) 将数组的所有的键都转换为大写字母或小写字母。

case:

CASE_LOWER - 默认值。将数组的键转换为小写字母。

CASE_UPPER - 将数组的键转换为大写字母。

$age=array("Peter"=>"35","Ben"=>"37","Joe"=>"43");
print_r(array_change_key_case($age,CASE_UPPER));

输出

Array
(
    [PETER] => 35
    [BEN] => 37
    [JOE] => 43
)

# array_chunk

array_chunk(array,size,preserve_keys) 函数把一个数组分割为新的数组块。

$age=array("Peter"=>"35","Ben"=>"37","Joe"=>"43","Harry"=>"50");
print_r(array_chunk($age,2,true));

输出

Array
(
  [0] => Array
  (
    [Peter] => 35
    [Ben] => 37
  )
  [1] => Array
  (
    [Joe] => 43
    [Harry] => 50
  )
)

# array_combine

array_combine(keys,values) 通过合并两个数组来创建一个新数组，其中的一个数组元素为键名，另一个数组的元素为键值。

注释

键名数组和键值数组的元素个数必须相同！

$fname=array("Peter","Ben","Joe");
$age=array("35","37","43");

$c=array_combine($fname,$age);
print_r($c);

输出:

Array
(
  [Peter] => 35
  [Ben] => 37
  [Joe] => 43
)

# array_count_values

array_count_values(array) 用于统计数组中所有值出现的次数。

$a=array("A","Cat","Dog","A","Dog");
print_r(array_count_values($a));

输出:

Array
(
    [A] => 2
    [Cat] => 1
    [Dog] => 2
)

# array_flip

array_flip(array) 用于反转/交换数组中的键名和对应关联的键值。

$a1=array("a"=>"red","b"=>"green","c"=>"blue","d"=>"yellow");
$result=array_flip($a1);
print_r($result);

输出:

Array
(
    [red] => a
    [green] => b
    [blue] => c
    [yellow] => d
)

# array_column

array_column(array,column_key,index_key) 返回输入数组中某个单一列的值。

不指定键名:

$a = array(
  array(
    'id' => 5698,
    'first_name' => 'Peter',
    'last_name' => 'Griffin',
  ),
  array(
    'id' => 4767,
    'first_name' => 'Ben',
    'last_name' => 'Smith',
  ),
  array(
    'id' => 3809,
    'first_name' => 'Joe',
    'last_name' => 'Doe',
  )
);

$last_names = array_column($a, 'last_name');
print_r($last_names);

输出:

Array
(
  [0] => Griffin
  [1] => Smith
  [2] => Doe
)

指定键名:

// 可能从数据库中返回数组
$a = array(
  array(
    'id' => 5698,
    'first_name' => 'Peter',
    'last_name' => 'Griffin',
  ),
  array(
    'id' => 4767,
    'first_name' => 'Ben',
    'last_name' => 'Smith',
  ),
  array(
    'id' => 3809,
    'first_name' => 'Joe',
    'last_name' => 'Doe',
  )
);

$last_names = array_column($a, 'last_name', 'id');
print_r($last_names);

输出:

Array
(
  [5698] => Griffin
  [4767] => Smith
  [3809] => Doe
)
