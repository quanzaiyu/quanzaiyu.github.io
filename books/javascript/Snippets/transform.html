      # 转化函数

      # 将数字转化为整数数组

      将数字转换为字符串，使用 split() 来转换构建一个数组。 使用 Array.map() 和 parseInt() 将每个值转换为整数。

      const digitize = n => (''+n).split('').map(i => parseInt(i));

digitize(2334) // [2, 3, 3, 4]

      # 将一个字符串转换为数组的简易方法

      使用 ES6 的扩展操作符可以很容易的将一个字符串(若不是字符串自动转化为字符串)转化为数组。

      const strToArr = str => [...(str+'')]

strToArr('abc') // ["a", "b", "c"]
strToArr(123) // ["1", "2", "3"]

      但是这是将字符串逐一分割，没法指定分隔符，如果需要指定分隔符则需要使用 split() 方法。
