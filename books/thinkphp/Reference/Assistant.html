# 助手函数配置

核心框架不依赖任何助手函数，系统只是默认加载了助手函数，配置如下：

// 扩展函数文件定义
'extra_file_list'   => [THINK_PATH . 'helper' . EXT],

因此，可以随意修改助手函数的名称或者添加自己的助手函数，然后修改配置为：

// 扩展函数文件定义
'extra_file_list'   =>  [APP_PATH . 'helper' . EXT],

// 使用扩展函数文件
'extra_file_list'   => [
    THINK_PATH . 'helper' . EXT,
    APP_PATH . 'helper' . EXT
],
