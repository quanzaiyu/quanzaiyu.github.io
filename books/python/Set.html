# 数据类型 - 集合(set)

set是一组key的集合(无序无重复元素集合)，key不能重复，但不存储value。

# set的创建

# 字面量创建

basket = {'apple', 'orange', 'apple', 'pear', 'orange', 'banana'}

# 构造函数创建

使用set()构造函数创建，传入一个list

>>> s = set([1, 2, 3])
>>> s
{1, 2, 3}

::: warning 注意 如果要创建一个空集合，必须用 set() 而不是 {} ；后者创建一个空的字典。 :::

# set元素的增加与删除

>>> s.add(4)
>>> s
{1, 2, 3, 4}
>>> s.add(4)
>>> s
{1, 2, 3, 4}
>>> s.remove(4)
>>> s
{1, 2, 3}
  • add(key) 可以添加元素到set中，可以重复添加，但不会有效果
  • remove(key) 可以删除元素
