# 删除数据

# 根据主键删除数据

使用 delete 先查找再删除

$user = User::get(17);
$user->delete();
1
2

转化为 SQL 为

DELETE FROM `user` WHERE `id` = 17
1

或者直接调用静态方法 destroy

User::destroy(1);
// 支持批量删除多个数据
User::destroy('1,2,3');
// 或者
User::destroy([1,2,3]);
1
2
3
4
5

WARNING

当 destroy 方法传入空值（包括空字符串和空数组）的时候不会做任何的数据删除操作，但传入0则是有效的

# 条件删除

还支持使用闭包删除，例如：

User::destroy(function($query){
  $query->where('id','>',10);
});
1
2
3

或者通过数据库类的查询条件删除

User::where('id','>',10)->delete();
1

WARNING

直接调用数据库的 delete 方法的话无法调用模型事件。

# 最佳实践

TIP

删除的最佳实践原则是：如果删除当前模型数据，用delete方法，如果需要直接删除数据，使用destroy静态方法。
