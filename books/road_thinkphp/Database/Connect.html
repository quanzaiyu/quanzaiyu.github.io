连接数据库

Db 类是数据库操作的入口类，一定是静态调用，数据库类定义文件位于 thinkphp/library/think/Db.php。

假设有数据库 :

全局配置

配置好数据库连接文件 database.php 后，就可在控制器中进行数据库连接。

使用 table 语句选择数据表。

测试是否连接正确，在控制器中 :

namespace app\index\controller;

use think\Controller;
use think\Db;

class Index extends Controller
{
    public function index()
    {
      return Db::table('user')
        ->where('id', 1)
        ->value('name');
    }
}

动态配置

如果没有配置 连接文件 database.php，或者需要一次性连接某个数据库，可以使用 动态配置。

namespace app\index\controller;

use think\Controller;
use think\Db;

class Index extends Controller
{
    public function index()
    {
      return Db::connect([
        'type'            => 'mysql',
        'hostname'        => '127.0.0.1',
        'database'        => 'test',
        'username'        => 'root',
        'password'        => 'root'
      ])
        ->table('user')
        ->where('id', 1)
        ->value('name');
    }
}

DSN 连接

使用以下格式的连接字符串 :

数据库类型://用户名:密码@数据库地址:端口号/数据库名称#字符集
1

进行连接。

如:

namespace app\index\controller;

use think\Controller;
use think\Db;

class Index extends Controller
{
    public function index()
    {
      $dsn = 'mysql://root:root@127.0.0.1:3306/test#utf8';
      return Db::connect($dsn)
        ->table('user')
        ->where('id', 2)
        ->value('name');
    }
}
