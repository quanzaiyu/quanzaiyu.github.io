# 请求参数

# 请求参数类型

变量类型方法包括：

方法 描述
param 获取当前请求的变量
get 获取 $_GET 变量
post 获取 $_POST 变量
put 获取 PUT 变量
delete 获取 DELETE 变量
session 获取 $_SESSION 变量
cookie 获取 $_COOKIE 变量
request 获取 $_REQUEST 变量
server 获取 $_SERVER 变量
env 获取 $_ENV 变量
route 获取 路由（包括PATHINFO） 变量
file 获取 $_FILES 变量

# 获取全部参数

这里均使用 Controller 中的 request 获取请求参数示例。

首先定义如下路由，方便测试

Route::get('hello', 'index/index/test')->name('hello_page');
1

通过对应的请求方式获取所有指定请求方式的参数，如果是 param 者获取所有参数。

dump($this->request->get());
1

浏览器输入 http://www.my-tp5.com/hello?test=yu，将返回

array(1) {
  ["test"] => string(2) "yu"
}

# 获取指定参数

通过将参数名传入获取指定参数:

dump($this->request->param('test'));
1

浏览器输入 http://www.my-tp5.com/hello?test=yu，将返回

string(2) "yu"
1

# param 参数

// 获取当前请求的name变量
Request::param('name');
// 获取当前请求的所有变量（经过过滤）
Request::param();
// 获取当前请求的所有变量（原始数据）
Request::param(false);
// 获取当前请求的所有变量（包含上传文件）
Request::param(true);

WARNING

变量名区分大小写，除了server和env方法的变量名不区分大小写（会自动转为大写后获取）。

# only 和 except

使用 only 获取部分变量，使用 except 排除部分变量

多个变量可以使用字符串 , 分割，也可以使用数组分割

Request::instance()->only('id,name'); // 请求参数：仅包含 id 和 name
Request::instance()->except('id,name'); // 请求参数：排除 id 和 name

dump($request->only(['id', 'name']));
dump($request->except(['id', 'name']));

# 助手函数

可以使用助手函数 input 获取所有请求参数

获取指定请求类型下所有参数在结尾处跟上 .

input('get.id');
input('get.name');
input('get.');

input('post.name');
input('post.');

input('put.name');
input('put.');

input('request.id');
input('request.');

input('server.PHP_SELF');
input('server.');

input('session.user_id');
input('session.');

input('cookie.user_id');
