$ 请求

$ 请求对象

当前的请求对象由 think\Request 类负责，在很多场合下并不需要实例化调用，通常使用依赖注入即可。在其它场合（例如模板输出等）则可以使用 think\facade\Request 静态类操作。

$ 引入请求对象的几种方法

这里获取请求参数以 get 请求方式为例。

方法一: 使用 Facade 静态代理

dump(\think\facade\Request::get());
1

方法二: 使用实例化 Request 类

$req = new \think\Request();
dump($req->get());
1
2

方法三: 使用依赖注入

public function index(\think\Request $req) {
  dump($req->get());
}
1
2
3

方法四: 使用 Controller 的 request 属性

$this->request->get()
1

其中 $this 就代表当前控制器的实例。

此时，输入 http://www.my-tp5.com/?name=xiaoyu&age=20 可以看到输出:

array(2) {
  ["name"] => string(6) "xiaoyu"
  ["age"] => string(2) "20"
}
1
2
3
4

方法五: 使用助手函数

return request()->get();
1

在地址栏输入 http://www.dyt_api.com/test/index/test?name=yu，即可看到输出

{
  name: "yu"
}

方法六: 使用 instance

$request = Request::instance();
$request->get();
