验证器

ThinkPHP 中自带验证器模块，只需要在自定义验证器中继承即可，核心文件位于 thinkphp/library/think/Validate.php 中。

一般，我们会在 application 下新建一个 validate 文件夹，用于存放自定义验证文件。

比如创建一个验证个人信息的验证器 :

application/validate/User.php

namespace app\validate;
use think\Validate;

class User extends Validate
{
  protected $rule = [
    'name|姓名' => [
      'require',
      'min' => 5,
      'max' => 20
    ],
    'email|邮箱' => [
      'require',
      'email'
    ],
    'password|密码' => [
      'require',
      'min' => 3,
      'max' => 12,
      'alphaNum'
    ],
    'mobile|电话' => [
      'require',
      'mobile'
    ]
  ];
}

可以看到，这里只需要继承 think\Validate 即可轻松实现验证器，通过重写 $rule 属性创建自定义验证规则。

$rule 中指定要验证的字段名，通过 | 对字段进行命名，如果不命名默认为字段名，在输出提示中会显示。

要使用这个验证器，在控制器中指定 :

application/index/controller/Index.php

namespace app\index\controller;

use think\Controller;
use app\validate\User;

class Index extends Controller
{
  public function index() {
    $validate = new User();
    $data = [
      'name' => 'xiaoyu',
      'email' => 'xiaoyu@xiaoyulive.top',
      'password' => 123456,
      'mobile' => 18388019780
    ];
    if(!$validate->check($data)) {
      return $validate->getError();
    }
    return '验证成功';
  }
}

可以看到，首先需要 new 一个自定义验证器，通过验证器的 check 方法验证数据，如果验证失败使用 getError 获取错误信息。

也可使用助手函数 validate()，比如

if(!validate('User')->check($_GET)) {
  return validate('User')->getError();
}

验证格式

验证器可以是一个数组，也可以使用字符串，比如:

$rule = [
  'name' => [
    'require',
    'min' => 5,
    'max' => 20
  ],
  'age' => [
    'between' => '0, 120'
  ]
]
