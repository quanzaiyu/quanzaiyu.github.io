# Rekit Studio

  • 官网: http://rekit.js.org/
  • Demo: http://demo.rekit.org/
  • GitHub: https://github.com/supnate/rekit

Rekit Studio，根据文件结构解析出逻辑结构的工具，让开发以逻辑结构管理项目，真的可能带来项目维护、开发成本的大幅优化。

# 使用

Install rekit

$ npm install -g rekit

Create an app

$ rekit create app1
$ cd app1
$ npm install

Start the app

$ npm start
