格式化

Date

date() 函数用于格式化时间/日期，返回将整数 timestamp 按照给定的格式字串而产生的字符串。

date() 函数返回将整数timestamp按照给定的格式字串而产生的字符串。

语法:

string date ( string $format [, int $timestamp ] )
1

返回将整数 timestamp 按照给定的格式字串而产生的字符串。如果没有给出时间戳则使用本地当前时间。

常用的格式字串:

  • Y - 年 (四位数)
  • m - 月 (01 - 12)
  • d - 日 (01 - 31)
  • H - 24 小时制（00 到 23）
  • h - 12 小时制（01 到 12）
  • i - 分（00 到 59）
  • s - 秒（00 到 59）
  • M - 三个字母缩写表示的月份，Jan 到 Dec
  • y - 年 (二位数)
  • n - 月，没有前导零 (1 到 12)
  • j - 日，没有前导零 (1 到 31)
  • g - 12 小时制，没有前导零，1 到 12
  • G - 24 小时制，没有前导零，0 到 23
  • a - 小写的上午和下午值，am 或 pm
  • A - 大写的上午和下午值，AM 或 PM
  • w - 星期几，数字表示，0（表示星期天）到 6（表示星期六）
  • l - 星期几，完整的文本格式，Sunday 到 Saturday
  • D - 星期几，3 个字母简写，Mon 到 Sun
  • t - 指定的月份有几天，28 到 31
  • L - 是否为闰年，如果是闰年为 1，否则为 0

详细的格式字串参考: function.date、Supported Date and Time Formats

如:

echo date("Y/m/d H:i:s") . "<br>"; // 2018/07/02 15:55:21
echo date("Y-m-d H:i:s") . "<br>"; // 2018-07-02 15:55:21
echo date("l") . "<br>"; // Monday

时间戳

mktime

返回指定日期的 Unix 时间戳。

语法:

int mktime(int $hour = date("H"), int $minute = date("i"), int $second = date("s"), int $month = date("n"), int $day = date("j"), int $year = date("Y"), int $is_dst = -1)
1

参数可以从右向左省略，任何省略的参数会被设置成本地日期和时间的当前值。

如:

$tomorrow = mktime(0, 0, 0, date("m"), date("d")+1, date("Y"));
echo $tomorrow . PHP_EOL; // 注意，PHP时间戳精确到秒，而前端JS的时间戳为毫秒
echo "Tomorrow is " . date("Y/m/d", $tomorrow);
1

time

返回自从 Unix 纪元（格林威治时间 1970 年 1 月 1 日 00:00:00）到当前时间的秒数。

语法:

int time ( void )
1

$nextWeek = time() + (7 * 24 * 60 * 60);
echo date('Y-m-d') ."\n"; // 2018-03-30
echo date('Y-m-d', $nextWeek) ."\n"; // 2018-04-06
echo date('Y-m-d', strtotime('+1 week')) ."\n"; // 2018-04-06

strtotime

将任何字符串的日期时间描述解析为 Unix 时间戳。

语法:

int strtotime ( string $time [, int $now = time() ] )
1

基本符号:

Description Format
dayname 'sunday' | 'monday' | 'tuesday' | 'wednesday' | 'thursday' | 'friday' | 'saturday' | 'sun' | 'mon' | 'tue' | 'wed' | 'thu' | 'fri' | 'sat'
daytext 'weekday' | 'weekdays'
number [+-]?[0-9]+
ordinal 'first' | 'second' | 'third' | 'fourth' | 'fifth' | 'sixth' | 'seventh' | 'eighth' | 'ninth' | 'tenth' | 'eleventh' | 'twelfth' | 'next' | 'last' | 'previous' | 'this'
reltext 'next' | 'last' | 'previous' | 'this'
space [ \t]+
unit (('sec' | 'second' | 'min' | 'minute' | 'hour' | 'day' | 'fortnight' | 'forthnight' | 'month' | 'year') 's'?) | 'weeks' | daytext

常用的格式字串:

  • 'today' - 今天，将时间设置为 00:00:00
  • 'yesterday' - 昨天
  • 'tomorrow' - 明天
  • 'midnight' - 将时间设置为 00:00:00，如: "yesterday midnight"
  • 'noon' - 将时间设置为 12:00:00，如: "yesterday noon"
  • 'ago' - 之前的时间，如: "2 days ago", "8 days ago 14:00", "2 months 5 days ago", "2 months ago 5 days"
  • number space? (unit | 'week') - 处理相关时间，如: "+5 weeks", "12 day", "-7 weekdays"
  • reltext space 'week' - 处理相关时间，如: "Monday next week"
  • 'first day of' - 指定月份的第一天，如: "first day of January 2008"
  • 'last day of' - 指定月份的最后一天，如: "last day of next month"
  • ordinal space unit - 处理相关日期/时间，如: "fifth day", "second month"，"first sat"

参考: datetime.formats.relative

如:

echo strtotime("now"), "\n";
echo strtotime("10 September 2000"), "\n";
echo strtotime("+1 day"), "\n";
echo strtotime("+1 week"), "\n";
echo strtotime("+1 week 2 days 4 hours 2 seconds"), "\n";
echo strtotime("next Thursday"), "\n";
echo strtotime("last Monday"), "\n";

格式检查

checkdate

检查由参数构成的日期的合法性。如果每个参数都正确定义了则会被认为是有效的。

语法:

bool checkdate ( int $month , int $day , int $year )
1

如:

var_dump(checkdate(12, 31, 2000)); // true
var_dump(checkdate(2, 29, 2001)); // false
