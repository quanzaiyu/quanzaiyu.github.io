# Koa简介

为什么说是新一代?

新一代是koa，那上一代呢? 上一代就是大名鼎鼎的Express，koa经过了两个大版本，比起koa1，koa2使用了ES7进行开发，完全使用Promise并配合async来实现异步。

本文编写的时候使用的是koa2

# 安装和使用

新建一个项目名叫myapp，进入此目录

$ npm init
$ npm i koa -s
$ touch index.js
$ vim idnex.js

编写:

var koa = require('koa');
var app = new koa();
app.use(function* () { 
  this.body = 'Hello World';
});
app.listen(3000);

运行node index，在浏览器输入http://localhost:3000/即可看到输出Hello World。

# 使用生成器生成项目

# 安装

$ yarn global add koa-generator
$ koa2 <my-app>
$ cd <my-app> && npm i
$ npm start
