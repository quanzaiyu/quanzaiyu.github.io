# JSX

React 使用 JSX 来替代常规的 JavaScript。

JSX 是一个看起来很像 XML 的 JavaScript 语法扩展。

我们不需要一定使用 JSX，但它有以下优点：

  • JSX 执行更快，因为它在编译为 JavaScript 代码后进行了优化。
  • 它是类型安全的，在编译过程中就能发现错误。
  • 使用 JSX 编写模板更加简单快速。

# JSX 的几种创建方式

方式一, 标签形式

const element = (
  <h1 className="greeting">
    Hello, world!
  </h1>
);

方式二, 对象形式

const element = React.createElement(
  'h1',
  {className: 'greeting'},
  'Hello, world!'
);

# JSX 属性

在 JSX 中, 使用引号 "" 传入普通字符串值, 使用花括号 {} 传入 JS 表达式:

const element = <div tabIndex="0"></div>;
const element = <img src={user.avatarUrl}></img>;

# 样式绑定

样式可以采用对象的写法进行绑定:

const style = {
  fontSize: 100,
  color: '#FF0000'
};

const element = <h1 style={style}>Hello</h1>;

# 数组展开

JSX 允许在模板中插入数组，数组会自动展开所有成员：

const arr = [
  <h1>Hello</h1>,
  <h2>React</h2>,
];
const element = <div>{arr}</div>;

# JSX 渲染

使用 ReactDOM.render 进行 JSX 的渲染:

ReactDOM.render(
  <h1>Hello React</h1>,
  document.getElementById('App')
);

# JSX 注释

JSX 注释需要写在花括号中:

ReactDOM.render(
  <div>
    <h1>Hello React</h1>
    {/*注释...*/}
  </div>,
  document.getElementById('example')
);

# JSX 条件渲染

第一种方式: 三元表达式

class App extends React.Component {
  render() {
    let isLoggedIn = true
    return (
      <div>
        <div>
          {isLoggedIn ? (
            <div>true</div>
          ) : (
            <div>false</div>
          )}
        </div>
      </div>
    );
  }
}

第二种方式: && 运算符

class App extends React.Component {
  render() {
    let isLoggedIn = false
    return (
      <div>
        <div>
          {isLoggedIn && <div>true</div>}
          {!isLoggedIn && <div>false</div>}
        </div>
      </div>
    );
  }
}

第三种方式: 提前处理后渲染

class App extends React.Component {
  render() {
    let isLoggedIn = false
    let ele = ''
    if (isLoggedIn) {
      ele = <div>true</div>
    } else {
      ele = <div>false</div>
    }
    return (
      <div>
        <div>
          {ele}
        </div>
      </div>
    );
  }
}

# JSX 循环渲染

第一种方式, 使用 JSX 表达式直接渲染

要使用 JSX 进行列表循环渲染, 首先得理解 JSX 也是 JS 表达式, 然后就容易多了:

class App extends React.Component {
  render() {
    return (
      <div>
        {
          ['a','b','c'].map((item, index) => {
            return <div>{index} - {item}</div>
          })
        }
      </div>
    );
  }
}

渲染结果:

0 - a
1 - b
2 - c

另一种方式, 先循环再渲染:

class App extends React.Component {
  render() {
    let list = ['a','b','c']
    let row = []
    for (let key in list) {
      row.push(<div className="board=row" key={key}>{key} - {list[key]}</div>)
    }

    // or
    /*
      const row = list.map((item, key) =>
        <div className="board=row" key={key}>{key} - {item}</div>
      );
    */

    return (
      <div>
        {row}
      </div>
    );
  }
}
