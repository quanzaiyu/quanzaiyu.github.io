# 模板引擎

express默认使用jade(pug) 模板引擎。

设置模板引擎:

app.set('view engine', 'jade')

在views目录下新建index.jade:

html
  head
    title!= title
  body
    h1!= message

渲染模板:

app.get('/', function (req, res) {
  res.render('index', { title: 'Hey', message: 'Hello there!'})
})

# jade

jade 是根据缩进式语法，是一种 破坏式的、侵入式的、强依赖的模板引擎。

一旦使用 jade，便不能与传统的 HTML 共存。

首先需要下载 jade

$ yarn add jade

# 简单使用

使用 jade.render 渲染一段模板字符串。

const jade = require('jade')

var str = jade.render(`
html
  head
    style
  body
    div
`)

console.log(str)

渲染结果:

<html><head><style></style></head><body><div></div></body></html>

# 文件渲染

使用 jade.renderFile 渲染指定文件。

const jade = require('jade')

var str = jade.renderFile('./views/index.jade')

console.log(str)

view/index.jade

doctype
html
  head
    style
  body
    div

渲染结果为

<!DOCTYPE html><html><head><style></style></head><body><div></div></body></html>

# 美化输出

jade.renderFile 可以加入 pretty 选项，以美化输出

const jade = require('jade')

var str = jade.renderFile('./views/index.jade', {pretty: true})

console.log(str)

输出为:

<!DOCTYPE html>
<html>
  <head>
    <style></style>
  </head>
  <body>
    <div></div>
  </body>
</html>

# 模板传值

配置选项中还可以向模板中传入自定义的值:

index.js

jade.renderFile('./views/index.jade', {
  pretty: true,
  arr: ['abc']
})

views/index.jade

div(class=arr)

渲染为

<div class="abc"></div>

# 写入到文件

const jade = require('jade')
const fs = require('fs')

var str = jade.renderFile('./views/index.jade', {pretty: true})

fs.writeFile('./build/index.html', str, (err) => {
  if (err) {
    console.log('写入失败')
  } else {
    console.log('写入成功')
  }
})

结合 fs 模块，渲染后将写入到 build 目录下。

# 基础语法

# 属性

普通属性

meta(charset='utf-8')
style(href='style.css',ref='stylesheet')

渲染为


<style href="style.css" ref="stylesheet"></style>

对象属性

style 属性可以使用字符串传值，也可使用对象传值

div(style={width: '100px',height: '100px'})

渲染为

<div style="width:100px;height:100px"></div>

数组属性

class 属性可以使用字符串传值，也可以使用数组传值

div(class=['aaa', 'bbb'])

渲染为

<div class="aaa bbb"></div>

&attributes

&attributes 可以使用对象的形式为属性赋值

div&attributes({id: 'box', class: 'wrap'})

渲染为

<div id="box" class="wrap"></div>

id 与 class

div#box.wrap

渲染为

<div id="box" class="wrap"></div>

属性合并

index.js

jade.renderFile('./views/index.jade', {
  pretty: true,
  arr: ['abc']
})

views/index.jade

div(class=arr class='active')

渲染为

<div class="abc active"></div>

# 内容

内容

div Hello

渲染为

<div>Hello</div>

内容嵌套

div hello
	span xiaoyu

渲染为

<div>hello<span>xiaoyu</span></div>

| 原样输出内容

针对多行文本，可以使用 | 原样输出内容。

script
  |window.onload = function () {
  | alert(0)
  |}

渲染为

<script>
  window.onload = function(){
  alert(0)
  }
</script>

. 原样输出内容

针对长内容，可以使用 . 原样输出内容。

script.
  window.onload = function () {
   alert(0)
  }

渲染为

<script>
  window.onload = function(){
  	alert(0)
  }
</script>

内容自动转义

index.js

jade.renderFile('./views/index.jade', {
  pretty: true,
  content: '<p>Hello</p>'
})

views/index.jade

div=content

渲染为

<div>&lt;p&gt;Hello&lt;/p&gt;</div>

若要jade不自动转义，需要使用 !=

div!=content

渲染为

<div><p>Hello</p></div>

# 变量

使用 -var 定义变量，事实上，所有 js 语句都能使用 - 进行声明

-var a = 'hello'

使用 #{} 输出

div #{a}

渲染为

<div>hello</div>

使用 = 输出

div=a

渲染为

<div>hello</div>

# 循环

如果前面已经使用 - 声明了变量，之后的 js 代码都不需要 -

for

-var arr = ['a', 'b', 'c']
for(var i = 0; i < arr.length; i++)
  div=arr[i]

渲染为

<div>a</div>
<div>b</div>
<div>c</div>

while

-var a = 0
while a<4
  div=a++

渲染为

<div>0</div>
<div>1</div>
<div>2</div>
<div>3</div>

# 条件

if-else

-var a = 12
if(a%2 === 0)
  div 偶数
else
  div 基数

渲染为

<div>偶数</div>

case-when

-var a = 1
case a
  when 1
    div (1)
  when 2
    div (2)
  default
    div (0)

渲染为

<div>(1)</div>

# ejs

ejs 使用嵌入式语法，是一种 温和的、非侵入式的、若依赖的模板引擎。

使用 ejs，可以嵌入 HTML 代码。

首先需要下载 ejs

$ yarn add ejs

# 渲染文件

使用 jade.renderFile 渲染文件。

const ejs = require('ejs')

ejs.renderFile('./views/index.ejs', {name: 'xiaoyu'}, (err, data) => {
  if (err) {
    console.log('编译失败')
  } else {
    console.log(data)
  }
})

views/index.ejs

<html>
  <head>
    <style></style>
  </head>
  <body>
    <div>Hello <%= name %></div>
  </body>
</html>

编译为:

<html>
  <head>
    <style></style>
  </head>
  <body>
    <div>Hello xiaoyu</div>
  </body>
</html>

# 输出到文件

const ejs = require('ejs')
const fs = require('fs')

ejs.renderFile('./views/index.ejs', {name: 'xiaoyu'}, (err, data) => {
  if (err) {
    console.log('编译失败')
  } else {
    fs.writeFile('./build/index.html', data, (err) => {
      if (err) {
        console.log('写入失败')
      } else {
        console.log('写入成功')
      }
    })
  }
})

views/index.ejs

<html>
  <head>
    <style></style>
  </head>
  <body>
    <div>Hello <%= name %></div>
  </body>
</html>

渲染为

<html>
  <head>
    <style></style>
  </head>
  <body>
    <div>Hello xiaoyu</div>
  </body>
</html>

# 基础语法

凡是 js 语法都能写在 <% %> 之中，比如

<%
var arr = ['a', 'b', 'c']
for(var i = 0; i < arr.length; i++) { %>
  <div><%=arr[i]%></div>
<% } %>

渲染为

<div>a</div>
<div>b</div>
<div>c</div>

# 转义输出与不转义输出

  • 使用 <%= %> 是转义输出
  • 使用 <%- %> 是不转义输出
<% var str = '<p>Hello</p>' %>
<%=str%>
<%-str%>

&lt;p&gt;Hello&lt;/p&gt;
<p>Hello</p>

# 文件包含

使用 <% include ... %> 可以指定文件包含，被包含的文件当作普通文本嵌入到指定位置，被包含的文件不限格式，可以是 html、ejs、css、js等。注意，include 语句不能与其他js语句共存，必须单独使用 <% %> 包含。

index.ejs

<html>
  <head>
    <style></style>
  </head>
  <body>
    <% include ./header.html %>
  </body>
</html>

header.html

<header>Hello</header>

渲染为

<html>
  <head>
    <style></style>
  </head>
  <body>
    <header>Hello</header>
  </body>
</html>

# consolidation

consolidation 是一系列模板引擎的集合，可以方便地设置模板引擎。

安装

$ yarn add consolidate

比如要使用 ejs 模板引擎开发项目:

const express = require('express')
const consolidate=require('consolidate')

const app = express()

// 配置模板引擎
app.set('view engine', 'html'); // 输出什么东西
app.set('views', './views'); // 模板文件放在哪儿
app.engine('html', consolidate.ejs); // 哪种模板引擎

app.get('/', (req, res)=>{
  console.log(res.datas)
  // 指定渲染模板
  res.render('index.ejs', {data: 'hello'});
})

// 静态资源
app.use(expressStatic('./www'))

使用 app.set('view engine', 'html') 设置输出格式为html，使用 app.set('views', './views') 指定存放模板文件的目录，使用 app.engine('html', consolidate.ejs) 设置模板引擎，其中 consolidate.ejs 就是引用了 consolidate 模块的 ejs 模板引擎。

直接使用 res.render('index.ejs', {data: 'hello'}); 就可以使用指定目录下的模板进行渲染。
