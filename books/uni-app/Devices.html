# uni-app 设备

# 设备信息

使用 uni.getSystemInfo() 获取设备信息。

uni.getSystemInfo({
  success(info) {
    console.log(info);
  }
})

返回信息包括：

{
  errMsg:"getSystemInfo:ok",
  brand:"devtools", // 手机品牌
  model:"iPhone 6", // 手机型号
  system:"iOS 10.0.1", // 操作系统版本
  version:"6.6.3", // 引擎版本号
  pixelRatio:2, // 设备像素比
  screenWidth:375, // 屏幕宽度
  screenHeight:667, // 屏幕高度
  statusBarHeight:20, // 状态栏的高度
  language:"zh", // 应用设置的语言
  SDKVersion:"2.0.4", // 客户端基础库版本
  batteryLevel:100,
  benchmarkLevel:1,
  fontSizeSetting:16, // 用户字体大小设置
  platform:"devtools", // 客户端平台
  windowHeight:603, // 可使用窗口高度
  windowWidth:375 // 可使用窗口宽度
}

uni-app 也提供了一个同步接口：

let info = uni.getSystemInfoSync()

# 网络状态

使用 uni.getNetworkType() 获取网络状态。

uni.getNetworkType({
  success: function (res) {
    console.log(res);
  }
});

返回信息包括：

{
  errMsg:"getNetworkType:ok",
  networkType:"wifi" // 网络状态，包括 wifi、2g、3g、4g、none、unknown
}

网络状态的改变，可以使用 uni.onNetworkStatusChange() 进行监听：

onLoad () {
  uni.onNetworkStatusChange(res => {
    console.log(res);
  });
}

返回信息包括：

{
  isConnected: true, // 当前是否有网络连接
  networkType: "3g" // 网络类型
}

# 拨打电话

使用 makePhoneCall 拨打电话。

uni.makePhoneCall({
  phoneNumber: '114'
});

# 扫码

通过 uni.scanCode() 进行扫码。

参数包括：

  • scanType Array 扫码类型，二维码是'qrCode'，一维码是'barCode'，DataMatrix是'datamatrix'，pdf417是'pdf417'。
  • onlyFromCamera Boolean 是否只能从相机扫码，不允许从相册选择图片
// 允许从相机和相册扫码
uni.scanCode({
  scanType: ['qrCode', 'barCode'],
  success: function (res) {
    console.log('条码类型：' + res.scanType);
    console.log('条码内容：' + res.result);
  }
});

// 只允许通过相机扫码
uni.scanCode({
  onlyFromCamera: true,
  success: function (res) {
    console.log('条码类型：' + res.scanType);
    console.log('条码内容：' + res.result);
  }
});

扫码成功后的返回数据包括：

  • result 所扫码的内容
  • scanType 所扫码的类型
  • charSet 所扫码的字符集
  • path 当所扫的码为当前应用的合法二维码时，会返回此字段，内容为二维码携带的 path。

# 剪贴板

设置剪贴板

uni.setClipboardData({
  data: 'hello',
  success: function () {
    console.log('success');
  }
});

获取剪贴板数据

uni.getClipboardData({
  success: function (res) {
    console.log(res.data); // hello
  }
});

# 屏幕亮度

设置屏幕亮度

uni.setScreenBrightness({
  value: 0.5,
  success: function () {
    console.log('success');
  }
});

value 为屏幕亮度值，0为最暗，1为最亮。

获取屏幕亮度

uni.getScreenBrightness({
  success: function (res) {
    console.log('屏幕亮度值：' + res.value);
  }
});

保持屏幕常亮

uni.setKeepScreenOn({
    keepScreenOn: true
});

# 震动

使手机发生较长时间的震动（400ms）

uni.vibrateLong({
  success: function () {
    console.log('success');
  }
});

使手机发生较短时间的振动（15ms）

uni.vibrateShort({
  success: function () {
    console.log('success');
  }
});
