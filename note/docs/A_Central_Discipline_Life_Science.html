# 一门枢纽性的学科 - 生命科学

最近读到得到的一个系列课程：《王立铭.生命科学50讲》，觉得特别有意思，用一种大众都能听得懂的方式向不是从事专业研究的人也能了解生命科学。

王立铭，何许人也，浙江大学生命科学研究院的教授，美国加州理工学院博士，也是中国最会讲故事的科学家之一。最近在得到开展了这门付费课程。

听了这门课，我产生了一些思考。

# 总述

在王老师看来，在人类所有的科学领域中，生命科学是最谦卑，但又是最自负的一门科学。

我们身上的所有特征，几乎都是基因所决定的，没有多少使我们自己能够决定的。面对自然规律，生命和智慧都只是演化的旁观者。生命科学是谦卑的。

说它自负，因为它与我们息息相关，生命科学让我们站在了人类历史和命运的转折点。每一次生命科学革命，都在影响着全人类甚至全世界的命运，它与其他学科相互交融，相互影响，改变着一切。

首先是达尔文的进化论，打破了神创世界的固有教义，让人类知道，我们并不是神所创作的，更不是神的后裔或者使者，而是跟地球上万千生物一样，是一个普通的存在，甚至跟那些猩猩猴子拥有同样的祖先，我们就是猴子变的。

生物的进化，收到自然选择和基因突变的影响，这是中学生物课本就向我们介绍的。但是，这些原理的背后到底是什么呢？

# 面度生命，我们别无选择

王立铭老师给出一个结论是：生命是40亿年的无可奈何。

他从三个角度论述了这个观点：

第一个观点：对于我们的身体，我们没有话语权。

几乎所有的特性甚至行为，我们都受到基因的控制。无论我们喜欢与否，好的基因不好的基因我们都无法选择。比如镰刀型贫血症，是一种很严重的遗传病，但是人类竟然遗留至今，没有被自然选择所淘汰，为什么？

原来，虽然镰刀型贫血症是一种很严重的疾病，但是却可以抵抗疟疾。要知道，在人类发明抵抗疟疾的药物之前，这是唯一抵抗疟疾的方法。因此，在人类漫长的演化史中，这种基因得以保留，虽然伤敌一千自损八百，但却长期保留在人类基因中。

这些如今看起来不好的基因，却往往是人类在漫长历史上生存繁衍到今天的本钱。不过无奈的是，这些现象至今仍保留在我们身体中，我们无法选择。

第二个观点：智慧来自基因突变

人类之所以与其他生命不同，是由于人类拥有高级的智慧。而这种智慧的形成，却也是一种无奈。

语言是人类非常重要的一种能力，但却不是人类所特有的，很多动物也有他们自己的语言，比如说猴子、狼。而人之所以特殊，是由于人类形成了构成语言的语法。什么是语法，就是能将各种单词按照一定的顺序、规则，随心所欲地拼接在一起的能力，这是人类所特有的。而其他动物的语言，只是停留在单个词汇的阶段，比如猴子的某种叫声，是警告同伴说：“河边有一头狮子，大家不要去喝水”。另一种叫声，又意味着另一种意思。可想而知，如果人类的每个意思甚至每个想法都得用一个独特的单词来承载，那得有多少？

有证据显示，人类基因组中的一个名叫 FOXP2 的基因跟人类形成语言语法息息相关，如果这个基因受损，人类说话将毫无条理性，有一个个间断的词汇构成，无法明确传递其真正想表达的意思。

而 FOXP2 这个基因，人类的近亲黑猩猩也有，不过人类的这个基因与之有一点微小的差异，而造成这个差异的，就是基因突变。人类的这个基因大约在距今10-20万年之间形成。大约就是人类走出非洲，开始向全世界迁移的时间。

这种例子还很多，比如人类的学习能力、爱情、自尊心、责任感、对他人的关心爱护等。这些都不是人类凭空创造出来的，都是经过漫长时间在一次次基因突变中形成的。他们的背后，却是冷冰冰的生物学规律。

第三个观点：在衰老和死亡面前，我们别无选择

我们都厌恶死亡，但是死亡却又是每个人的最终归宿。这一点，用生命科学框架解释就是：生物演化并不排斥死亡，甚至在某些条件下，它会主动选择让我们死亡。

如果一个遗传变异在生物演化中，可以更好地发育、成熟、求偶、交配，那么这个基因就很容易被自然选择说青睐，而遗留下来。因为一个生命的使命，就是繁衍后代，完成这个任务，他的使命也就完成了。

比如男性的睾丸酮，这是一个非常重要的雄性激素，跟男性性器官的形成、精子的发育、生育功能都密切相关，会让男人看起来功能具备“男子汉气概”。但是，这个激素却与很多疾病相关，有证据显示，睾丸酮的含量与人类寿命成反比。

但是，这种基因却更加受到自然选择的青睐，因为一个充满战斗力和交配欲望的男性，更容易留下自己的后代。

所以，在生存与死亡的问题面前，选择权也并不在我们手中，这一切都是生物演化的必然归宿。

总结就是：生命只是进化的产品，从来都不是自己的主人。

# 人类是如何超越其他生命，称为世界主宰

在王立铭老师看来，人类之所以能够有异于其他生物，有以下三个重要的原因：

  1. 人类的语言
  2. 自我意识
  3. 自由意志

但为什么是这三个而不是其他原因呢？

比如说人类的社会结构，这其实很多动物都拥有，比如蚂蚁、蜜蜂，甚至比人类的社会结构更加高效。

# 人类的语言

其实，很多动物也有自己的语言，比如猴子、狼等。但是只有人类的语言能够将一个个独立的词语使用语法结合成拥有特定意义的句子。关于这一点，在上面已经说过了，研究表明这跟灵长目的一个基因 FOXP2 的形成有关。

# 自我意识

在弗洛伊德的心理学理论中，人有三种意识：本我、自我、超我。

本我是人最原始的本能与欲望，超我是人的内在的良知与道德判断，而自我，协调两者的关系，负责处理显示世界的事情。

人类之所以能够形成那么复杂的感情，比如荣誉感、羞耻心、自律能力等等，都与自我意识有关。

# 自由意志

人能拥有完全自由控制自己行为的能力。

# 人类真的有自由意志吗

上面说道，自由意志是人类有异于其他动物的一个重要原因。有了自由意志，我们可以控制我们的思维、判断，然后付诸于行动，并对一切行为的结果负责。

大家都知道，各国法律基本都规定，精神病人在不能辨认和控制自己行为的时候，造成的危害不用承担法律责任。而正常人就必须为自己的所作所为负责，因为正常人拥有自由意志。

但是，从基因的角度分析，这一点真的可靠吗？

科学家们对上千名杀人犯进行基因测序，发现带有某一种基因的人，更有可能产生攻击性行为。这种基因变异，会减少人脑内一种酶的代谢，从而削弱人控制感情的能力。而这种基因又是可以遗传的，也就是说，暴力的父亲可能生出有暴力倾向的儿子。一项研究发现，在决定暴力倾向的因素中，遗传竟然占到了40%。

也就是说，如果一个人仅仅因为祖上有某种基因，公众就会默认他也带有这种基因，那即使他什么都没有做，也很有可能会受到整个社会的不公平对待。

事实上，现在就有一家意大利法庭表示，如果杀人犯能够给出大脑扫描和基因测序的证据，证明自己是因为基因问题而犯罪的，就可以降低刑罚。

你看，无论是享有的权利，还是承担的责任，社会的边界都在发生飘移。生命科学实在是太奇怪，因为从来没有一项科学能像生命科学那样，以如此的强度，反作用于我们的社会规则。

再举一个例子，现代西方社会的基础是民主选举。这项制度的基础其实也是自由意志。也就是，每个人独立地做出自己的判断，民主选举整合大家的意愿，形成政治的民意基础。

但是，2015年，伦敦国王学院对双胞胎的研究显示，“一个人投票是给共和党还是民主党、工党还是保守党，57%是出生之前就被基因决定了。”类似的研究还有很多。你看，你的意见不全是你的意见，这里面有大量的基因因素。

投票的人到底是有自由意志的选民？还是被基因操纵的木偶？投票表达出来的，到底是人们理性的选择，还是不由自主的梦话？投票的最终结果，到底是民意，还是古老的遗传基因的表达？这些问题一旦存疑，那现代民主制度的基础也就松动了。

生命科学研究还发现了一个更要命的现象，就是自由意志可能只是一个幻觉。

研究表明，一个人产生行动意识，到他做出这项行动，比如点击一下鼠标，时间差是200毫秒。这个好理解。但是更进一步的研究就有点吓人了。早在这个人产生行动意识之前，从他大脑的神经活动图谱当中，科学家就已经能看出来，他要做这个决定了！这个时间差是300毫秒。换句话说，当我们自以为我们在自由地做一个决定，哪怕是按一下鼠标这个非常简单的决定的时候，我们其实只不过是在执行大脑已经给我们提前准备好的一个决定。

这个实验是1985年，美国加州大学旧金山分校的神经生物学家李贝特做的。30多年来，随着类似实验的重复进行，实验工具越来越先进，结论也越来越明晰。李贝特现象的确存在。

王立铭老师给我讲完这个例子后，引用了《黑客帝国》里的一句话：“其实你已经做了选择，只不过你在试图理解这个选择背后的原因。”说的就是这个机制。

# 从生命科学到社会学

人类理解社会学，一开始其实是使用物理学的方式进行研究。社会学的创始人孔德就是这样，因为但是牛顿对那一代人的影响实在是太大了，孔德就想将人类的社会现象使用类似于物理学中牛顿定理这样的的方式进行解释。因此，最初社会学被称之为“社会物理学”。

但是，大家很快就意识到，人类社会不是物体，而是像生物那样的有机体。对于社会现象而言，生物学的模型比物理学更有解释力。

对于社会现象而言，生物学的模型比物理学更有解释力。并不是光是在学者圈子里，现在创业者圈子里，也都流行着演化、迭代、生态，这些词来描述商业现象，这些词也都是从生命科学里面衍生出来的。

我们从最初的生命说起。

最初，生物都是单细胞生物，觅食、成长、繁殖、防御什么都是自己干。

然而，一个细胞的体积是有限的，不可能无限制地生长，因此，就有很多细胞集聚到一起，形成多细胞生物。

为了提高效率，既然都已经是多细胞生物了，就不能像以前一样一群乌合之众，各干各的。因此，各部分的细胞开始做各自擅长的事情。表皮细胞专门负责防御、神经细胞专门负责信息传递、血液细胞专门负责运输营养。

但是，其他事情都可以交给别人干，唯有一件事是所有细胞的本能：繁殖。凭什么只有生殖细胞和干细胞可以进行生殖？有的时候，这种本能会被唤醒，偷偷地进行自我繁殖。大部分情况下，这种背叛都会受到免疫系统的背叛。但是，如果这种背叛成功，就将引发癌症。

其实，癌症并不是什么病，它是被压抑的正常状态，恶性的、非正常的大爆发。每种细胞有各自的叛变方式，理论上个体有多少种细胞就有多少种癌症。这也就是癌症之所以难以治愈的原因之一。

对应到社会学上，一开始，每个人都可以独立存在，自己采摘果实，自给自足。但是，在面对自然，个体实在太渺小，每个人都会面临野兽、天灾的危害。于是，很多人聚集在一起，一起捕猎，搭建房屋以躲避风雨，形成部落。

随着部落之间的战争与吞并，部落越来越大，社会形成明确分工，以往的工作方式不能满足现在的需求，逐渐形成城市、邦国。

将视野转到现代，公司的其实也是如此，聚集一群人员，分工合作，但是，如果内部什么地方出了纰漏，就有可能导致整个企业土崩瓦解，对应到上面说的癌症。

# 上帝的手术刀 - 基因编辑技术

这是我在王立铭老师课程中听到的最具有震撼力一个词。

什么是基因编辑？

就是在生物庞大的遗传信息中，精准寻找到出现问题的基因位点，然后把错误的位点剪切、替换成正确的位点。

比如，可以让农作物和牲畜的产量大大提高，在全世界范围内消除饥饿；病菌、病毒、害虫，也可以在地球上根除掉。

这听起来是件非常不错的技术，剔除人类的劣质基因。但是，任何好的技术都有其忧患的一面，事实上，人类的每次技术革命都或多或少的引起了一些不好的后果，甚至战争。

比如工业革命，这是一个有利于全人类的革命，但是在当时，却引发了大量工人失业，机器代替了人工劳作。又比如，科学家进行了原子弹的研究，但是原子核最终却应用到战场上。

那么，基因编辑技术呢，没人说得清这是好是坏。

比如，可能有钱人会利用基因编辑技术，将自己的基因进行改造，永久性地遗传给后代，造成更大的贫富差异。本来穷人能够通过努力换来不一样的人生，却由于技术革新，使得所有优质资源都向富人聚集。

乍一看，人类的品质终将因此得到提升，但是如果从生物多样性的角度出发，最终所有人的基因都趋同，不利于持续发展，这样的后果，可能是毁灭性的。

给我一个启发：任何技术都能不能只能贪图眼前的好处，我们现在所能够预见的好处，可能正在为后代埋下一个永远也填不满的坑。
