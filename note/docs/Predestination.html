          # 爱上自己, 生下自己, 杀死自己, 这是怎样一个烧脑的神脑洞科幻故事

          今天来聊聊一部非常烧脑的科幻神剧: 《前目的地》

          影片改编自罗伯特•海因莱恩（1907-1988）的著名短篇小说《你们这些回魂尸》，最早发表于1959年，其中的脑洞真的让人惊叹不已。

          作者以硬科幻小说而著名，他赢得星云奖5次，雨果奖7次，也曾被誉为“美国现代科幻小说之父”。

          其实早就听说过这部神剧, 一直没有时间看, 今天看了一下原剧, 的确是非常有意思, 而且非常烧脑。

          一句话概括故事就是：主角自己上了自己，自己生了自己，自己偷了自己，又自己杀了自己，最后自己又招募自己的故事。

          # 剧情描述

          男主角是一名时空特工，他经常穿梭时空为提前阻止犯罪。

          1985年的特工，接受任务来到1970年阻止“炸弹狂人”的犯罪，但是被罪犯偷袭导致任务失败，他也受炸弹影响被毁容。

          在经过漫长的康复后，他收到了最后的一次任务：“招募约翰”。

          特工乔装打扮成一个酒吧来到了1970年11月6日的“老爹酒店”。

          这天晚上，约翰来酒吧喝酒并对特工讲述了自己匪夷所思的人生。

          他说自己是笔名叫“未婚妈妈”的作家，他小时候是一个名叫“简”的小女孩，1945年她出生时就被遗弃在孤儿院门口。

          “简”渴望正常家庭的温暖，一直努力学习想改变自己的现状，这样的她也成了孤儿院里的异类。

          她跟普通孩子并不太一样，有着强健的体格，甚至比男孩子力量还要强。她热爱航空，1963年，简被选中参加太空公司的面试，是一份关于让男宇航员在太空中抒发压力和情绪的工作。

          简非常聪明，体格强壮，很快成为了候选人中的佼佼者，但她很快又成了其他人嫉妒的目标。在一次打架斗殴事件后，她被取消了资格。

          万念俱灰的简找了一份白天当廉价保姆的工作维持生计，晚上去夜校学习，希望有天能回到太空公司。

          在夜校学习的时候她遇到了一个男人，并且很快爱上他，成为恋人。但好景不长，那个男人偷偷走了，再也没有出现。

          在这期间, 太空公司的罗伯森先生找到简，邀请她加入，其实太空公式只是一个幌子，他们想找一批拥有超能力的人，没有家人，没有丈夫，没有妻子，没有孩子，没有过去，也不必担心将来的人。然而此时简怀孕了, 罗伯森只好放弃。

          在1964年, 简生下来一个女婴。

          坏消息接踵而至，他的小孩被偷走了。

          剧情到这, 已经非常悲剧了, 从小就被遗弃, 在孤儿院长大, 受人欺负, 长大后终于遇到自己爱的人, 但是却也悄悄离自己而去, 甚至自己的唯一希望孩子也被偷走。

          你以为作者就这点脑洞, 大错特错, 故事还在继续, 这才是悲剧的开始。

          紧接着, 医生告诉她一个非常惊人的消息，她是一个双性人。因为生育，女性生殖器官无法继续使用，她以后只能做一个男人。

          后来他成为了一个作家“未婚妈妈”，并改名约翰。

          特工听完问他，如果有一个机会能让你回到过去，那个背叛你的男人就在你面前，你会怎么做？

          约翰毫不犹豫地回答“杀了他！”

          特工带着他回到了1963年4月3日（简和那个男人相遇的那天）。

          让人始料未及的是，约翰没有等到那个负心汉，却和简相遇了，并且爱上了她。

          在约翰和简相爱的时候，特工来到1970年再次试图阻止炸弹狂人，但被炸弹狂人发现任务再次失败，这时他也发现曾救过自己的神秘人竟是自己。

          特工来到1964年，抱走简的女儿，穿梭到1945年把女婴放到孤儿院门口。

          特工再次回到1963年带约翰去到1985年，成功招募约翰成为一名时空特工。

          退休以后的特工来道1975年享受退休生活，完成了一部名叫时间、爱情、还有未婚妈妈的自传体小说。

          这时特工忽然顿悟，再次穿梭时空回到炸弹狂人第一次作案的洗衣店，终于看到了自己苦苦追寻的罪犯，竟是自己。

          特工最终将炸弹狂人的自己杀死, 最终成为炸弹狂人。

          # 循环的时间线

          实际上，《前目的地》展现的时间结构即无明确的初始端点，亦无最终目的地的归宿。本片描述的是一个无限自相缠绕、循环往复的时间轮回。在这个故事里，女婴、简、约翰、特工(克洛斯)、炸弹狂人，这些不同的角色都是同一个人，不同时期的自己。只不过TA是在不同时间线中自己与自己恋爱、自己与自己斗争罢了。TA是在时间的空间化中跳着带着枷锁的小步舞曲，也是一种走钢丝的冒险行动。TA总认为调整时间坐标系（USFF）的小偏差不至于造成大跨度时间线（“前后不能超过53年”——克洛斯）的恶果。然而，就是屡次三番的小偏差的积累、叠加与措置、放大效应造成TA人生命运的迷乱、纠缠与业缚！

          下面来简单理一下故事的时间线:

          • （1）1945年9月13日，TA被送至孤儿院。长大经历“太空集团”的开除。
          • （2）后于1963年4月3日邂逅穿越时空的TA，之后被TA抛弃。
          • （3）产下婴儿TA后于1964年3月被TA抱走。
          • （4）之后从事文学写作，在1970年11月6日的酒吧结识穿越时空的TA。
          • （5）两个时空汇合的TA与TA一起来到1985年8月12日的时间管理局总部。
          • （6）第二天1985年8月13日，TA接受任务抓捕不同时空里作案的TA。由于TA成为时间特工拥有时间坐标变换器（USFF）可以得到自由任意穿越时空的能力，于是可以发泄多年内心对社会的不满。之后TA穿越时空制造数百起爆炸案，成为“炸弹狂人TA”。
          • （7）由于任意穿越时空会造成大脑损伤，记忆力减退而无法修复（“你每次穿越导致的记忆碎片我们无法做到彻底修复。这会导致你的记忆出现问题，后果很严重！”——罗伯森），作为时间特工的TA不记得炸弹狂人就是TA，而在多维时空中对炸弹狂人TA展开追捕。在某次纽约大爆炸追捕失败后，TA严重面部烧伤。回到时间管理局整容，养病7年后TA只记得自己是时间特工。
          • （8）TA前往1970年11月6日的酒吧【上接（4）】截获TA，两个时空的TA与TA在酒吧结识。在酒吧同情彼时的TA。
          • （9）带TA来到1963年4月3日【上接（2）】与女性的TA恋爱生子。
          • （10）而时间特工TA趁此时机来到1970年3月2日【上接（7）】爆炸案现场抓捕罪犯TA。被打晕厥后，罪犯TA逃走。
          • （11）醒来后又撞见1975年3月【上接（7）】的罪犯TA，罪犯TA逃跑至1992年2月21日作案。
          • （12）时间特工TA又回到1964年3月【上接（3）】抱走婴儿TA。
          • （13）之后又来到1945年9月13日【上接（1）】将之送往孤儿院，进行又一重TA的轮回。
          • （14）之后TA又回到1963年6月24日【上接（2）（9）】找TA，而抛弃TA。
          • （15）两个时空中的TA与TA汇合一起来到1985年8月12日【上接（5）】的时间管理局总部。
          • （16）之后于1985年8月13日【上接（6）】TA成为新时间特工接受任务。
          • （17）而时间特工TA退休去往1975年1月7日。交女友，写小说。小说写完后是1975年3月某日罪犯TA的犯罪日。时间特工TA抓捕炸弹狂人TA与之时空汇合。
          • （18）炸弹狂人TA告诉时间特工TA退休没有结束，而来到此时此地将炸弹狂人TA击毙，从而时间特工TA重演轮回以至于成为现在的炸弹狂人TA。
          • （19）而另一条时间线是，时间特工TA退休与女友结婚，但被泄露秘密身份，又逼得TA重操旧业。想要改变命运建议TA重新爱上TA，从而有机会创造新因果。
          • （20）时间特工TA击毙罪犯TA，面对时间分叉难以抉择……

          通过以上20个阶段的时间分析可以看到，TA是在不同时间线中相互交错、穿越的。时间的无限循环像西方神秘主义符号的衔尾蛇一样口尾相衔，自己是自己的因，同时自己为自己的果，因造果果造因，也就是——相因互果的循环往复，而不得出离。他的时间线是个圆，一直在轮回，就像神话中靠吞食自己尾巴而存活下去的蛇。

          主人公就是一个没有历史，没有祖先的人。他来自悖论，他既存在又不存在。而他存在的意义只有一个，成为原因和结果，为了存在而存在，这是他孤独和注定的宿命。

          # 人物与背景解析

          本片主要出现了五个角色：简、约翰、克洛斯、炸弹狂人与罗伯森。他们到底谁是真的明白人呢？我们可以大致将这五个角色分为三组，并标志他们的智慧层次与状态：

            1. 简与约翰——懵懂无知、痛苦呻吟。
            1. 克洛斯与炸弹狂人——明白一部分、自由创造、自作自受。
            1. 罗伯森——超然、无为的智慧层级，统观大局，在关键时刻进行点拨、引导，决定格局走向。

          本片中克洛斯曾多次对罗伯森的金玉良言只知其然而不知其所以然，这是因为他数次穿越后落下的记忆减退所致，“你每次穿越导致的记忆碎片我们无法做到彻底修复。这会导致你的记忆出现问题，后果很严重！”罗伯森提醒道。
