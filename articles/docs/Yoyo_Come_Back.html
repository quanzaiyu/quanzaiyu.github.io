          # 十多年翘首相待，终盼心然回归

          （我是一个工科生，并不擅长抒情表达，写文章也特别烂，有的时候一句话手指都到键盘上了却不知该敲什么，用了很长时间才码出这一篇马马虎虎的文章，我最近也尽力在学习写作方面的知识，写得不好就不要喷我了-_-）

          昨夜，在5sing原创音乐基地，偶然在洛洛的个人首页中发现“心然”这个名字。激动，不安，甚至有些怀疑，各种复杂的情感充斥在脑海。

          洛洛是我非常喜欢的一个古风原创歌手，洛洛关注的人，我想都不是等闲之辈，应该是心然姐本人没错。而心然姐，可以说是我接触到的第一个对我影响深刻的古风歌手。当时大概是我还在上初中的时候吧，

          为了验证我的想法是否正确，直接点了心然的头像进入心然的5sing首页，看了下介绍，“心然回来了，未公开歌曲已解锁完毕”。看了下歌单，发现果然是心然姐曾经创作的歌曲没错。又查询了一些资料，至此，我完全相信心然姐已经回归了。

          具体什么时候回归的我也不是太清楚，不过看发的歌与录制的节目，推算应该是去年（2017年6-7月的样子吧），网上搜索了一下证实了我的猜想。

          去年刚刚毕业，忙着找工作，找到工作后忙于事业，也没太关注这方面的事情。今天看到，纯属偶然，因为前几个月偶然听了洛洛唱的《纸短情长》，偶然发现洛洛居然是个男孩纸，偶然发现有一个叫5sing的原创音乐基地，然后今天偶然在洛洛的关注中发现了心然姐！！！

          # 初识心然：朦胧记忆，仙侠伴我成长的岁月

          最初认识心然，是在初中时，当时特别喜欢仙剑系列的游戏，觉得仙剑奇侠传简直就是国产RPG的一个传奇。游戏中那种优美的曲调，直接能透过心灵引起情感的共鸣。

          我有个同学，也喜欢玩仙剑系列的游戏，当时我也是通过他认识心然姐的。当时应该还没有5sing这种平台，主要还是通过酷我酷狗进行音乐的下载。当时，我几乎下载了心然姐所有的歌曲。（不过可惜，岁月流转，当时存放的设备现在已经不能用了，不过缘聚缘灭，现在心然姐回归了，又能听到曾经的美妙歌声了）

          记得那个时候还是仙剑4的时代吧，一首《千年缘》，荡澈心灵，清澈干净的声音每句都深深映入心灵，仿佛得到净化。

          也是心然姐，引领我喜欢上了古风这一曲风的歌曲。我本人并不喜欢那种闹哄哄的音乐（诸如摇滚，以及整体节奏快声音大的那种），我比较喜静，从那个时候起，就奠定了我喜欢古风歌曲这一基调。

          # 心然隐退：不舍之情难以言表

          2008年，心然姐退隐。

          听说心然姐隐匿的消息，内心基本上是崩溃的。不过我一直都是后知后觉，当时并不是马上就知道这个消息，年龄又小，不能随时上网，对互联网的东西也是一知半解懵懵懂懂，不记得什么时候偶然看到的消息。

          据说是由于生活受到了严重的影响，更多的东西，不得而知。

          据知情人透露，然姐姐最初只是想以歌会友，在网络上交一群志同道合的朋友，在心然吧，那曾经是一片心灵净土，很多小伙伴在里面编曲作词，讨论有关游戏、二次元，甚至将古风上升至一种文化。

          然而，随着“心然”这个名字的知名度不断上升，很多喷子水军在心吧里面肆无忌惮地进行攻击，曾经的一篇乐土不复存在，有的更多是永无止境的纷争。在心吧最初的成员都是一些内心纯真的孩子，很多人受不了离开了心吧。然姐姐甚至被人肉攻击，以至于最终身心俱疲，最终悄然离开了这片曾经的心灵净土。

          从此，“心然”这个名字消失在大家的视野当中，于三次元的心然姐本人，二次元的“心然”应该是已经陷入了永恒吧。

          心然姐在这些年经历了些什么，不得而知，不过就如同现在回归后的“然爷爷的葫芦娃已经长大，可以保护爷爷了”。每个心宝宝都是然姐姐坚强的后盾，刺猬魍魉岂敢造次。

          # 心然回归：冥冥之中注定的缘分

          就我说的，昨晚一夜，我几乎都兴奋得睡不着。刷着5sing中然姐姐的音乐与是视频，到B站、微博、贴吧搜集各种然姐姐回归的证据。

          十年，是可以让曾经的记忆变得模糊，人们不会一直记得什么，即使它很重要，仍然会被岁月的浪花清洗得干干净净。

          然而，在记忆深处，很多人内心还记得，有那么一个人，曾经给自己心灵上很多慰藉，通过音乐的方式，将内心深处的那些黑暗驱逐。音乐真的是一个好东西，虽然我不懂音乐，但是听着这么清澈善良的歌曲，内心是会得到共鸣的。

          然姐姐的回归就是带给每个心宝宝最好的礼物，大家珍藏于心。

          好吧，工科生也没那么多叽叽歪歪的东西，能写出那么一大段，我也觉得是超常发挥了。
