    精品网址收藏 - Node框架、库与插件

    框架

    Electron

    Electron 是由 Github 开发，用 HTML，CSS 和 JavaScript 来构建跨平台桌面应用程序的一个开源库。 Electron 通过将 Chromium 和 Node.js 合并到同一个运行时环境中，并将其打包为 Mac，Windows 和 Linux 系统下的应用来实现这一目的。

    中文官网、GitHub

    Express

    Express 是一个基于 Node.js 平台的极简、灵活的 web 应用开发框架，它提供一系列强大的特性，帮助你创建各种 Web 和移动设备应用。

    Express 官网、GitHub

    Koa

    Koa 是由 Express 原班人马打造的 koa，致力于成为一个更小、更健壮、更富有表现力的 Web 框架。

    Koa 官网、GitHub

    Hapi

    Hapi 是用于构建应用程序和服务的框架。

    Hapi 官网、GitHub

    Egg

    为企业级框架和应用而生。

    Egg 官网、GitHub
