      # 优质的 Markdown-It 插件

      # 前言

      初识 markdown-it，是在 VuePress 中。

      以前特别想自己定制 Markdown 的解析，试过 marked.js[^1]，但是功能较少，局限性很大，也不容易扩展。

      玩了一阵 Gitbook[^2]，挺好用的，其本身是一个开源电子书托管平台，可以直接安装 npm 全局依赖，以创建基于 Markdown 的电子书，支持的插件也特别多，个人挺喜欢的，不过只适用于管理电子书，并不能做其他事情。

      到了最近，打算使用 VuePress 搭建自己的静态网站，偶然发现其 Markdown 渲染引擎是 markdown-it，出于好奇，研究了一番，发现真不得了，不但本身功能强大，还支持丰富的扩展，开发自己的扩展也很容易。

      下面，介绍一下我非常喜欢的一些 markdown-it 插件。

      # 标签型插件

      # markdown-it-sup

      使用 markdown-it-sup 生成上标标签。

      29^th^
      1

      输出

      <p>29<sup>th</sup></p>
      1

      渲染示例

      29th

      # markdown-it-sub

      使用 markdown-it-sub 生成下标标签。

      H~2~0
      1

      输出

      <p>H<sub>2</sub>O</p>
      1

      渲染示例

      H20

      # markdown-it-mark

      使用 markdown-it-sub 生成高亮标签。

      ==marked==
      1

      输出

      <p><mark>marked</mark></p>
      1

      渲染示例

      marked

      # markdown-it-ins

      使用 markdown-it-sub 生成插入标签。

      ++inserted++
      1

      输出

      <p><ins>inserted</ins></p>
      1

      渲染示例

      inserted

      # markdown-it-abbr

      使用 markdown-it-sub 生成缩写标签。

      *[HTML]: Hyper Text Markup Language
*[W3C]:  World Wide Web Consortium
The HTML specification
is maintained by the W3C.
      1
      2
      3
      4

      输出

      <p>The <abbr title="Hyper Text Markup Language">HTML</abbr> specification
is maintained by the <abbr title="World Wide Web Consortium">W3C</abbr>.</p>
      1
      2

      渲染示例

      The HTML specification is maintained by the W3C.

      # markdown-it-kbd

      使用 markdown-it-sub 生成键位标签。

      [[x]]
      1

      输出

      <kbd>x</kbd>
      1

      渲染示例

      x

      # markdown-it-underline

      通常情况下，_a_ 的写法会渲染为加粗显示，使用 markdown-it-underline 插件，可以将之渲染为下划线。

      _a_
      1

      输出

      <u>underline</u>
      1

      渲染示例

      a

      # 渲染型插件

      # markdown-it-emoji

      使用 markdown-it-emoji 可以很方便的输入表情符号。

      注意

      不要同 markdown-it-span 插件一起使用，否则会出现解析错误。

      官方定义的表情符号参见 shortcuts.js

      :tada: :100: :-( :") </3 <3 x-) :/ :'( :amphora: :alien: :D ,:( :*
      1

      渲染为

      🎉 💯 😦 😊 💔 ❤️ 😆 😕 😢 🏺 👽 😄 😓 😗

      也可以自定义表情符号，详见配置

      # 功能增强型插件

      # markdown-it-anchor

      使用 markdown-it-anchor 可以在标题前加上链接符号。

      参数

      Name Description Default
      level Minimum level to apply anchors on or array of selected levels. 1
      permalink Whether to add permalinks next to titles. false
      permalinkClass The class of the permalink anchor. header-anchor
      permalinkSymbol The symbol in the permalink anchor. ¶
      permalinkBefore Place the permalink before the title. false
      callback Called with token and info after rendering. undefined

      # markdown-it-table-of-contents

      使用 TOC (table of contents) 可以提取当前 md 文件的标题，以生成目录树。

      [[TOC]]
      1

      参数

      Name Description Default
      includeLevel Headings levels to use (2 for h2:s etc) [1, 2]
      containerClass The class for the container DIV "table-of-contents"
      markerPattern Regex pattern of the marker to be replaced with TOC /^\[\[toc\]\]/im
      listType Type of list (ul for unordered, ol for ordered) ul
      format A function for formatting headings undefined

      # markdown-it-footnote

      使用 markdown-it-footnote 可以生成脚注。

      Here is a footnote reference,[^1] and another.[^longnote]

[^1]: Here is the footnote.

[^longnote]: Here's one with multiple blocks.
      1
      2
      3
      4
      5

      # markdown-it-imsize

      使用 markdown-it-imsize 可以指定图片大小，如果只填写第一个参数，则高度自适应。

      ![test](/404.png =100x)
![test](/404.png =100x200)
      1
      2

      # markdown-it-implicit-figures

      使用 markdown-it-implicit-figures 可以在 img 标签外嵌套 figure 标签。

      [![](img.png)](page.html)
      1

      输出

      <figure><a href="page.html"><img src="img.png" alt=""></a></figure>
      1

      配置

      dataType: true,  // 输出 <figure data-type="image">, default: false
figcaption: true,  // 输出 <figcaption>alternative text</figcaption>, default: false
tabindex: true, // 输出 <figure tabindex="1+n">..., default: false
link: true // 输出 <a href="img.png"><img src="img.png"></a>, default: false
      1
      2
      3
      4

      # markdown-it-inline-comments

      使用 markdown-it-inline-comments 可以去除 md 文件中的注释，不渲染到 HTML 页面。

      # markdown-it-attrs

      使用 markdown-it-attrs 可以方便的为指定内容添加属性。

      # header{.style-me}

paragraph{data-toggle=modal}

paragraph *style me*{.red} more text
      1
      2
      3
      4
      5

      输出

      <h1 class="style-me">header</h1>
<p data-toggle="modal">paragraph</p>
<p>paragraph <em class="red">style me</em> more text</p>
      1
      2
      3

      $$$
P(A | B) = (P(B | A)P(A)) / P(B)
$$$
