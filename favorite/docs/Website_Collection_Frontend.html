# 动画

  • favico.js、GitHub、Awesomes

用动画徽章的方式激活你的网站图标。你可以自定义动画类型、位置、背景颜色和文本颜色

  • SVG.js、GitHub

  • AniJS、GitHub

  • Move.js、GitHub

Move.js是一个很小的函数库，能简单而优雅的支持CSS3.

  • GreenSock-JS、GitHub

  • Snap.svg、GitHub

SVG是一种创建交互式动画非常棒的方式，独立的分辨率的矢量图形在任何大小的屏幕上看起来效果都很好。Snap.svg库使操作SVG变得更jQuery操作DOM一样简单。

  • textillate.js、GitHub

一个简单的CSS文本动画插件，Textillate.js结合了一些极好的函数库，目的是为应用CSS3动画的任何文本提供一个简单易用的插件。

  • ZRender、Runoob、[GitHub](GitHub: https://github.com/ecomfe/zrender)

ZRender是二维绘图引擎，提供 Canvas、SVG、VML 等多种渲染方式。MVC封装，数据驱动，提供类Dom事件模型。ZRender 也是 ECharts 的渲染器。

  • RGraph、介绍

RGraph是一个使用HTML5 Canvas标签实现的图表制作Library。利用该Library生成的Chart具有可交互性，当鼠标点击或移过时会显示相应的信息，可以动态加载Chart或对特殊点进行缩放。

-ZingChart、介绍

ZingChart也是一个优秀的基于 HTML5 Canvas 开发的图表库。支持鼠标交互及数据钻取。

  • Processing.js、介绍

Processing.js是一个开放的编程语言，在不使用Flash或Java小程序的前提下，可以实现程序图像、动画和互动的应用。Processing.js是轻量，易于了解掌握的理想工具，可用于可视化的数据，创建用户界面和开发基于Web的游戏。

  • FABRIC.JS、介绍、GitHub

FABRIC.JS是一款简单而强大的JavaScript Canvas 库，提供了互动的对象模型，同时还包含 Canvas-to-SVG 解析器。

  • jCanvas、介绍、手册

jCanvas 就是一个 jQuery 的绘图插件，它封装了一些绘制图形的方法，只需编写几行代码即可生成图形。

  • two.js、介绍、GitHub、Awesome

Two.js 是面向现代 Web 浏览器的一个二维绘图 API。Two.js 可以用于多个场合：SVG，Canvas 和 WebGL，旨在使平面形状和动画的创建更方便，更简洁。

  • three.js、简介、WebGLobal中文网

webGL可以让我们在canvas上实现3D效果。而three.js是一款webGL框架，由于其易用性被广泛应用。如果你要学习webGL，抛弃那些复杂的原生接口从这款框架入手是一个不错的选择。

  • Paper.js、介绍、jq22

Paper.js是一款开源的矢量图形脚本框架，基于 HTML5 Canvas 开发，提供清晰的场景图、DOM和大量强大的功能用来创建各种向量图和贝塞尔曲线。

  • CreateJS、介绍、中文网

CreateJS是基于HTML5开发的一套模块化的库和工具。基于这些库，可以非常快捷地开发出基于HTML5的游戏、动画和交互应用。

  • KineticJS、介绍

Kinetic.JS 是一个封装了 HTML5 Canvas的JavaScript 库，能为桌面和移动应用提供高性能动画，转场效果，节点嵌套，分层，滤镜，缓存，事件处理以及更多功能。

  • pixi.js、介绍

Pixi.js 是一个 2D webGL 渲染器，提供无缝 Canvas 回退，支持主流浏览器，包括桌面和移动。

  • Mojs.io、GitHub

一个基于动画和平移的简单但强大的JavaScript库

  • animo.js、GitHub

堆栈动画，可以创建跨浏览器平台的模糊效果，在动画完成可以调用回调函数，创建更魔幻的动画。

  • Anima、GitHub

用Anima可以很容易的同时激活100多个对象，每一个元素都有大小和黏性去模拟一个真实对象

  • Animatic.js、jquery-plugins、GitHub

  • GSAP、GitHub、npm

GSAP是一套针对原生的、高性能HTML5动画的工具，并能在所有主流浏览器中工作。

  • WOW.js

WOW.js是一个当滚动时显示动画的JavaScript插件。非常适合用Animate.css的朋友

  • Velocity.js、GitHub

Velocity.js：对于运动设计者来说，这是一个极其迅速的动画渲染引擎。
