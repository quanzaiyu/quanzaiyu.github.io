                                    # 优质的 Chrome 插件

                                    # Tampermonkey、Violentmonkey

                                    Greasy Fork 可以为浏览器嵌入一些脚本，它们能够优化您的网页浏览体验。安装之后，有些脚本能为网站添加新的功能，有些能使网站的界面更加易用，有些则能隐藏网站上烦人的部分内容。在 Greasy Fork 上的用户脚本都是由用户编写并向全世界发表的，您可以免费安装，轻松体验。

                                    • Violentmonkey 暴力猴
                                    • Tampermonkey 油猴

                                    好用的脚本插件参考我的另一篇文章：优质的 Greasy Fork 插件

                                    参考文章：免费看VIP视频、屏蔽广告、无限制下载百度云，你的浏览器也能做到！

                                    # 右键搜

                                    扩展右键菜单，超过300种功能，方便的选择、组合或自定义页面、划词、图片菜单，并有生成、解析二维码和短网址功能。设置可使用云存储。最新增加功能共享、资源下载、重命名等功能。右键搜伴侣增加超级拖拽。

                                    # Imagus

                                    鼠标指针悬停在链接或缩略图上时直接在当前页面的弹出视图上显示这些图片、HTML5 视频/音频和内容专辑。

                                    # 图片助手

                                    图片助手(ImageAssistant) 批量图片下载器。一款用于嗅探、分析网页图片并提供批量下载等功能及在线收藏、检索、分享服务的浏览器扩展程序。

                                    # Vue.js devtools

                                    Chrome and Firefox DevTools extension for debugging Vue.js applications.

                                    # 谷歌访问助手

                                    最简单易用的谷歌访问助手，为chrome扩展用户量身打造。可以解决chrome扩展无法自动更新的问题，同时可以访问谷歌google搜索，gmail邮箱，google+等谷歌服务。

                                    # 高效网页截图编辑插件

                                    截取任意网页，可以像画图软件那样用直线、箭头、圆圈、文字做出标识。并可以方便的通过链接或者附件分享。

                                    # WEB前端助手(FeHelper)

                                    FE助手：包括JSON格式化、二维码生成与解码、信息编解码、代码压缩、美化、页面取色、Markdown与HTML互转、网页滚动截屏、正则表达式、时间转换工具、编码规范检测、页面性能检测、Ajax接口调试、密码生成器、JSON比对工具

                                    # Chrono下载管理器

                                    做Chrome浏览器中最好的下载管理器。

                                    # JSON Viewer

                                    The most beautiful and customizable JSON/JSONP highlighter that your eyes have ever seen.

                                    # Infinity 新标签页(Pro)

                                    Infinity 标签页，让你更轻松的接触互联网。

                                    # Restlet Client

                                    Restlet Client - REST API Testing.

                                    # Keeper

                                    通过网络浏览器来安全地储存，创建或自动填写您的登录信息。

                                    # 二维码(QR码)生成器(QR Code Generator)

                                    可以生成当前页面二维码,手机直接扫二维码就可以访问当前页面。

                                    # OneTab

                                    节省高达95％的内存，并减轻标签页混乱现象

                                    # The Great Suspender

                                    冻结暂时用不到的分页，达到系统资源释放。
