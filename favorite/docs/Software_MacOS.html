                    mindNode 是一款适用于Mac平台的思维导图软件，MindNode是一个可视化的表达想法的应用软件,简洁的界面让您可以集中精力产生和关联您的思路,画布可无限扩展,这意味着无论您的项目多大或多复杂，都能从容应对，简单易用，能够让我们快速的创建出美观的思维导图。
