# 高级替换

可以使用正则表达式进行高级替换，比如下图：

将内容

(.*)https://img.xiaoyulive.top/img/(.*\.jpg).*\)

替换为：

<imgLink src='$2' />

其中，$2 代表正则匹配到的第二个位置，每个匹配项需要使用括号()包裹。
